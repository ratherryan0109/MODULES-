Module 77: Open Graph Tags — Examples

Note: Open Graph tags are placed in the <head> section. View the source of this page to see included OG tags. Social previews below are simulated.

Example 1: Basic Open Graph Tags

The minimum set of OG tags needed for social sharing.

<meta property="og:title"
      content="Open Graph Tags Complete Guide">
<meta property="og:description"
      content="Learn how to implement Open Graph tags for rich social media sharing previews on Facebook, LinkedIn, and Twitter.">
<meta property="og:image"
      content="https://example.com/images/og-default.jpg">
<meta property="og:url"
      content="https://example.com/guides/open-graph">
<meta property="og:type"
      content="article">
<meta property="og:site_name"
      content="HTML Mastery Course">

Example 2: Article-Specific Open Graph Tags

For blog posts and news articles, add article-specific properties.

<meta property="og:type" content="article">
<meta property="article:published_time"
      content="2026-06-12T08:00:00+00:00">
<meta property="article:modified_time"
      content="2026-06-12T12:30:00+00:00">
<meta property="article:author"
      content="https://facebook.com/htmlmastery">
<meta property="article:section"
      content="Web Development">
<meta property="article:tag" content="HTML">
<meta property="article:tag" content="SEO">
<meta property="article:tag" content="Social Media">

Example 3: OG Image with Dimensions and Alt Text

Always specify image dimensions for optimal rendering.

<meta property="og:image"
      content="https://example.com/images/og-image.jpg">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta property="og:image:type" content="image/jpeg">
<meta property="og:image:alt"
      content="A laptop displaying code with social media icons around it">

<!-- Multiple images (platform chooses best match) -->
<meta property="og:image"
      content="https://example.com/images/og-square.jpg">
<meta property="og:image:width" content="800">
<meta property="og:image:height" content="800">

Example 4: Locale Configuration

Specify language and region for international content.

<!-- Primary locale -->
<meta property="og:locale" content="en_US">

<!-- Alternate locales for translated content -->
<meta property="og:locale:alternate" content="fr_FR">
<meta property="og:locale:alternate" content="es_ES">
<meta property="og:locale:alternate" content="de_DE">

Example 5: Video and Audio OG Tags

For pages featuring video or audio content.

<!-- Video -->
<meta property="og:video"
      content="https://example.com/videos/tutorial.mp4">
<meta property="og:video:width" content="1920">
<meta property="og:video:height" content="1080">
<meta property="og:video:type" content="video/mp4">
<meta property="og:video:secure_url"
      content="https://example.com/videos/tutorial.mp4">

<!-- Audio -->
<meta property="og:audio"
      content="https://example.com/audio/podcast.mp3">
<meta property="og:audio:type" content="audio/mpeg">
<meta property="og:audio:secure_url"
      content="https://example.com/audio/podcast.mp3">
Video/Audio Properties:

Use og:video and og:audio to link to media files. Always include width/height for video and use secure_url for HTTPS versions.

Example 7: Profile and Website OG Tags

For personal profiles or brand pages.

<!-- Website homepage -->
<meta property="og:title" content="HTML Mastery Course">
<meta property="og:type" content="website">
<meta property="og:description"
      content="Master HTML from beginner to advanced with our comprehensive course.">

<!-- Profile page -->
<meta property="og:type" content="profile">
<meta property="profile:first_name" content="John">
<meta property="profile:last_name" content="Doe">
<meta property="profile:username" content="johndoe">
<meta property="profile:gender" content="male">

Example 8: Complete OG-Ready Head Section

A production-ready head section with full social sharing support.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
          content="width=device-width, initial-scale=1.0">
    <title>Open Graph Guide | HTML Mastery</title>
    <meta name="description"
          content="Complete guide to implementing Open Graph tags.">

    <!-- Open Graph -->
    <meta property="og:title"
          content="Open Graph Guide | HTML Mastery">
    <meta property="og:description"
          content="Complete guide to implementing Open Graph tags for rich social sharing previews.">
    <meta property="og:image"
          content="https://example.com/images/og-guide.jpg">
    <meta property="og:image:width" content="1200">
    <meta property="og:image:height" content="630">
    <meta property="og:url"
          content="https://example.com/guides/opengraph">
    <meta property="og:type" content="article">
    <meta property="og:site_name" content="HTML Mastery">
    <meta property="og:locale" content="en_US">

    <!-- Article specifics -->
    <meta property="article:published_time"
          content="2026-06-12T08:00:00+00:00">
    <meta property="article:author"
          content="https://facebook.com/htmlmastery">
    <meta property="article:tag" content="HTML">
    <meta property="article:tag" content="Open Graph">

    <!-- Twitter -->
    <meta name="twitter:card" content="summary_large_image">
    <meta name="twitter:site" content="@htmlmastery">
    <meta name="twitter:title"
          content="Open Graph Guide | HTML Mastery">
    <meta name="twitter:description"
          content="Complete guide to implementing Open Graph tags.">
    <meta name="twitter:image"
          content="https://example.com/images/twitter-guide.jpg">
</head>
<body>
    <!-- Page content -->
</body>
</html>
Pro Tip: After adding OG tags, test them using the Facebook Sharing Debugger (developers.facebook.com/tools/debug), LinkedIn Post Inspector, and Twitter Card Validator (cards-dev.twitter.com/validator).
