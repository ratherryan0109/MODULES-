Module 7: HTML Paragraphs — Examples

Example 1: Basic Paragraphs

Notice the automatic spacing between paragraphs.

<p>This is the first paragraph. It contains several sentences that
demonstrate how text flows within a paragraph element.</p>
<p>This is the second paragraph. Browsers automatically add vertical
margin between paragraphs to improve readability.</p>
<p>This is the third paragraph. Each paragraph should focus on one
main idea or topic for the best reading experience.</p>
Rendered Output:

This is the first paragraph. It contains several sentences that demonstrate how text flows within a paragraph element.

This is the second paragraph. Browsers automatically add vertical margin between paragraphs to improve readability.

This is the third paragraph. Each paragraph should focus on one main idea or topic for the best reading experience.

Example 2: Line Breaks with <br>

<p>
    123 Main Street<br>
    New York, NY 10001<br>
    United States
</p>

<p>
    Roses are red,<br>
    Violets are blue,<br>
    HTML is fun,<br>
    And so are you!
</p>
Rendered Output:

123 Main Street
New York, NY 10001
United States

Roses are red,
Violets are blue,
HTML is fun,
And so are you!

Example 3: Preformatted Text with <pre>

<pre>
function greet(name) {
    console.log("Hello, " + name + "!");
    return true;
}
</pre>
Rendered Output:
function greet(name) {
    console.log("Hello, " + name + "!");
    return true;
}

Example 4: Thematic Break with <hr>

<section>
    <h2>Chapter 1: The Beginning</h2>
    <p>Story content for chapter one...</p>
</section>

<hr>

<section>
    <h2>Chapter 2: The Middle</h2>
    <p>Story content for chapter two...</p>
</section>
Rendered Output:

Chapter 1: The Beginning

Story content for chapter one...

Chapter 2: The Middle

Story content for chapter two...

Example 5: Paragraphs with Inline Elements

<p>
    This paragraph contains <strong>strong text</strong>,
    <em>emphasized text</em>, a <a href="#">hyperlink</a>,
    <code>code snippets</code>, and <mark>highlighted content</mark>.
</p>
Rendered Output:

This paragraph contains strong text, emphasized text, a hyperlink, code snippets, and highlighted content.

Example 6: Common Mistakes to Avoid

<!-- ❌ Don't use br for paragraph spacing -->
Text here<br><br><br>More text

<!-- ❌ Don't nest paragraphs -->
<p>This is <p>nested</p> wrong</p>

<!-- ✅ Correct spacing with paragraphs -->
<p>First paragraph content.</p>
<p>Second paragraph content.</p>
Summary:

❌ Never use <br><br><br> for spacing between paragraphs.

❌ Never nest a <p> inside another <p>.

✅ Always use separate <p> elements for separate paragraphs.

💡 Tip: Use CSS properties like margin and line-height to control paragraph spacing instead of empty paragraphs or multiple <br> tags.
