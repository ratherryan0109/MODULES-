# Mini Project: Build a 'News Article' Page with Proper Paragraph Structure

## Objective
Create a news article page that demonstrates correct paragraph usage, including multi-paragraph content, pull quotes, author bylines, and proper text flow.

## Requirements

Build a single HTML page for a news article with:

1. **Article Header**
   - Headline (using proper heading hierarchy)
   - Byline with author name and date
   - Category label

2. **Article Body**
   - At least 6 paragraphs of meaningful content (write about a fictional tech event)
   - One section must include a pull quote using `<blockquote>`
   - Another section must include an address using `<address>`
   - Use `<hr>` to separate sections of the article

3. **Sidebar (Aside)**
   - Related stories using a list
   - Author bio section

4. **Comments Section**
   - A sample comment with name, date, and text
   - A form to add a new comment

5. **Footer**
   - Copyright and links

## Step-by-Step Instructions

1. Create `news-article.html`
2. Build the HTML5 skeleton
3. Add the article header with proper heading hierarchy
4. Write the article body with proper paragraph use
5. Use `<blockquote>` for a pull quote
6. Add `<address>` for contact info
7. Create an aside with related stories
8. Build a comments section
9. Complete with footer
10. Validate the HTML

## Starter Code

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tech Breakthrough: New HTML Standard Announced</title>
</head>
<body>
    <!-- Build your news article here -->
</body>
</html>
```

## Bonus Challenges
- Add a sticky header that stays at the top
- Create a print-friendly version section
- Add social media share links at the end of each paragraph
- Include a readability score at the bottom

## Submission
Open in a browser and check that all paragraphs render with proper spacing. Validate with the W3C validator.
