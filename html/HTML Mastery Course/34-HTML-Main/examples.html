HTML <main> Element Examples

Run these examples to see how the <main> element structures content.

Example 1: Basic Document Structure with <main>

This demonstrates the proper placement of <main> between <header> and <footer>.

Rendered Output:

Home About Contact

Welcome to Our Website

This is the main content area. Screen readers will announce "main landmark" when entering this section.

The <main> element gives semantic meaning to the primary content of the page.

© 2026 HTML Mastery Course

HTML Code:

<header>
  <nav>
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </nav>
</header>

<main>
  <h1>Welcome to Our Website</h1>
  <p>This is the main content area.</p>
</main>

<footer>
  <p>&copy; 2026 HTML Mastery Course</p>
</footer>

Example 4: Common Mistakes (What NOT to Do)

Warning: The following demonstrates incorrect usage patterns. Do not use these in production.

Mistake 1: Multiple Visible <main> Elements

<!-- INVALID: Multiple visible <main> elements -->
<main>
  <h1>First Main</h1>
</main>
<main>  <!-- This is invalid -->
  <h1>Second Main</h1>
</main>

Mistake 2: Nesting <main> Inside <article>

<!-- INVALID: <main> inside <article> -->
<article>
  <main>
    <p>Content</p>
  </main>
</article>

Mistake 3: Wrapping Everything in <main>

<!-- INVALID: <main> should not wrap header/footer -->
<body>
  <main>
    <header>...</header>
    <section>...</section>
    <footer>...</footer>
  </main>
</body>
Correction: Use <main> only for content between <header> and <footer>, not wrapping them.
