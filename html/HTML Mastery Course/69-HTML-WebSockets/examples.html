HTML WebSockets - Interactive Examples

WebSockets enable real-time, bidirectional communication between browser and server.

Note: WebSocket examples require a WebSocket server. Examples 1-3 connect to the public echo server at wss://echo-websocket.websocket.org. Examples 4-6 simulate WebSocket behavior client-side for learning purposes.

const socket = new WebSocket('wss://echo.example.com'); socket.onopen = (e) => { console.log('Connected'); }; socket.onmessage = (e) => { console.log('Received:', e.data); }; socket.onclose = (e) => { console.log('Disconnected'); }; socket.onerror = (e) => { console.error('Error:', e); };

WebSocket ReadyState Values

ConstantValueDescription
CONNECTING0Socket has been created. Connection not yet open.
OPEN1Connection is open and ready to communicate.
CLOSING2Connection is in the process of closing.
CLOSED3Connection is closed or could not be opened.
let socket = new WebSocket('wss://echo.example.com'); console.log('State:', socket.readyState); // 0: CONNECTING → 1: OPEN → 2: CLOSING → 3: CLOSED

Example 4: Reconnection with Exponential Backoff

Demonstrates intelligent reconnection strategy.

Click "Start Demo" to see reconnection strategy in action...
class ReconnectManager { constructor(url, options = {}) { this.url = url; this.maxRetries = options.maxRetries || 10; this.baseDelay = options.baseDelay || 1000; this.maxDelay = options.maxDelay || 30000; this.retryCount = 0; } getDelay() { // Exponential backoff with jitter const delay = Math.min( this.baseDelay * Math.pow(2, this.retryCount), this.maxDelay ); return delay + Math.random() * 1000; } }
