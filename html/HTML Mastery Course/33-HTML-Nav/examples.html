🧭 HTML Nav Element - Examples

Example 1: Primary Horizontal Navigation

A standard top navigation bar with active page highlighted:

<nav aria-label="Main navigation">
  <ul>
    <li><a href="/" class="active" aria-current="page">Home</a></li>
    <li><a href="/about">About</a></li>
    <li><a href="/services">Services</a></li>
    <li><a href="/portfolio">Portfolio</a></li>
    <li><a href="/contact">Contact</a></li>
  </ul>
</nav>
  • Home
  • About
  • Services
  • Portfolio
  • Contact

Example 2: Vertical Sidebar Navigation

A vertical navigation menu commonly used in dashboards:

<nav aria-label="Sidebar navigation">
  <ul>
    <li><a href="/dashboard" class="active">Dashboard</a></li>
    <li><a href="/profile">Profile</a></li>
    <li><a href="/settings">Settings</a></li>
    <li><a href="/billing">Billing</a></li>
  </ul>
</nav>
  • Dashboard
  • Profile
  • Settings
  • Billing
  • Logout

Example 3: Breadcrumb Navigation

A breadcrumb trail using <ol> (order matters):

<nav aria-label="Breadcrumb">
  <ol>
    <li><a href="/">Home</a></li>
    <li><a href="/products">Products</a></li>
    <li><a href="/products/laptops">Laptops</a></li>
    <li aria-current="page">ProBook X1</li>
  </ol>
</nav>
  1. Home
  2. Products
  3. Laptops
  4. ProBook X1

Example 4: Pagination Navigation

Pagination controls with current page highlighted:

<nav aria-label="Pagination">
  <ul>
    <li><a href="#" aria-label="Previous page">« Prev</a></li>
    <li><a href="#" aria-label="Page 2">2</a></li>
    <li><a href="#" aria-current="page" aria-label="Page 3, current">3</a></li>
    <li><a href="#" aria-label="Next page">Next »</a></li>
  </ul>
</nav>
  • « Prev
  • 1
  • 2
  • 3
  • 4
  • 5
  • Next »

Example 5: Tab Navigation

Tab-style navigation with content sections:

<nav aria-label="Tabs">
  <ul>
    <li><a href="#" aria-current="page">Overview</a></li>
    <li><a href="#">Features</a></li>
    <li><a href="#">Pricing</a></li>
  </ul>
</nav>
  • Overview
  • Features
  • Pricing
  • Reviews
  • FAQ

Example 6: Footer Navigation

Navigation links in a page footer:

<footer>
  <nav aria-label="Footer navigation">
    <ul>
      <li><a href="/privacy">Privacy Policy</a></li>
      <li><a href="/terms">Terms of Service</a></li>
      <li><a href="/sitemap">Sitemap</a></li>
    </ul>
  </nav>
  <p>&copy; 2025 Company Name.</p>
</footer>

  • Terms of Service
  • Sitemap
