Module 4: HTML Elements — Examples

Example 1: Block-Level vs Inline Elements

Block elements take full width; inline elements flow with text.

<!-- Block-level elements -->
<div style="background: #3498db; color: white; padding: 10px;">Block 1</div>
<div style="background: #3498db; color: white; padding: 10px;">Block 2</div>

<!-- Inline elements -->
<span style="background: #e74c3c; color: white; padding: 5px;">Inline 1</span>
<span style="background: #e74c3c; color: white; padding: 5px;">Inline 2</span>
Rendered Output:
Block 1
Block 2
Inline 1 Inline 2

Example 2: Void (Self-Closing) Elements

These elements have no content and no closing tag.

<p>Line 1<br>Line 2</p>
<hr>
<img src="https://via.placeholder.com/100x50?text=Image" alt="Sample">
<input type="text" placeholder="Enter text...">
Rendered Output:

Line 1
Line 2

Example 3: Text Formatting Elements

Various elements for styling and emphasizing text.

<p>
    <strong>Bold text</strong>,
    <em>Italic text</em>,
    <u>Underlined</u>,
    <mark>Highlighted</mark>,
    <small>Small text</small>,
    <del>Deleted text</del>,
    <ins>Inserted text</ins>,
    H<sub>2</sub>O (subscript),
    E=mc<sup>2</sup> (superscript)
</p>
Rendered Output:

Bold text, Italic text, Underlined, Highlighted, Small text, Deleted text, Inserted text, H2O (subscript), E=mc2 (superscript)

Example 4: List Elements

Unordered, ordered, and description lists.

<h3>Unordered List</h3>
<ul>
    <li>Apples</li>
    <li>Oranges</li>
</ul>

<h3>Ordered List</h3>
<ol>
    <li>Step one</li>
    <li>Step two</li>
</ol>

<h3>Description List</h3>
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>
Rendered Output:

Unordered List

  • Apples
  • Oranges

Ordered List

  1. Step one
  2. Step two

Description List

HTML
HyperText Markup Language
CSS
Cascading Style Sheets

Example 5: Table Elements

<table border="1">
    <thead>
        <tr><th>Name</th><th>Age</th></tr>
    </thead>
    <tbody>
        <tr><td>Alice</td><td>30</td></tr>
        <tr><td>Bob</td><td>25</td></tr>
    </tbody>
    <tfoot>
        <tr><td colspan="2">Two people total</td></tr>
    </tfoot>
</table>
Rendered Output:
NameAge
Alice30
Bob25
Two people total

Example 6: Semantic Elements in Action

A mini page layout using only semantic elements.

<header>
    <h1>My Portfolio</h1>
    <nav><a href="#">Work</a> | <a href="#">About</a></nav>
</header>
<main>
    <article>
        <h2>Project: Website Redesign</h2>
        <p>Completed in 2026</p>
    </article>
    <aside>
        <h3>Skills</h3>
        <ul><li>HTML</li><li>CSS</li></ul>
    </aside>
</main>
<footer>&copy; 2026</footer>
Rendered Output:

My Portfolio

Work | About

Project: Website Redesign

Completed in 2026

Skills

  • HTML
  • CSS
© 2026
