HTML <article> Element Examples

Example 1: Basic Blog Article

Rendered Output:

How to Master HTML5 Semantics

By Jane Doe | June 1, 2026

HTML5 introduced a set of semantic elements that give meaning to web content. Unlike generic <div> elements, semantic elements describe their purpose to both developers and browsers.

Using semantic HTML improves accessibility, SEO, and code maintainability. Screen readers can navigate landmarks, search engines understand content structure, and developers write cleaner code.

Tags: HTML5, Semantics, Web Development

HTML Code:

<article>
  <header>
    <h2>How to Master HTML5 Semantics</h2>
    <p>By Jane Doe | <time datetime="2026-06-01">June 1, 2026</time></p>
  </header>
  <p>Content here...</p>
  <footer>
    <p>Tags: <a href="#">HTML5</a></p>
  </footer>
</article>
