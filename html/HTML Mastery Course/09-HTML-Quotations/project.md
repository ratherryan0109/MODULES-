# Mini Project: Build an 'HTML Quotation & Citation Guide'

## Objective
Create a comprehensive reference page that explains and demonstrates all HTML quotation and citation elements with real-world examples.

## Requirements

Build an HTML page with:

1. **Header & Navigation**
   - Title: "HTML Quotation & Citation Guide"
   - Links to each section

2. **Inline Quotations Section**
   - Explain `<q>` with examples
   - Show nested quotations
   - Show the `cite` attribute

3. **Block Quotations Section**
   - Explain `<blockquote>` with a multi-paragraph quote
   - Show proper attribution with `<cite>` inside
   - Demonstrate styling options

4. **Citations Section**
   - Explain `<cite>` with examples of books, articles, movies
   - Show the difference between `<cite>` and regular italic

5. **Abbreviations Section**
   - Explain `<abbr>` with `title` attribute
   - Show examples of common abbreviations

6. **Contact Information Section**
   - Show a properly formatted `<address>` block

7. **Comparison Table**
   - Table comparing all elements: element, type, purpose, visual

8. **Footer** with copyright

## Step-by-Step Instructions

1. Create `quotations-guide.html`
2. Build the HTML5 skeleton with full metadata
3. Create navigation
4. Build each quotation section with live examples
5. Include code samples in `<pre>` blocks
6. Add the comparison table
7. Validate the HTML

## Bonus Challenges
- Add a section on bidirectional text elements `<bdi>` and `<bdo>`
- Create a quote of the day feature using `<blockquote>`
- Add styling to make blockquotes visually appealing
