Web Accessibility - Interactive Examples

Example 1: Image Alt Text

✅ GOOD vs ❌ BAD

✅ Good

Screen reader: "Bar chart showing 40% increase in Q4 revenue"

❌ Bad

Screen reader: "chart.png" (unhelpful)

<!-- ✅ Good --> <img src="chart.png" alt="Bar chart showing 40% increase in Q4 revenue"> <!-- ❌ Bad --> <img src="chart.png" alt="chart.png"> <!-- Or worse: missing alt attribute entirely -->

❌ Non-Semantic

<div class="nav">

<!-- ✅ Semantic --> <header role="banner">...</header> <nav role="navigation">...</nav> <main role="main"> <article>...</article> <section>...</section> </main> <footer role="contentinfo">...</footer>

Example 3: Color Contrast

WCAG AA requires a minimum 4.5:1 contrast ratio for normal text.

✅ Good Contrast (15:1)

This text is easy to read on white background.

This text is easy to read on dark background.

❌ Poor Contrast (2.5:1)

Light gray text on white is hard to read.

Red text on light red background strains eyes.

/* ✅ Good contrast ratios */ .text { color: #333333; background: #ffffff; } /* 15:1 ratio */ .important { color: #0066cc; background: #ffffff; } /* 6:1 ratio */ /* ❌ Poor contrast */ .bad { color: #cccccc; background: #ffffff; } /* 1.5:1 ratio - fails! */ .error { color: #ff0000; background: #ffcccc; } /* ~2.5:1 ratio - fails! */

Example 5: Form Labels and Instructions

✅ Proper Labels






We'll never share your email.

❌ Missing Labels

Full Name:


Email:

(Screen reader users won't know what to enter)
<!-- ✅ Good --> <label for="email">Email Address</label> <input type="email" id="email" aria-describedby="hint"> <p id="hint">We'll never share your email.</p> <!-- ❌ Bad --> <span>Email:</span> <!-- Not associated with input --> <input type="email"> <!-- No label or id -->

Example 6: Focus Indicators

Visible focus indicators are essential for keyboard users.

✅ Visible Focus

Tab to these elements to see focus ring:

Link 1

❌ No Focus

Link without focus

Keyboard users can't see where they are!

/* ✅ Always show focus */ :focus-visible { outline: 3px solid #0066cc; outline-offset: 2px; } /* ❌ Never do this */ *:focus { outline: none; } /* Makes site unusable for keyboard users! */
Accessibility Tip: Use the tab key to navigate through examples. Notice how some elements are reachable and some are not. This is the reality for keyboard-only users every day.
