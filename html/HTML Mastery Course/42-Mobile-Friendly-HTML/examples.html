Mobile-Friendly HTML Examples

1. Touch-Friendly Targets (48x48px)

Home About Services Blog Contact 
button, a, .tap-target {
  min-height: 48px;
  min-width: 48px;
  padding: 12px;
}

2. Mobile-Optimized Form Inputs

<input type="email" autocomplete="email">
<input type="tel" autocomplete="tel">
/* 16px prevents iOS zoom on focus */
input { font-size: 16px; min-height: 48px; }
Note: These inputs use font-size: 16px to prevent iOS automatic zoom when focused. Always set 16px minimum on form elements.

3. Prevent Common Mobile Issues

/* Prevent iOS text size adjustment */
html { -webkit-text-size-adjust: 100%; }

/* Prevent overflow */
img, video, iframe { max-width: 100%; height: auto; }
pre { overflow-x: auto; word-wrap: break-word; }

/* Prevent double-tap zoom on buttons */
button { touch-action: manipulation; }

/* Prevent 300ms tap delay (handled by viewport tag) */
<meta name="viewport" content="width=device-width, initial-scale=1.0">

4. Lazy Loading Images

<img src="image.jpg" loading="lazy" 
     alt="Description" width="800" height="600">

<iframe src="map.html" loading="lazy"></iframe>

Using loading="lazy" defers loading off-screen resources, improving initial page load performance on mobile.
