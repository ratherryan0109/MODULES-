YouTube Embed — Interactive Examples

Demo videos use a public domain test video. Replace VIDEO_ID with your own content.

1. Basic YouTube Embed

Standard iframe embed with required attributes.

<iframe width="560" height="315"
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"
  title="YouTube video player"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>

2. Custom Player Parameters

Using URL parameters: muted autoplay, no controls, modest branding, start at 30s.

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ?autoplay=1&mute=1&controls=0&modestbranding=1&start=30">
Autoplay with mute is required by modern browser policies.

3. Privacy-Enhanced Mode (No Cookies)

Using www.youtube-nocookie.com prevents YouTube from storing cookies on user devices.

<iframe src="https://www.youtube-nocookie.com/embed/VIDEO_ID">

4. Looping a Video

To loop a single video, you must include the loop=1 and playlist=VIDEO_ID parameters.

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ?loop=1&playlist=dQw4w9WgXcQ">

5. Start and End Times

Use start and end parameters to play a specific segment.

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ?start=10&end=30">
