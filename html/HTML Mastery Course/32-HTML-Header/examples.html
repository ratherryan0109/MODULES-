🏷️ HTML Header Element - Examples

Example 1: Simple Page Header

A minimal header with a heading and tagline:

<header class="simple-header">
  <h1>My Website</h1>
  <p>Welcome to my personal website.</p>
</header>

My Website

Welcome to my personal website. Explore my portfolio and blog.

Example 2: Site Header with Logo, Navigation, and Search

A complete page header with branding, links, and a search form:

<header class="site-header">
  <div class="logo">
    <div class="logo-box">B</div>
    <span>BrandName</span>
  </div>
  <nav aria-label="Main">
    <ul>
      <li><a href="/">Home</a></li>
      <li><a href="/products">Products</a></li>
      <li><a href="/contact">Contact</a></li>
    </ul>
  </nav>
  <form class="search-form">
    <input type="search" placeholder="Search...">
    <button>Go</button>
  </form>
</header>

Example 3: Article Header with Metadata

A header inside an article with title, author, date, and categories:

<article>
  <header class="article-header">
    <h2>Understanding CSS Grid</h2>
    <p class="meta">
      <span>By Jane Doe</span>
      <span>March 15, 2025</span>
      <span>Category: CSS</span>
    </p>
  </header>
  <p>CSS Grid is a powerful layout system...</p>
</article>

Understanding CSS Grid

👤 By Jane Doe 📅 March 15, 2025 🏷️ Category: CSS

CSS Grid is a powerful two-dimensional layout system that allows you to create complex web layouts with rows and columns. It is supported in all modern browsers and has revolutionized the way developers approach page layout.

Example 4: Section-Level Header

A header used inside a section to introduce its content:

<section>
  <header class="section-header">
    <h2>Our Services</h2>
    <p>We offer professional services to grow your business.</p>
  </header>
  <div class="service-cards"> ... </div>
</section>

Our Services

We offer a wide range of professional services to help your business grow and succeed in the digital landscape.

Web Design

Beautiful, responsive websites.

Development

Full-stack web applications.

SEO

Search engine optimization.
