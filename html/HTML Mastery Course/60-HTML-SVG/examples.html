HTML SVG — Interactive Examples

1. Basic SVG Shapes

SVG provides several basic shape elements: circle, rect, ellipse, line, polygon, and polyline.

<circle cx="50" cy="50" r="40" fill="coral" />

<rect x="10" y="10" width="80" height="60" rx="5" fill="steelblue" />

<polygon points="50,5 95,90 5,90" fill="gold" />

2. Path Element — All Commands

The <path> element can create any shape using path commands.

<!-- Triangle -->
<path d="M50 10 L90 90 L10 90 Z" fill="coral" />

<!-- Quadratic Bezier -->
<path d="M10 80 Q50 10,90 80" fill="none" stroke="steelblue" stroke-width="3" />

<!-- Cubic Bezier -->
<path d="M10 80 C30 10,70 10,90 80" fill="none" stroke="gold" stroke-width="3" />
