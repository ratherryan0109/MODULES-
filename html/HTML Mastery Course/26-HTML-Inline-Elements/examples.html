Example 2: Box Model on Inline Elements

Inline elements ignore width and height. Padding and borders apply but may overlap:

Text before This inline span has padding, border, and margin text after.

The margin-top/bottom has no visual effect. Padding extends the background but does not increase line height.

Example 4: Whitespace Between Inline Elements

Notice the gaps between the red boxes below — these are caused by whitespace in the HTML:

A B C

To remove gaps, either remove whitespace in HTML, use font-size: 0 on parent, or use flexbox.

<span>A</span><span>B</span><span>C</span> <!-- No gaps -->
