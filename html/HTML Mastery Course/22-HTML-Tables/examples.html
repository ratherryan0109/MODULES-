📊 HTML Tables - Examples

Example 1: Basic Table

A simple table with headers and data rows:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>28</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Bob</td>
    <td>34</td>
    <td>London</td>
  </tr>
  <tr>
    <td>Charlie</td>
    <td>45</td>
    <td>Tokyo</td>
  </tr>
</table>
Name Age City
Alice 28 New York
Bob 34 London
Charlie 45 Tokyo

Example 3: Colspan and Rowspan

Merging cells with colspan and rowspan:

<tr>
  <th>Name</th>
  <th colspan="2">Contact Information</th>
</tr>
<tr>
  <td rowspan="2">Alice</td>
  <td>Email</td>
  <td>alice@example.com</td>
</tr>
Name Contact Information
Alice Johnson Email alice@example.com
Phone +1-555-0100
Bob Smith Email bob@example.com
Phone +1-555-0101

Example 5: Full Semantic Structure

Using <thead>, <tbody>, and <tfoot>:

<table>
  <thead>
    <tr><th>Department</th><th>Q1</th><th>Q2</th></tr>
  </thead>
  <tbody>
    <tr><td>Sales</td><td>$50K</td><td>$65K</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>$150K</td><td>$185K</td></tr>
  </tfoot>
</table>
