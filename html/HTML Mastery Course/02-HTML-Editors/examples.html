Example 2: Setting Up Live Server

Instructions to get live reload working in VS Code.

Step 1: Install the "Live Server" extension from the VS Code marketplace.
Step 2: Create an index.html file with the following content:
        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Live Server Demo</title>
</head>
<body>
    <h1>Live Server is Running!</h1>
    <p>Edit this file and save — the browser will auto-refresh.</p>
</body>
</html>
Step 3:

Right-click on the HTML file → "Open with Live Server"
Your default browser opens at http://127.0.0.1:5500

Example 6: Well-Formatted vs Poorly-Formatted HTML

Good formatting makes code readable. Both examples render identically.

<!-- Poor formatting (works but hard to read) -->
<div><h1>Title</h1><p>Text</p></div>

<!-- Good formatting (readable, maintainable) -->
<div>
    <h1>Title</h1>
    <p>Text</p>
</div>
Both render as:

Title

Text

💡 Tip: Press Shift+Alt+F in VS Code to automatically format your HTML document.
