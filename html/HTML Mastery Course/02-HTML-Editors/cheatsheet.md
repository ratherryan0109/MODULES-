# HTML Editors — Cheatsheet

## Recommended Editors

| Editor | Type | Price | Best For |
|--------|------|-------|----------|
| VS Code | Code Editor | Free | General web development |
| Sublime Text | Code Editor | $99 eval | Speed, lightweight editing |
| Brackets | Code Editor | Free | Live Preview, Adobe tools |
| CodePen | Online Editor | Free/Pro | Prototyping, sharing |
| Dreamweaver | WYSIWYG | Paid | Visual design |

## VS Code Setup Checklist

- [ ] Install from code.visualstudio.com
- [ ] Enable Auto Save (File → Auto Save)
- [ ] Install Live Server extension
- [ ] Install Prettier extension
- [ ] Install Auto Rename Tag extension
- [ ] Configure setting: `"editor.formatOnSave": true`
- [ ] Learn Emmet (built-in)

## Essential Extensions

| Extension | Purpose |
|-----------|---------|
| Live Server | Auto-reload browser on save |
| Prettier | Auto-format code |
| Auto Rename Tag | Rename paired tags |
| HTML CSS Support | Class autocomplete |
| Path Intellisense | Path autocomplete |

## Emmet Shorthand

| Abbreviation | Result |
|-------------|--------|
| `!` | Full HTML5 boilerplate |
| `h1{text}` | `<h1>text</h1>` |
| `ul>li*3` | `<ul>` with 3 `<li>` |
| `div#id` | `<div id="id">` |
| `.class` | `<div class="class">` |
| `a[href="#"]` | `<a href="#">` |
| `h1{Title}+p` | Sibling elements |
| `nav>ul>li*3>a` | Deep nesting |
