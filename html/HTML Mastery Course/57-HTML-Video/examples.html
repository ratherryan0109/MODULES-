HTML Video — Interactive Examples

All examples use freely available demo videos from the web.

1. Basic Video with Controls

The simplest way to embed video — the controls attribute provides native play/pause, seek, and volume.

<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Your browser does not support video.
</video>

2. Video with Poster Image

The poster attribute displays an image before the user plays the video.

<video controls poster="thumbnail.jpg" width="640" height="360">

3. Background Video (Autoplay, Loop, Muted)

Use autoplay loop muted playsinline for background videos that play without sound.

<video autoplay loop muted playsinline class="bg-video">
  <source src="bg.mp4" type="video/mp4">
</video>

5. Responsive Video Sources

Use the media attribute on <source> to serve different qualities based on screen width.

<video controls>
  <source src="high.mp4" type="video/mp4" media="(min-width: 1200px)">
  <source src="medium.mp4" type="video/mp4" media="(min-width: 768px)">
  <source src="low.mp4" type="video/mp4">
</video>

Resize your browser window to see different sources load (conceptual — same video used here).

6. Video with Subtitles (WebVTT Tracks)

The <track> element provides captions, subtitles, and chapters using WebVTT files.

<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="captions.vtt" kind="captions" srclang="en" label="English" default>
</video>
WebVTT captions require an actual .vtt file. This example shows the markup pattern.

Inline WebVTT captions are embedded. Enable captions from the video controls.

8. Picture-in-Picture Mode

Enable floating video that stays on top of other windows.

video.requestPictureInPicture();
document.exitPictureInPicture();

9. Video Format Support Detection

Check which video formats your browser supports.

const v = document.createElement('video');
v.canPlayType('video/mp4');
