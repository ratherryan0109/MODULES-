📋 HTML Description Lists - Examples

Example 1: Basic Glossary

A simple description list defining web technologies:

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language - the standard language for creating web pages.</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets - used for styling HTML elements.</dd>

  <dt>JavaScript</dt>
  <dd>A programming language that enables interactive web pages.</dd>
</dl>
HTML
HyperText Markup Language - the standard language for creating web pages.
CSS
Cascading Style Sheets - used for styling HTML elements.
JavaScript
A programming language that enables interactive web pages.

Example 2: Multiple Descriptions for One Term

One term with three different descriptions:

<dl>
  <dt>Color</dt>
  <dd>A property that defines the text color of an element.</dd>
  <dd>Can be specified using named colors, HEX, RGB, or HSL values.</dd>
  <dd>Inherited property in CSS.</dd>
</dl>
Color
A property that defines the text color of an element.
Can be specified using named colors, HEX, RGB, or HSL values.
Inherited property in CSS.

Example 3: Multiple Terms, Same Description

Two terms sharing one description:

<dl>
  <dt>HTTP</dt>
  <dt>HTTPS</dt>
  <dd>Protocols used for transferring data over the web.</dd>
</dl>
HTTP
HTTPS
Protocols used for transferring data over the web.

Example 4: Nested HTML Inside Description

Using paragraphs and unordered lists inside <dd>:

<dl>
  <dt>Frontend Development</dt>
  <dd>
    <p>Technologies used on the client side:</p>
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </dd>
</dl>
Frontend Development

Technologies used on the client side:

  • HTML
  • CSS
  • JavaScript

Example 5: Metadata Pairs

Using a description list for document metadata:

  <dt>Category</dt>
  <dd>Web Development</dd>
  <dt>Tags</dt>

</dl>

Example 6: CSS Styled Description List

A description list with custom styling applied:

.styled-dl dt {
  font-size: 1.2em;
  color: #7c3aed;
  border-bottom: 1px dashed #ddd;
  padding-bottom: 4px;
}
.styled-dl dd {
  font-style: italic;
  color: #6b7280;
}
API
Application Programming Interface - a set of rules for building software applications.
DOM
Document Object Model - a programming interface for web documents.
AJAX
Asynchronous JavaScript and XML - a technique for creating dynamic web pages.

Example 7: FAQ Using Description List

FAQ pairs are a natural fit for description lists:

<dl>
  <div>
    <dt>What is HTML?</dt>
    <dd>HTML stands for HyperText Markup Language.</dd>
  </div>
  <div>
    <dt>What is CSS?</dt>
    <dd>CSS stands for Cascading Style Sheets.</dd>
  </div>
</dl>
What is HTML?
HTML stands for HyperText Markup Language. It is the standard language for creating web pages.
What is CSS?
CSS stands for Cascading Style Sheets. It is used to style and layout web pages.
What is JavaScript?
JavaScript is a programming language that adds interactivity to websites.
