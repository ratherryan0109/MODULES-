Example 4: Standard Title Patterns

<!-- Homepage --> <title>SiteName — Web Development Tutorials & Resources</title> <!-- Article/ Blog Post --> <title>How to Center a Div — CSS Tutorial | SiteName</title> <!-- Category Page --> <title>CSS Tutorials — Learn Cascading Style Sheets | SiteName</title> <!-- Product Page --> <title>Product Name — Category | BrandName</title> <!-- About Page --> <title>About Us — Our Story & Team | SiteName</title> <!-- Contact Page --> <title>Contact Us — Get in Touch | SiteName</title> <!-- 404 Error Page --> <title>404: Page Not Found — SiteName</title> <!-- Search Results --> <title>Search Results for "keyword" — SiteName</title> <!-- Multi-step Process --> <title>Checkout — Step 2 of 3: Shipping | StoreName</title>

Example 5: Dynamic Title with JavaScript

Current page title:

<script> // Change title dynamically document.title = "New Page Title"; // Add notification badge function addNotificationBadge(count) { document.title = `(${count}) New Messages — Mail App`; } // Update title based on page visibility document.addEventListener('visibilitychange', () => { if (document.hidden) { document.title = 'Come back! — SiteName'; } }); </script>
