Example 3: Basic Favicon (ICO format)

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Website</title> <!-- Basic ICO favicon --> <link rel="icon" type="image/x-icon" href="/favicon.ico"> </head> <body> ... </body> </html>
Tip: If you place favicon.ico in the website root directory, most browsers will find it automatically without needing a <link> tag.

Example 4: Modern Multi-Format Favicon Setup

<!-- SVG Favicon (modern browsers) --> <link rel="icon" type="image/svg+xml" href="/favicon.svg"> <!-- PNG fallback (multiple sizes) --> <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png"> <!-- Apple Touch Icon (iOS) --> <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png"> <!-- Android / PWA --> <link rel="manifest" href="/site.webmanifest"> <!-- Windows Tiles --> <meta name="msapplication-TileColor" content="#ffffff"> <meta name="msapplication-TileImage" content="/mstile-144x144.png"> <!-- Browser theme color --> <meta name="theme-color" content="#ffffff">
Best practice: Use an online favicon generator to create all these sizes automatically.

Example 5: Emoji Favicon with Data URI

This page itself uses an emoji favicon via a data URI — a quick trick for development:

<link rel="icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>🚀</text></svg>">
This works by encoding the SVG inline as a data URI. Change the emoji to any character to create instant favicons!

Example 6: SVG Favicon File

Contents of favicon.svg:

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32"> <rect width="32" height="32" rx="6" fill="#2563eb"/> <text x="16" y="22" font-family="Arial" font-size="18" fill="white" text-anchor="middle" font-weight="bold">M</text> </svg>

This creates a simple blue square with the letter "M" — perfectly crisp at any size.

Example 7: Web Manifest (site.webmanifest)

{ "name": "My Awesome Website", "short_name": "MySite", "description": "A description of my website", "start_url": "/", "display": "standalone", "background_color": "#ffffff", "theme_color": "#2563eb", "icons": [ { "src": "/icon-192x192.png", "sizes": "192x192", "type": "image/png" }, { "src": "/icon-512x512.png", "sizes": "512x512", "type": "image/png" } ] }

Link it in your HTML:

<link rel="manifest" href="/site.webmanifest">
