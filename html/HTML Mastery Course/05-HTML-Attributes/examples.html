Module 5: HTML Attributes — Examples

Example 1: Global Attributes (id, class, title, style)

These attributes can be used on any HTML element.

<div id="main-content" class="container">
    <p title="This is a tooltip" style="color: blue;">
        Hover over me to see the title!
    </p>
    <p class="highlight important">
        This paragraph has two classes.
    </p>
</div>
Rendered Output:

Hover over me to see the title!

This paragraph has two classes.

Example 2: Anchor Tag Attributes

<a href="https://example.com" target="_blank" rel="noopener noreferrer">
    Opens in new tab (safe)
</a>

<a href="mailto:hello@example.com">Send Email</a>

<a href="#section2">Jump to Section 2</a>

<a href="document.pdf" download="MyDocument.pdf">Download PDF</a>
Rendered Output:
Opens in new tab (safe)
Send Email
Jump to Section 2
Download PDF

Example 4: Boolean Attributes

Boolean attributes need no value — their presence means true.

<input type="text" placeholder="I am required" required>
<input type="text" placeholder="I am disabled" disabled>
<input type="text" value="Read only" readonly>
<details open>
    <summary>Click to toggle</summary>
    <p>This is open by default.</p>
</details>
Rendered Output:






Click to toggle

This is open by default.

Example 5: Custom Data Attributes

Use data-* to store custom information in elements.

<div data-product-id="101" 
     data-category="electronics" 
     data-price="299.99" 
     data-in-stock="true">
    <h3>Wireless Headphones</h3>
    <p data-discount="0.15">Price: $299.99</p>
</div>
Rendered Output:

Wireless Headphones

Price: $299.99

Example 6: ARIA Accessibility Attributes

<nav aria-label="Main navigation">
    <a href="#" aria-current="page">Home</a>
    <a href="#">About</a>
</nav>

<button aria-pressed="false" onclick="this.ariaPressed='true'">
    Toggle Me
</button>

<div role="alert" aria-live="assertive">
    Form submitted successfully!
</div>
Rendered Output:
Home | About

Form submitted successfully!
💡 Tip: Use Ctrl+Shift+C in browser DevTools to inspect any element's attributes on any website.
