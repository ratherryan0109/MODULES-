# Mini Project: Responsive Blog Homepage

## Overview

Build a responsive blog homepage combining Flexbox and CSS Grid. Use Grid for the page layout and Flexbox for components like nav, cards, and footer sections.

## Requirements

1. CSS Grid Holy Grail layout (header, nav, main, aside, footer)
2. Blog post cards in main using Flexbox for layout
3. Responsive: mobile (single column), tablet (2 columns), desktop (Holy Grail)
4. Navigation bar with Flexbox (horizontal desktop, hamburger mobile)
5. Sidebar with categories, popular posts, and newsletter signup
6. Multi-column footer with Flexbox
7. All images responsive with `max-width: 100%`
8. Proper semantic HTML5 elements

## Steps

1. Create HTML with `<header>`, `<nav>`, `<main>`, `<aside>`, `<footer>`
2. Apply CSS Grid for page layout
3. Style nav with Flexbox
4. Style blog cards with Flexbox
5. Style footer with Flexbox
6. Add media queries for responsive breakpoints
7. Test layout at 320px, 768px, 1024px

- [ ] Semantic HTML5 elements
- [ ] Responsive images
