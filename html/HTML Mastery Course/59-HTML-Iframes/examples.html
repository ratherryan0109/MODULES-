HTML Iframes — Interactive Examples

1. Basic Iframe Embed

Simple iframe with src, width, height, and title attributes.

<iframe src="https://example.com" width="600" height="400" title="Example page">

</iframe>

2. Inline HTML with srcdoc

The srcdoc attribute allows embedding inline HTML without a separate URL.

<iframe srcdoc="<h1>Hello!</h1><p>This is inline content.</p>"></iframe>

3. Sandbox Attribute

The sandbox attribute restricts iframe capabilities. Compare the sandboxed iframe (no scripts) vs unrestricted.

4. Responsive Iframe

Using the padding-bottom trick to maintain aspect ratio.

.container { position: relative; padding-bottom: 56.25%; height: 0; }
.container iframe { position: absolute; width: 100%; height: 100%; }

5. Lazy Loading Iframe

Use loading="lazy" to defer loading off-screen iframes until they are near the viewport.

<iframe src="page.html" loading="lazy" title="..."></iframe>

7. Permissions Policy (allow attribute)

The allow attribute controls which browser features the iframe can access.

<iframe src="camera.html" allow="camera; microphone"></iframe>
Note: The allow attribute only grants permission; the browser still prompts the user.

8. Targeting Forms and Links with name Attribute

Use the name attribute to make an iframe the target of links or form submissions.

<iframe name="myframe"></iframe>
<a href="page.html" target="myframe">Load in iframe</a>
Load Example.com → Load Wikipedia →

9. Referrer Policy

The referrerpolicy attribute controls what referrer information is sent when following links inside the iframe.

<iframe src="page.html" referrerpolicy="no-referrer"></iframe>
