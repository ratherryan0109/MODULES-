Module 11: HTML Colors — Examples

Example 1: Named Colors

HTML supports 140+ predefined color names.

<p style="color: red;">Red text</p>
<p style="color: blue; background-color: yellow;">Blue on yellow</p>
Rendered Output:
Red
Blue
Green
Orange
Purple
Teal
Coral
Crimson

Example 2: Hexadecimal Color Codes

<p style="color: #FF5733;">Custom orange</p>
<p style="color: #2ECC71;">Custom green</p>
<p style="color: #3498DB;">Custom blue</p>
Rendered Output:
#FF5733
#2ECC71
#3498DB

Example 3: RGB and RGBA Colors

<p style="color: rgb(255, 0, 0);">Red via RGB</p>
<p style="background-color: rgba(255, 0, 0, 0.3);">30% transparent red</p>
Rendered Output:
rgb(255,0,0)
rgb(0,255,0)
rgb(0,0,255)
rgb(255,255,0)
rgba(red, 0.3)
rgba(blue, 0.5)

Example 4: HSL and HSLA Colors

<p style="color: hsl(0, 100%, 50%);">Red via HSL</p>
<p style="color: hsla(240, 100%, 50%, 0.5);">50% transparent blue</p>
Rendered Output:
hsl(0,100%,50%)
hsl(120,100%,50%)
hsl(240,100%,50%)
hsl(60,100%,50%)
hsl(300,100%,50%)
hsla(red, 0.4)

Example 5: Color Applications (Text, Background, Borders)

<p style="color: #e74c3c;">Colored text</p>
<p style="background-color: #3498db; color: white;">Background color</p>
<p style="border: 3px solid #2ecc71; padding: 10px;">Border color</p>
Rendered Output:

This text is colored using color: #e74c3c

This has a blue background with white text

This has a green border

Example 6: Good vs Poor Contrast

<!-- Good contrast -->
<p style="color: #333; background: #fff;">Dark text on white</p>

<!-- Poor contrast -->
<p style="color: #ccc; background: #fff;">Light text on white</p>
Rendered Output:

✅ Good contrast: #333 on white

✅ Good contrast: white on #333

❌ Poor contrast: #ccc on white (hard to read)

❌ Poor contrast: #999 on #ddd

💡 Tip: Use tools like WebAIM Contrast Checker or Chrome DevTools to verify your color combinations meet WCAG accessibility standards (minimum 4.5:1 contrast ratio).
