# Mini Project: Build a 'Color Palette & Accessibility Dashboard'

## Objective
Create a color dashboard page that demonstrates all color methods, provides a palette reference, and educates about accessibility.

## Requirements

Build an HTML page with:

1. **Header**
   - Title: "HTML Colors & Accessibility Dashboard"

2. **Named Colors Section**
   - Display 20+ named colors in a grid with their names

3. **Hex Color Section**
   - Show a gradient of 8 shades of a color of your choice
   - Display the hex codes

4. **RGBA Section**
   - Show overlapping color boxes with different alpha values
   - Explain how transparency works

5. **HSL Section**
   - Show the color wheel (0°, 60°, 120°, 180°, 240°, 300°, 360°)

6. **Accessibility Section**
   - Table showing good vs bad contrast examples
   - WCAG AA and AAA requirements

7. **Application Section**
   - Color a sample webpage section (header, buttons, text) with colors
   - Show light and dark theme examples

8. **Footer** with copyright

## Step-by-Step Instructions

1. Create `color-dashboard.html`
2. Build the HTML5 skeleton
3. Create each color section with demonstrations
4. Add the accessibility contrast table
5. Create sample themed sections
6. Validate the HTML

## Bonus Challenges
- Add a color blindness simulation note for each combination
- Create a CSS variables section showing theme management
- Include a color picker reference
