HTML Audio — Interactive Examples

All examples use a generated sine wave audio (no external files needed).

1. Basic Audio with Controls

The simplest way to embed audio — the controls attribute provides native play/pause and volume.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Your browser does not support HTML5 audio.
</audio>

Using a demo track from SoundHelix. If it doesn't load, try another source.

2. Autoplay with Loop (Muted)

Autoplay requires the muted attribute in modern browsers. The audio loops indefinitely.

<audio autoplay loop muted>
  <source src="background.mp3" type="audio/mpeg">
</audio>
Note: Click the button below to allow unmuted autoplay once.

Autoplay: disabled (muted)

3. Preload Strategies

The preload attribute controls how much audio data the browser loads before play.

<audio controls preload="metadata">

preload="none": Loads nothing until user clicks play

preload="metadata": Loads duration and basic info

preload="auto": Loads entire file (default)

4. JavaScript Audio API — Programmatic Control

Create and control audio entirely from JavaScript.

const audio = new Audio('track.mp3');
audio.volume = 0.5;
audio.playbackRate = 1.5;
audio.play();

// Events
audio.addEventListener('ended', () => console.log('Finished'));

Ready

7. Audio Features Detection

Check which audio formats the user's browser supports.

const audio = document.createElement('audio');
audio.canPlayType('audio/mpeg'); // 'probably' | 'maybe' | ''

9. Media Session API Integration

Integrate with system media controls (lock screen, notification bar) using the Media Session API.

if ('mediaSession' in navigator) {
  navigator.mediaSession.metadata = new MediaMetadata({
    title: 'Song Title',
    artist: 'Artist Name',
    artwork: [{ src: 'cover.jpg', sizes: '512x512' }]
  });
}

10. Error Handling and Fallbacks

Robust audio implementations handle loading errors gracefully.

audio.addEventListener('error', (e) => {
  const msg = audio.error.message;
  showFallback();
});
