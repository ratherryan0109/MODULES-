Module 10: HTML Comments — Examples

Example 1: Basic Comment Syntax

Comments are not displayed in the browser. Here is the code:

<!-- This is a basic HTML comment. It will not appear on the page. -->
<p>This text is visible.</p>
<!-- <p>This paragraph is commented out and invisible.</p> -->
<p>This is also visible.</p>
Rendered Output:

This text is visible.

This is also visible.

(Note: only 2 paragraphs appear — the middle one is commented out)

Example 2: Multi-Line Comments for Documentation

<!--
========================================
  SECTION: PRODUCT GALLERY
  Author: Jane Doe
  Last Updated: 2026-06-12
  Description: Displays featured products
  in a responsive grid layout.
========================================
-->
<section class="product-gallery">
    <p>Featured products go here.</p>
</section>
Rendered Output:

Featured products go here.

Example 3: Using Comments for Debugging

<h2>Dashboard</h2>

<!-- DEBUG: Temporarily hiding chart until API is fixed
<div class="chart">
    <img src="chart.png" alt="Sales Chart">
</div>
-->

<p>Chart will be restored soon.</p>
Rendered Output (chart section is hidden):

Dashboard

Chart will be restored soon.

Example 4: Using Comments for Organization

<!-- ======= HEADER ======= -->
<header>
    <h1>My Website</h1>
</header>

<!-- ======= MAIN CONTENT ======= -->
<main>
    <!-- Hero Section -->
    <section>
        <h2>Welcome</h2>
    </section>
    
    <!-- Features Section -->
    <section>
        <h2>Features</h2>
    </section>
</main>

<!-- ======= FOOTER ======= -->
<footer>
    <p>&copy; 2026</p>
</footer>
Rendered Output:

My Website

Welcome

Features

© 2026

Example 6: Security Warning — What NOT to Put in Comments

⚠ DANGER: Never put sensitive information in HTML comments. They are visible to anyone who views the page source (Ctrl+U). 
<!-- ❌ NEVER DO THIS -->
<!-- Database password: SuperSecret123! -->
<!-- API Key: sk-abcdef123456789 -->
<!-- TODO: Remove debug code before launch -->
<!-- Credit card: 4111-1111-1111-1111 -->
How to view comments:

Press Ctrl+U (Windows) or Cmd+U (Mac) to view the HTML source of any page. All comments are visible!

Always review your comments before deploying to production.

💡 Tip: In VS Code, use Ctrl+/ to quickly comment or uncomment a line of HTML.
