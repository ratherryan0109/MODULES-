Example 1: Inline CSS

Inline CSS is applied directly on an element using the style attribute. It has the highest specificity but mixes content with presentation.

This div is styled with inline CSS — red background, white text, centered.
<div style="background-color: #e74c3c; color: white; padding: 15px; border-radius: 6px; text-align: center; font-weight: bold;"> This div is styled with inline CSS </div>

Example 2: Class Selector vs ID Selector

This is a regular card.
This card has both "card" and "highlight" classes.
This card uses an ID selector (unique).
<div class="card">Regular card</div> <div class="card highlight">Highlighted card</div> <div id="special-card">Special card (ID)</div> /* CSS */ .card { background: #ecf0f1; border-left: 5px solid #3498db; } .card.highlight { border-left-color: #e67e22; background: #fef9e7; } #special-card { background: #2c3e50; color: white; border-left-color: #e74c3c; }
Note: Classes can be reused on multiple elements. IDs must be unique per page.

Example 4: The CSS Box Model

Margin → Border → Padding → Content
This is the content area
width: 300px

Total width = 300px (content) + 40px (padding) + 10px (border) = 350px

.box { width: 300px; margin: 25px auto; /* orange - space outside border */ padding: 20px; /* green - space inside border */ border: 5px solid #3498db; /* blue - border line */ background: #fff; }

Example 6: CSS Specificity

This paragraph uses element selector — color: #333

This paragraph uses class selector — color: #e67e22 (wins over element)

This paragraph uses an ID selector — color: #2980b9 (wins over class)

This paragraph uses inline style — color: #2c3e50 (wins over ID)

/* Specificity weights: */ p { color: #333; } /* 0,0,0,1 — weakest */ .text { color: #e67e22; } /* 0,0,1,0 */ #override-text { color: #2980b9; } /* 0,1,0,0 */ style="color:#2c3e50;" /* 1,0,0,0 — strongest */

Example 7: Margin and Padding Shorthand

padding: 10px (all sides equal)
padding: 15px 30px (top/bottom 15, left/right 30)
padding: 10px 20px 30px 40px (top, right, bottom, left)
margin: 5px 10px 15px (top, left/right, bottom)
/* Shorthand order: top, right, bottom, left (clockwise) */ padding: 10px; /* all four sides */ padding: 15px 30px; /* top/bottom, left/right */ padding: 10px 20px 30px; /* top, left/right, bottom */ padding: 10px 20px 30px 40px; /* top, right, bottom, left */

Example 10: How to Link External CSS

In a real project, you would create a styles.css file and link it in the <head>:

<!-- index.html --> <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Website</title> <link rel="stylesheet" href="styles.css"> </head> <body> <h1>Hello World</h1> </body> </html>
/* styles.css */ * { margin: 0; padding: 0; box-sizing: border-box; } body { font-family: system-ui, sans-serif; background: #f8f9fa; color: #333; } h1 { color: #2563eb; text-align: center; margin-top: 50px; }
Key point: Always include rel="stylesheet" in the <link> tag, otherwise the browser won't load your CSS.
