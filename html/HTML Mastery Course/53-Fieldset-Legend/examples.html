Fieldset & Legend — Interactive Examples

1. Basic Fieldset with Legend

The simplest usage: a <fieldset> wrapping form controls with a <legend> caption.

<fieldset>
  <legend>Personal Information</legend>
  <label>Name: <input type="text"></label><br>
  <label>Email: <input type="email"></label>
</fieldset>
Personal Information

2. Radio Button Group (Required Fieldset)

Radio buttons must be grouped with <fieldset> and <legend> for accessibility.

<fieldset>
  <legend>Preferred Contact Method</legend>
  <label><input type="radio" name="contact" value="email"> Email</label>
  <label><input type="radio" name="contact" value="phone"> Phone</label>
</fieldset>
Preferred Contact Method

3. Checkbox Group

Checkbox groups benefit from fieldset grouping, especially when there are multiple related options.

4. Disabled Fieldset

The disabled attribute disables all form controls inside the fieldset.

<fieldset disabled>
  <legend>Premium Features</legend>
  <input type="checkbox"> Advanced Analytics
</fieldset>
Premium Features (Upgrade to Unlock)



Note: Notice how all inputs, the button, and even the labels appear grayed out and non-interactive.

5. CSS Styled Fieldset

You can remove the default groove border and apply custom styles to match your design.

fieldset { border: 2px solid #6C63FF; border-radius: 8px; padding: 20px; }
legend { background: #6C63FF; color: white; padding: 5px 15px; border-radius: 20px; }
Shipping Address

8. Fieldset with form Attribute

The form attribute on fieldset associates it with a form element by ID — even if placed outside the form tags.

<form id="externalForm" action="/submit" method="post">
  <button>Submit (from anywhere)</button>
</form>

<fieldset form="externalForm">
  <legend>External Group</legend>
  <input name="data">
</fieldset>
External Fields (outside form tags)

Click Submit — these fields are part of the form even though they are placed outside it.

10. Fieldset Reset for Custom Designs

When building fully custom designs, reset the default fieldset styles and start fresh.

fieldset { border: none; margin: 0; padding: 0; }
legend { padding: 0; font-weight: bold; }
