HTML <section> Element Examples

Example 1: Basic Landmark Sections

Each <section> below groups related thematic content.

Rendered Output:

About Us

We are a team dedicated to teaching web development. Our mission is to make learning accessible to everyone.

Our Services

We offer online courses, one-on-one mentoring, and corporate training programs.

Contact Information

Email: info@example.com
Phone: (555) 123-4567

HTML Code:

<main>
  <section>
    <h2>About Us</h2>
    <p>We are a team dedicated to teaching web development.</p>
  </section>
  
  <section>
    <h2>Our Services</h2>
    <p>We offer online courses...</p>
  </section>
  
  <section>
    <h2>Contact Information</h2>
    <p>Email: info@example.com</p>
  </section>
</main>

Example 2: Nested Sections for Hierarchical Content

Nested <section> elements create a logical document hierarchy.

Rendered Output:

Chapter 1: Getting Started

This chapter covers the basics.

1.1 Installation

How to install the required tools.

1.2 First Project

Creating your first project step by step.

1.2.1 Project Setup

Initialize the project structure.

1.2.2 Configuration

Configure build tools and dependencies.

HTML Code:

<section>
  <h2>Chapter 1: Getting Started</h2>
  <p>This chapter covers the basics.</p>
  
  <section>
    <h3>1.1 Installation</h3>
    <p>How to install the required tools.</p>
  </section>
  
  <section>
    <h3>1.2 First Project</h3>
    <p>Creating your first project.</p>
  </section>
</section>

Example 3: Sections as ARIA Landmarks

Sections become accessible landmarks when they have an accessible name via aria-labelledby or aria-label.

Rendered Output:

Key Features

Fast

Lightning-fast performance

Secure

Enterprise-grade security

Reliable

99.9% uptime guaranteed

What Our Customers Say

"Amazing service! Highly recommended."
"Transformed our business processes."

HTML Code:

<!-- Becomes a region landmark -->
<section aria-labelledby="features-title">
  <h2 id="features-title">Key Features</h2>
  <!-- Feature cards -->
</section>

<!-- Also becomes a region landmark -->
<section aria-label="Customer testimonials">
  <h2>What Our Customers Say</h2>
  <!-- Testimonials -->
</section>

Example 5: Common Mistakes (What NOT to Do)

Warning: The following demonstrates incorrect usage patterns.

Mistake 1: Section without a heading

<!-- INCORRECT: Missing heading -->
<section>
  <p>Content without a thematic heading.</p>
</section>

Mistake 2: Using section as a styling hook

<!-- INCORRECT: Use div instead -->
<section class="red-bg">
  <img src="photo.jpg" alt="">
</section>
Remember: Always ask yourself: "Does this content represent a thematic group?" If yes, use <section>. If you just need a container for styling, use <div>.
