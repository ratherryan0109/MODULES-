Colspan & Rowspan Examples

Example 1: Basic Colspan

The "Name" header spans 2 columns. Notice how colspan="2" merges First Name and Last Name into one header.

Name Age City
JohnDoe30New York
JaneSmith25Los Angeles
<tr> <th colspan="2">Name</th> <th>Age</th> <th>City</th> </tr>
Row 1 has 3 cells (one spans 2 columns = 3). Row 2 has 4 cells. Both total 4 columns.

Example 2: Basic Rowspan

The "Department" header spans 3 rows vertically using rowspan="3".

Engineering Alice Johnson Senior Dev
Bob Chen Junior Dev
Carol Davis QA Engineer
Marketing Dan Wilson Marketing Lead
Eve Martinez Content Writer
<tr> <th rowspan="3">Engineering</th> <td>Alice Johnson</td> <td>Senior Dev</td> </tr> <tr> <!-- th cells are omitted because rowspan covers them --> <td>Bob Chen</td> <td>Junior Dev</td> </tr>
When rowspan is used, the spanned rows omit the cell in that column position.

Example 3: Multi-Level Headers

Headers can be organized hierarchically with colspan for grouping.

Personal Info Contact Details
First Name Last Name Email Phone
JohnDoejohn@example.com555-0100
JaneSmithjane@example.com555-0102
Row 1 has 2 cells (colspan=2 each). Row 2 has 4 cells. Both total 4 columns.
