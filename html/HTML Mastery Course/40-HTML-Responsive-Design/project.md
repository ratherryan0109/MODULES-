# Mini Project: Complete Responsive Portfolio

## Overview

Build a fully responsive portfolio website that looks great on all devices. Use mobile-first approach with fluid grids, responsive images, and media queries.

## Requirements

1. Mobile-first CSS (base styles for mobile, min-width queries for larger)
2. Responsive navigation (hamburger on mobile, horizontal on desktop)
3. Hero section with fluid typography using `clamp()`
4. Project grid using `auto-fit/minmax` (1 col mobile → 2 col tablet → 3 col desktop)
5. Skills section with progress bars
6. Contact form with responsive inputs
7. Responsive images with `srcset` and `sizes`
8. Smooth scroll behavior
9. Back-to-top button
10. Footer with social links

## Steps

1. Create HTML structure with semantic elements
2. Write mobile-first CSS
3. Add media queries at 576px, 768px, 1024px, 1200px
4. Implement responsive navigation with JavaScript toggle
5. Add responsive images
6. Test on multiple device sizes

## Submission Checklist

- [ ] Mobile-first CSS approach
- [ ] `clamp()` for fluid typography
- [ ] `auto-fit/minmax` for project grid
- [ ] Responsive images with `srcset`
- [ ] Hamburger navigation (mobile) → horizontal (desktop)
- [ ] 4+ responsive breakpoints
- [ ] Touch targets >= 48px
- [ ] Valid HTML5
- [ ] No horizontal scrolling
- [ ] Back-to-top button
