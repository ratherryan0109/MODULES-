Module 1: Introduction to HTML — Examples

Example 1: The Basic HTML Skeleton

Every HTML page starts with DOCTYPE, html, head, and body tags.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>This is a paragraph.</p>
</body>
</html>
Rendered Output:

Welcome!

This is a paragraph.

Example 2: Text Formatting Tags

HTML provides tags to emphasize and structure text.

<p>This is <strong>strong</strong> text.</p>
<p>This is <em>emphasized</em> text.</p>
<p>This is <u>underlined</u> text.</p>
<p>This is <mark>highlighted</mark> text.</p>
Rendered Output:

This is strong text.

This is emphasized text.

This is underlined text.

This is highlighted text.

Example 3: Links and Images

Hyperlinks (<a>) and images (<img>) are core to web pages.

<a href="https://example.com">Visit Example.com</a>
<br>
<img src="https://via.placeholder.com/150" alt="Placeholder Image">
Rendered Output:
Visit Example.com

Example 4: Ordered and Unordered Lists

<h3>Shopping List (Unordered)</h3>
<ul>
    <li>Milk</li>
    <li>Bread</li>
    <li>Eggs</li>
</ul>

<h3>Recipe Steps (Ordered)</h3>
<ol>
    <li>Boil water</li>
    <li>Add pasta</li>
    <li>Cook for 10 minutes</li>
</ol>
Rendered Output:

Shopping List (Unordered)

  • Milk
  • Bread
  • Eggs

Recipe Steps (Ordered)

  1. Boil water
  2. Add pasta
  3. Cook for 10 minutes

Example 5: Proper Nesting

Always close inner tags before outer tags.

<div style="background: #eef; padding: 10px;">
    <h2>Nested Structure</h2>
    <p>This is a <span style="color: red;">red span</span> inside a paragraph.</p>
</div>
Rendered Output:

Nested Structure

This is a red span inside a paragraph.

Example 6: Semantic HTML5 Elements

HTML5 introduced meaningful elements like <header>, <nav>, <main>, <article>, <section>, and <footer>.

<header>
    <h1>My Blog</h1>
    <nav>
        <a href="#">Home</a> |
        <a href="#">About</a> |
        <a href="#">Contact</a>
    </nav>
</header>
<main>
    <article>
        <h2>Blog Post Title</h2>
        <p>Article content goes here...</p>
    </article>
</main>
<footer>
    <p>&copy; 2026 My Blog</p>
</footer>
Rendered Output:

My Blog

Home | About | Contact

Blog Post Title

Article content goes here...

© 2026 My Blog

💡 Tip: Right-click any web page and select "View Page Source" to see its HTML. This is a great way to learn from real websites!
