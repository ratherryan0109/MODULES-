HTML Web Workers - Interactive Examples

Web workers run JavaScript in background threads without blocking the UI.

Note: Some examples require a local server or may not work when opened directly as a file (file:// protocol). Use VS Code Live Server or Python's http module.

Example 1: Basic Worker Communication

Create a simple worker that doubles numbers sent to it.

Worker not started...
// Main script const worker = new Worker('js/basic-worker.js'); worker.postMessage(42); worker.onmessage = (e) => console.log(e.data); // Worker script (basic-worker.js) self.onmessage = (e) => { self.postMessage(e.data * 2); };

Example 5: Data Processing Pipeline

A multi-stage data processing pipeline using a worker: filter, transform, and aggregate.

Configure and click to process...
// Pipeline stages in worker self.onmessage = (e) => { let data = e.data; // Stage 1: Filter positives data = data.filter(n => n > 0); // Stage 2: Transform data = data.map(n => Math.sqrt(n)); // Stage 3: Aggregate const result = { sum: data.reduce((a, b) => a + b, 0), avg: data.reduce((a, b) => a + b, 0) / data.length, count: data.length }; self.postMessage(result); };
