# Mini Project: Build an 'HTML Text Formatting Encyclopedia'

## Objective
Create a comprehensive encyclopedia page that demonstrates every HTML text formatting element with examples, explanations, and proper usage guidance.

## Requirements

Build a single HTML page with:

1. **Header**
   - Title: "HTML Text Formatting Encyclopedia"
   - Navigation with links to all categories

2. **Category Sections** (one per category):
   - **Emphasis & Importance**: `<strong>`, `<em>`, `<mark>`, `<b>`, `<i>`
   - **Technical Elements**: `<code>`, `<kbd>`, `<samp>`, `<var>`, `<data>`
   - **Editorial Changes**: `<del>`, `<ins>`, `<s>`
   - **Typography**: `<sub>`, `<sup>`, `<small>`, `<u>`
   - **Quotations & Citations**: `<q>`, `<blockquote>`, `<cite>`, `<abbr>`, `<dfn>`
   - **Bi-directional Text**: `<bdi>`, `<bdo>`

3. **For Each Element** show:
   - Element name as a heading
   - A rendered example in context
   - The HTML code inside `<pre><code>`
   - A description of when to use it
   - A "Do not use for" note

4. **Comparison Section**
   - Table comparing `<strong>` vs `<b>`, `<em>` vs `<i>`, `<del>` vs `<s>`

5. **Footer** with copyright

## Step-by-Step Instructions

1. Create `formatting-encyclopedia.html`
2. Build the HTML5 skeleton
3. Create navigation linking to all sections
4. Build each category section with all elements
5. Add the comparison table
6. Include proper formatting in your own demo content
7. Validate the HTML

## Bonus Challenges
- Add interactive examples with toggles showing code/output
- Include a section on deprecated formatting tags
- Create a mini-quiz at the bottom using `<details>` elements
- Add a "copy code" button
