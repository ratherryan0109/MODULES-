Module 8: HTML Text Formatting — Examples

Example 1: Semantic vs Presentational Formatting

Strong has semantic meaning (importance). Bold is purely visual.

Emphasis has semantic meaning. Italic is purely visual.

<p>
    <strong>Strong</strong> has semantic meaning.
    <b>Bold</b> is purely visual.
</p>
<p>
    <em>Emphasis</em> has semantic meaning.
    <i>Italic</i> is purely visual.
</p>

Example 2: Complete Formatting Elements Demo

<p>
    <strong>Bold importance</strong>,
    <em>Italic emphasis</em>,
    <mark>Highlighted</mark>,
    <small>small text</small>,
    <del>deleted</del> <ins>inserted</ins>,
    H<sub>2</sub>O, E=mc<sup>2</sup>
</p>
Rendered Output:

Bold importance, Italic emphasis, Highlighted, small text, deleted inserted, H2O, E=mc2

Example 3: Technical Formatting Elements

<p>Use <code>document.getElementById()</code> in JavaScript.</p>
<p>Press <kbd>Ctrl</kbd> + <kbd>S</kbd> to save.</p>
<p>The server returned: <samp>Error 404: Not Found</samp></p>
<p>The variable <var>userName</var> stores the user's name.</p>
Rendered Output:

Use document.getElementById() in JavaScript.

Press Ctrl + S to save.

The server returned: Error 404: Not Found

The variable userName stores the user's name.

Example 4: Quotations and Citations

<p>
    As Shakespeare wrote in <cite>Hamlet</cite>:
    <q>To be, or not to be, that is the question.</q>
</p>
<p>
    The <abbr title="World Health Organization">WHO</abbr>
    published new guidelines today.
</p>
Rendered Output:

As Shakespeare wrote in Hamlet: To be, or not to be, that is the question.

The WHO published new guidelines today.

Example 5: Document Changes with <del> and <ins>

<p>Original price: <del>$49.99</del> <ins>$29.99</ins></p>
<p><del>This content has been removed.</del></p>
<p><ins>This new content has been added.</ins></p>
Rendered Output:

Original price: $49.99 $29.99

This content has been removed.

This new content has been added.

Example 6: Bi-directional Text Elements

<p><bdo dir="rtl">This text is right-to-left</bdo></p>
<p>User name: <bdi>أحمد</bdi> (isolated from direction)</p>
Rendered Output:

This text is right-to-left

User name: أحمد (isolated from direction)

💡 Tip: Always use semantic elements like <strong> and <em> over presentational <b> and <i> for better accessibility and SEO.
