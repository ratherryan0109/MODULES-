HTML Local Storage - Interactive Examples

Open your browser's Developer Tools (F12) → Application → Local Storage to see changes in real time.

Example 1: Basic CRUD Operations

Create, Read, Update, and Delete items in Local Storage.

Output will appear here...
// Set Item localStorage.setItem('username', 'JohnDoe'); // Get Item const value = localStorage.getItem('username'); // Update Item localStorage.setItem('username', 'JaneDoe'); // Remove Item localStorage.removeItem('username'); // Clear All localStorage.clear();

Example 2: Storing and Retrieving Objects

Local Storage only stores strings. Use JSON.stringify and JSON.parse for objects.

Click "Save User Object" to store a user profile...
// Storing an object const user = { id: 1, name: 'Alice', email: 'alice@example.com', preferences: { theme: 'dark', lang: 'en' } }; localStorage.setItem('user', JSON.stringify(user)); // Retrieving the object const storedUser = JSON.parse(localStorage.getItem('user')); console.log(storedUser.name); // 'Alice'

Example 4: Storage with Expiration (TTL Cache)

Store data that automatically expires after a set time.

Click "Cache Data" to store with expiration...
function setWithExpiry(key, value, ttlMs) { const item = { value: value, expiry: Date.now() + ttlMs }; localStorage.setItem(key, JSON.stringify(item)); } function getWithExpiry(key) { const itemStr = localStorage.getItem(key); if (!itemStr) return null; const item = JSON.parse(itemStr); if (Date.now() > item.expiry) { localStorage.removeItem(key); return null; } return item.value; }

function getStorageUsage() { let totalBytes = 0; for (let key in localStorage) { if (localStorage.hasOwnProperty(key)) { totalBytes += localStorage[key].length * 2; // UTF-16 } } return totalBytes; }

Example 6: Persisting User Preferences (Theme)

Change the theme and it persists across page reloads.

Current theme: light
// Save preference localStorage.setItem('preferredTheme', 'dark'); // Load preference on page load window.addEventListener('DOMContentLoaded', () => { const theme = localStorage.getItem('preferredTheme') || 'light'; document.body.className = `theme-${theme}`; });

Example 7: Form Autosave

Fill out the form, refresh the page - your data is restored!




Form data autosaves on every keystroke...
// Autosave on input document.querySelector('form').addEventListener('input', () => { const formData = new FormData(event.target.form); const data = Object.fromEntries(formData); localStorage.setItem('formDraft', JSON.stringify(data)); }); // Restore on load window.addEventListener('DOMContentLoaded', () => { const saved = localStorage.getItem('formDraft'); if (saved) { const data = JSON.parse(saved); Object.entries(data).forEach(([name, value]) => { const input = document.querySelector(`[name="${name}"]`); if (input) input.value = value; }); } });
