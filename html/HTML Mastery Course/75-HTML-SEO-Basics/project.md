# Project: SEO-Optimized Blog Platform

## Objective
Build a complete multi-page blog platform with full SEO optimization. The platform must demonstrate mastery of HTML SEO fundamentals: proper title/meta tags, semantic structure, heading hierarchy, image optimization, internal linking, structured data, and technical SEO.

## Requirements

### Pages (at least 5)
1. **Homepage** - Blog listing with post summaries
2. **Blog Post** (dynamic template) - Full article content
3. **Category Page** - Posts filtered by category
4. **About Page** - About the blog/author
5. **Contact Page** - Contact form with proper structure

### SEO Requirements
1. **Title Tags**: Unique per page, keyword-optimized, 50-60 chars, brand at end
2. **Meta Descriptions**: Unique per page, 150-160 chars, compelling, includes CTA
3. **Heading Hierarchy**: One h1 per page, proper nested structure, keyword-rich
4. **Semantic HTML**: header, nav, main, article, section, aside, footer
5. **Image Optimization**: Descriptive filenames, alt text, lazy loading, WebP, dimensions
6. **Internal Linking**: Related posts suggestions, breadcrumbs, contextual links
7. **URL Structure**: Clean, descriptive, hyphen-separated keywords
8. **Structured Data**: Article (for posts), BreadcrumbList (all pages), Organization (site-wide)
9. **Meta Robots**: index/follow for content pages, noindex for tag pages
10. **Canonical URLs**: Self-referencing canonical on every page
11. **XML Sitemap**: Include all pages with lastmod and priority
12. **Open Graph Tags**: og:title, og:description, og:image, og:url, og:type
13. **Twitter Cards**: twitter:card, twitter:title, twitter:description, twitter:image
14. **Performance**: Optimized Core Web Vitals (LCP < 2.5s, CLS < 0.1)

### Technical Requirements
- Valid HTML (Nu Checker - zero errors)
- Lighthouse SEO score 100%
- Mobile-friendly (viewport meta)
- HTTPS-ready
- Responsive design

### Delievables
- Complete HTML/CSS source code (5+ pages)
- XML sitemap
- robots.txt
- SEO audit report (pre-optimization vs post-optimization)
- Structured data test results
- Lighthouse performance report

### Evaluation Criteria
- Title/meta description optimization (20%)
- Semantic HTML and heading structure (20%)
- Structured data implementation (15%)
- Image SEO (10%)
- Internal linking strategy (10%)
- Technical SEO (XML sitemap, canonical, robots) (10%)
- Validation and testing (10%)
- Documentation (5%)
