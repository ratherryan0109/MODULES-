3. Heading Structure Comparison

Poor SEO

Services

Web Design

Good SEO

Web Design Services

Our Services

Professional web design, development, and SEO services.

Web Design

Custom responsive websites built with modern technologies.

<!-- Poor --> <div style="font-size:24px;font-weight:bold;">Welcome</div> <!-- Search engines see no heading structure --> <!-- Good --> <h1>Web Design Services</h1> <h2>Our Services</h2> <h3>Web Design</h3> <!-- Clear hierarchy for search engines -->

4. Image SEO Comparison

Poor
📸 IMG_001.jpg
<img src="IMG_001.jpg">
Good
📸 red-widget-product.jpg
<img src="red-widget-product.jpg" alt="Red widget with ergonomic design features" loading="lazy">

Rules: Descriptive filename, keyword-rich alt text (not stuffing), compress images, use WebP format, add lazy loading for below-fold images.

7. Structured Data (Schema.org)

Structured data helps search engines understand content context and enables rich snippets.

Article Schema

Breadcrumb Schema

<script type="application/ld+json"> { "@context": "https://schema.org", "@type": "Article", "headline": "HTML SEO Basics Guide", "description": "Complete guide...", "author": {"@type": "Person", "name": "Author Name"}, "datePublished": "2024-11-15" } </script>

8. Robots Meta Tags

Control how search engines interact with your page:

Meta Tag Effect Use Case
index, followIndex page, follow linksDefault for public pages
noindexDon't show in search resultsThank you pages, admin pages
nofollowDon't crawl links on this pageUntrusted content, login pages
noindex, nofollowDon't index or followPrivacy pages, staging sites
nosnippetDon't show description in resultsConfidential content
<meta name="robots" content="index, follow"> <meta name="googlebot" content="noindex"> <meta name="bingbot" content="index, follow">

9. Canonical URL

Prevents duplicate content issues by specifying the preferred URL:

<link rel="canonical" href="https://example.com/preferred-page-url">

Use when: Same content accessible via multiple URLs, printer-friendly versions, AMP pages, or URL parameters create duplicates.
