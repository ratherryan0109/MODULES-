Module 43: HTML Forms Introduction — Examples

Example 1: Basic Form Structure

<form action="#" method="GET">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <button type="submit">Submit</button>
</form>

Example 2: Labels and Accessibility

Click the label text — it focuses the input

<label for="email">Email Address:</label>
<input type="email" id="email" name="email" required>
The for attribute must match the input's id. This makes the label clickable and improves screen reader support.

Example 3: Fieldset and Legend

Contact Information 
<fieldset>
    <legend>Contact Information</legend>
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name">
</fieldset>

Example 4: Common Input Types

<input type="text">    <!-- Single line text -->
<input type="email">   <!-- Email with validation -->
<input type="password"> <!-- Masked input -->
<input type="number">   <!-- Numeric stepper -->
<input type="date">     <!-- Date picker -->
<input type="color">    <!-- Color picker -->

Example 5: Checkboxes and Radio Buttons

Select Your Interests (Checkboxes)
Experience Level (Radio Buttons) 
<!-- Checkboxes: multiple selections allowed -->
<input type="checkbox" name="interest" value="web">

<!-- Radio: single selection (same name groups them) -->
<input type="radio" name="level" value="beginner">

Example 6: Select Dropdown

<select id="country" name="country">
    <option value="">-- Select --</option>
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
</select>

Example 7: Textarea

<textarea id="message" name="message"
    rows="5" cols="50"
    placeholder="Write your message here..."></textarea>

Example 8: Complete Form Demo

Personal Info
Feedback 
<form action="#" method="POST">
    <fieldset>
        <legend>Personal Info</legend>
        <!-- form controls -->
    </fieldset>
    <button type="submit">Submit</button>
    <button type="reset">Reset</button>
</form>
Key Takeaways: Always use <label> with for, group related fields with <fieldset> and <legend>, use appropriate input types for validation, and remember the name attribute for form submission.
