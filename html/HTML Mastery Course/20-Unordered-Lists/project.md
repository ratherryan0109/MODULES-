# Mini Project: Build a Recipe Ingredient List

## Objective
Create a recipe page that uses nested unordered lists to display ingredients organized by categories, with CSS-styled custom bullet points.

## Requirements

Build a single HTML page with:

1. **Header**
   - Recipe title and a brief description
   - Estimated cooking time and servings

2. **Ingredients Section** (using nested `<ul>`)
   - Category: Produce (vegetables, fruits, herbs)
   - Category: Pantry (spices, oils, canned goods)
   - Category: Proteins (meat, alternatives)
   - Category: Dairy & Alternatives
   - Each category shows a minimum of 4 ingredients

3. **Equipment Section**
   - Unordered list of required cooking tools

4. **Instructions Section** (use `<ol>` for ordered steps)
   - Minimum 6 detailed steps

5. **Notes Section**
   - Tips, substitutions, and storage advice

6. **Footer**
   - Attribution and copyright

## Styling Requirements
- Custom bullet colors matching each category
- Hover effects on list items
- Checkbox-style to-dos for each ingredient
- Responsive layout

## Bonus Challenges
- Add a "scale recipe" feature using checkbox toggles
- Include dietary icons (vegan, gluten-free, etc.)
- Create a print-friendly stylesheet
- Add a shopping list summary (aggregates checked items)

```html
<!DOCTYPE html>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Recipe: Classic Spaghetti Bolognese</title>
</head>
<body>
    <!-- Build your recipe page here -->

```

## Submission
Validate with W3C validator, test in 2 browsers, and ensure all nested lists render correctly.
