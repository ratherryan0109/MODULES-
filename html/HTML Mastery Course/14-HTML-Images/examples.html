🖼️ HTML Images — Code Examples

Example 1: Basic Image Element

The simplest use of <img> requires src and alt attributes.

<img src="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=600&q=80" alt="A scenic mountain landscape at sunset" width="600" height="400" >
Note: The alt text describes the image for accessibility. Screen readers will read this text aloud.

Example 4: Width and Height Attributes

Set width and height: The browser reserves space for the image, preventing layout shifts.

The image has width="300" height="200" attributes set. CSS max-width:100% keeps it responsive.

<img src="photo.jpg" alt="Mountain landscape" width="300" height="200" style="max-width: 100%; height: auto;" >
Tip: Always set width and height attributes to prevent Cumulative Layout Shift (CLS). Use CSS for responsive scaling.

Example 6: Responsive Images with srcset and sizes

The browser automatically selects the best image size based on the viewport. Resize your browser to see it in action.

<img src="photo-400.jpg" srcset=" photo-400.jpg 400w, photo-800.jpg 800w, photo-1200.jpg 1200w " sizes=" (max-width: 600px) 100vw, (max-width: 1200px) 50vw, 800px " alt="Mountain landscape" >
How it works: The sizes attribute tells the browser the image display width at different viewport sizes. The browser then picks the best source from srcset.

Example 7: The <picture> Element (Art Direction)

Resize the browser to see different crops appear at different widths.

<picture> <source media="(min-width: 800px)" srcset="hero-wide.jpg"> <source media="(min-width: 400px)" srcset="hero-tablet.jpg"> <img src="hero-mobile.jpg" alt="Hero banner"> </picture>

Example 8: <picture> with Format Fallbacks

The browser picks AVIF → WebP → Fallback (JPEG/PNG) based on support.

<picture> <source type="image/avif" srcset="photo.avif"> <source type="image/webp" srcset="photo.webp"> <img src="photo.jpg" alt="Photo"> </picture>

Example 9: <figure> and <figcaption>

<figure> <img src="mountain.jpg" alt="Mountain range at sunrise"> <figcaption>Figure 1: Mountain range at sunrise</figcaption> </figure>

Example 10: Image Map (Clickable Regions)

Click on a region to see a message.

<img src="map.png" alt="Image map" usemap="#myMap"> <map name="myMap"> <area shape="rect" coords="0,0,100,100" href="page.html" alt="Region"> <area shape="circle" coords="150,150,50" href="other.html" alt="Circle"> </map>

Example 12: Inline SVG (Scalable Vector Graphics)

SVG scales perfectly at any resolution — no pixelation.

<svg width="400" height="120" viewBox="0 0 400 120"> <rect width="400" height="120" fill="#1a1a2e" rx="8"/> <text x="200" y="65" fill="white" text-anchor="middle"> Inline SVG </text> </svg>
