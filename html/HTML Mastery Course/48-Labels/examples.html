Label Techniques — Interactive Examples

1. Explicit Label (Recommended)

<label for="email">Email:</label><input id="email">

Click the label text — it focuses the input.

2. Implicit Label (Wrapping)

<label>Email: <input></label>

The checkbox is inside the label — click the text to toggle.

3. Required Label Indicator

<label class="required"></label>

4. Visually Hidden Label

<label class="visually-hidden"></label>

Label is hidden visually but available for screen readers.

5. aria-label

<input aria-label="Search">
