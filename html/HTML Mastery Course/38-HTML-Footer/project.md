# Mini Project: Complete Website Footer

## Overview

Build a production-quality website footer that includes multiple columns, navigation links, contact information, a newsletter signup form, social media links, and a copyright bar. The footer should be responsive and accessible.

## Requirements

1. Multi-column footer using CSS Grid (4 columns on desktop, 2 on tablet, 1 on mobile)
2. Sections: About, Quick Links, Contact, Newsletter
3. Newsletter form with email input and submit button
4. Social media icon links (SVG or emoji)
5. Copyright bar with current year and legal links
6. Back-to-top button
7. Sticky footer behavior (stays at bottom)
8. Proper ARIA labels: `aria-label` on nav, `aria-label` on form
9. Valid HTML5 and responsive design

## Steps

1. Create HTML structure with `<footer>` containing grid
2. Add sections for About, Quick Links, Contact, Newsletter
3. Style with CSS Grid, responsive breakpoints
4. Add back-to-top button with JavaScript scroll behavior
5. Test with screen reader and mobile devices

## Submission Checklist

- [ ] Multi-column CSS Grid layout
- [ ] Newsletter form with label and input
- [ ] Contact info in `<address>` element
- [ ] Social media links in `<nav>` with `aria-label`
- [ ] Copyright with `<small>` element
- [ ] Back-to-top button
- [ ] Sticky footer with flexbox
- [ ] Responsive (3 breakpoints)
- [ ] Valid HTML5
- [ ] ARIA labels on all landmarks
