Example 4: ID vs Class Comparison

ID (unique)

This element has an ID. It is unique.

Only one element can have #id-demo-1.

Class (reusable)

This element has a class.

This element also has the same class.

Multiple elements can share a class.

Use IDs when uniqueness matters (labels, anchors, JS). Use classes for reusable styling.
