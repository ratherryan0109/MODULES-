Example 6: Mobile-First Best Practices Checklist

✅ Viewport meta tag - <meta name="viewport" content="width=device-width, initial-scale=1.0">

✅ Relative units - Use rem, %, vw, vh instead of px for layout

✅ Flexible images - img { max-width: 100%; height: auto; }

✅ Mobile-first CSS - Base styles for mobile, media queries for larger screens

✅ Touch targets - Minimum 44x44px tap targets

✅ Readable text - Base font size 16px minimum

✅ Responsive navigation - Hamburger menu on mobile, horizontal on desktop

✅ Adequate contrast - WCAG AA minimum contrast ratio of 4.5:1
