HTML Block Elements Examples

Example 1: Basic Block Element Behavior

Each block element starts on a new line and takes full width:

Block 1: Full width, stacks vertically
Block 2: Also full width
Block 3: Notice how they stack
All three divs are block elements. Each occupies 100% width and appears on its own line.

Example 2: Width Control on Block Elements

Fixed width: 300px
Percentage width: 60%
Max-width: 600px (centered with margin: auto)
Block elements respect explicit width, percentage width, and max-width. Use margin: auto to center.

Example 3: Margin Collapsing

Vertical margins collapse to the larger of the two:

Box A (margin-bottom: 30px)
Box B (margin-top: 20px)
The gap is 30px (the larger margin), not 50px. This is margin collapsing.
.box-a { margin-bottom: 30px; }\n.box-b { margin-top: 20px; }\n/* Space between = 30px, not 50px */

Example 5: Box Model Visualization

Content area (blue)
