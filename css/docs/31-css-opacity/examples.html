CSS Opacity - Examples

Example 1: Opacity Values

Opacity ranges from 0 (fully transparent) to 1 (fully opaque).

opacity: 1
opacity: 0.75
opacity: 0.5
opacity: 0.25
<div style="opacity: 1;">Fully opaque</div>
<div style="opacity: 0.75;">75% opaque</div>
<div style="opacity: 0.5;">Half transparent</div>
<div style="opacity: 0.25;">Mostly transparent</div>

Example 2: rgba() vs Opacity

rgba() only affects the background; opacity affects the entire element and its children.

rgba(6,182,212,0.3)
Text is fully opaque
opacity: 0.3
Text is also transparent 
/* rgba — only background is transparent */
background: rgba(6, 182, 212, 0.3);

/* opacity — entire element + children are transparent */
opacity: 0.3;

Example 3: Hover Fade Effect

Combine opacity with transition for smooth hover effects.

Hover to fade in
Hover to fade out 
.box {
  opacity: 0.6;
  transition: opacity 0.3s;
}
.box:hover { opacity: 1; }

Example 4: Image Opacity

Images respond to opacity just like any other element.

opacity: 1
opacity: 0.5 (hover to reveal)
opacity: 0.2
img { opacity: 0.5; }
img:hover { opacity: 1; }

Example 5: Layered Opacity Effect (Watermark)

Low-opacity overlays create watermark or badge effects.

SAMPLE
.watermark {
  position: absolute;
  opacity: 0.1;
  font-size: 4rem;

Example 6: Opacity with Text Overlay

Dark overlay with opacity improves text readability on images.

Text with dark overlay 
.overlay {
  background: rgba(0, 0, 0, 0.5);
  /* Text remains fully readable */
}

Example 7: Progressive Disclosure

Use opacity to de-emphasize secondary content while keeping it accessible.

Primary action
Secondary option
Tertiary option
.primary { opacity: 1; }
.secondary { opacity: 0.6; }
.tertiary { opacity: 0.35; }
