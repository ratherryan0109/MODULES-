# Mini Project: Image Hover Overlay Gallery

## Goal
Build a gallery of cards with background images that reveal a semi-transparent overlay with text on hover.

## Requirements
- 4 gallery cards in a flex row
- Each card has a background image
- A colored overlay (using `rgba()`) covers the image
- Overlay is invisible by default (`opacity: 0`)
- On hover, overlay fades in (`opacity: 1`) with a smooth transition
- Text (title and description) appears on the overlay
- Cards are responsive (wrap on smaller screens)

## Stretch Goals
- Add a second overlay effect (e.g., a button that slides up)
- Use `hsla()` instead of `rgba()` for the overlay
- Animate the overlay from bottom to top instead of fading

## Evaluation Criteria
- Correct use of `opacity` and `rgba()`
- Smooth CSS transitions
- Responsive layout
- Clean, well-structured code
