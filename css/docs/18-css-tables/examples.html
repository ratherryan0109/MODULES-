Example 3: Column Alignment

Use text-align to align different columns appropriately (numbers right, text left).

ItemQtyUnit PriceTotal

td:nth-child(2),
td:nth-child(3),

Example 6: Border Collapse vs Separate

border-collapse: collapse merges adjacent borders into one. separate keeps them apart.

collapse
AB
CD
separate (default)
AB
CD
.collapse { border-collapse: collapse; }  /* Single border */
.separate { border-collapse: separate; }  /* Double border */

Example 7: Responsive Table (Horizontal Scroll)

Wrap tables in a container with overflow-x: auto for horizontal scrolling on small screens.

IDNameEmailDepartmentLocationPhoneStatus
001John Doejohn@example.comEngineeringNew York555-0101Active
002Jane Smithjane@example.comDesignSan Francisco555-0102Active
 
.responsive-wrap {
  overflow-x: auto;
}
/* The table can now be wider than the screen
   without breaking the layout */
