# Mini Project: 3D Product Showcase with Cube

## Objective
Build a 3D product showcase that displays product features on different faces of a rotating cube.

## Requirements
1. Create a 3D cube with 6 faces (product views, features, specs)
2. Cube rotates on hover with mouse tracking
3. Navigation buttons to manually rotate to specific faces
4. Perspective creates realistic depth
5. Text on each face is readable

## Specs
- Use CSS 3D transform functions (rotateX, rotateY, translateZ)
- perspective on parent container
- transform-style: preserve-3d on cube
- backface-visibility for clean rendering
- Smooth transitions on face changes

## Stretch Goals
- Add carousel of multiple cubes
- Implement drag-to-rotate with JavaScript
- Add shadow/glow effects that react to rotation

## Evaluation Criteria
- 3D cube is mathematically correct
- Faces are properly positioned
- Rotation animation is smooth
- Text is readable on all faces
