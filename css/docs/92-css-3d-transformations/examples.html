CSS 3D Transformations — Practical Examples

1. RotateX

Rotation around X-axis (forward/backward tilt).

rotateX
transform: rotateX(45deg); /* parent: perspective: 600px */

2. RotateY

Rotation around Y-axis (door opening).

rotateY
transform: rotateY(45deg);

3. TranslateZ

Moving toward/away from viewer.

translateZ
transform: translateZ(100px);
