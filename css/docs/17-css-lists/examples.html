CSS Lists - Examples

Example 1: List-Style-Type Values

Change bullet/number style with list-style-type. Unordered and ordered lists both support many values.

Unordered
  • disc (default)
  • circle
  • square
  • none
Ordered
  1. decimal
  1. decimal-leading
  1. lower-alpha
  1. lower-roman
More ordered
  1. upper-alpha
  1. upper-roman
  1. lower-greek
  1. georgian
ul { list-style-type: square; }
ol { list-style-type: lower-alpha; }
ul { list-style-type: none; }  /* Hides bullets */

Example 2: Custom Bullets with list-style-image

Replace bullets with a custom image using list-style-image.

  • Custom bullet image
  • Blue circle from SVG
  • Scales with font size
ul {
  list-style-image: url('bullet.svg');
}

Example 3: Fully Custom Styled List (No Bullets)

Remove default bullets and add custom styling for a modern card-like list appearance.

  • Custom styled list item with left border accent
  • Clean, modern appearance without bullets
  • Hover effect can be added for interactivity
  • Great for navigation menus or feature lists
.custom-list {
  list-style: none;   /* Remove bullets */
  padding: 0;
}
.custom-list li {
  padding: 0.6rem 0.8rem;
  background: #f0f9ff;
  border-left: 4px solid #06b6d4;
  border-radius: 0 6px 6px 0;
}

Example 4: Horizontal Navigation List

Use display: flex on a list to create a horizontal navigation bar.

  • Home
  • About
  • Services
  • Portfolio
  • Contact
.nav-list {
  list-style: none;
  display: flex;
  gap: 0.5rem;
}
.nav-list li {
  background: #06b6d4;
  color: white;
  padding: 0.5rem 1.2rem;
  border-radius: 6px;
}

Example 5: List-Style Position

list-style-position controls whether the bullet is inside or outside the content flow.

outside (default)
  • Text wraps under the bullet on long lines
  • Bullet hangs left of content
inside
  • Text wraps to same indent as bullet
  • Bullet is inline with content
.outside { list-style-position: outside; }
.inside  { list-style-position: inside; }

Example 6: Nested List Styling

Style nested lists differently at each level using descendant selectors.

  1. First level (decimal)
    1. Second level (lower-alpha)
      1. Third level (lower-roman)
      2. Another third
    2. Second level item
  2. First level item
ol { list-style-type: decimal; }
ol ol { list-style-type: lower-alpha; }
ol ol ol { list-style-type: lower-roman; }
