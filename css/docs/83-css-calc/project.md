# Project: calc() Layout Builder

## Objective
Build an interactive layout builder that demonstrates the power of calc() for creating responsive designs. The tool should allow users to construct common layout patterns using calc() expressions.

## Requirements
- Interactive sidebar + main layout builder with adjustable sidebar width
- Fluid grid builder where users control column count and gap size using calc()
- Full-bleed section demonstration
- Fluid typography preview with adjustable min/max sizes
- Display the generated calc() CSS code for each layout
- Responsive layout showing how calc() adapts

## Stretch Goals
- Visual representation of the calc() formula
- Comparison page showing with vs without calc()
- CSS variable integration examples
- Save and share layout configurations

## Learning Outcomes
- Master calc() expressions for real-world layouts
- Combine calc() with CSS variables for dynamic systems
- Build fluid layouts without media queries
- Understand calc() operator rules and unit mixing
