CSS calc() Function

Combine units, create fluid layouts, and solve layout problems with arithmetic

Basic Unit Mixing

Percentage - Pixels

calc(100% - 40px)

The box fills its container minus 40px on each side.

width: calc(100% - 40px);

Half with Gap

calc(50% - 10px)

Two items can sit side by side with a 20px gap.

width: calc(50% - 10px);

Fluid Typography

calc(1rem + 0.5vw)

Font size scales with viewport but never below 1rem.

font-size: calc(1rem + 0.5vw);

Fluid Typography Example

This heading scales fluidly with the viewport width. Resize the browser to see it change.

Subtext also scales: calc(0.8rem + 0.3vw)

font-size: calc(1rem + 1vw);

Sidebar Layout

Fixed sidebar: 250px
Main content: calc(100% - 250px - 1rem)
.main { width: calc(100% - 250px - 1rem); }

Column Calculations

Three equal columns with gaps:

calc(100% / 3 - 1rem)
calc(100% / 3 - 1rem)
calc(100% / 3 - 1rem)

Two-thirds + one-third:

calc(100% / 3 * 2 - 0.5rem)
calc(100% / 3 - 1rem)

Header Height Compensation

calc(100vh - 60px)  — Content area fills remaining viewport after a 60px header
