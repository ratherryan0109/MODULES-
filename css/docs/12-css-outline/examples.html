CSS Outline - Examples

Example 1: Outline vs Border

Outline does not affect layout — it draws over the element. Border takes up space inside the box model.

Border (affects layout)
This border adds to my total size.
Outline (no layout change)
This outline overlays, no size change.
.border { border: 4px solid #ef4444; }  /* Adds to box size */
.outline { outline: 4px solid #06b6d4; } /* Overlays, no size change */

Example 2: Outline Styles

Outline supports the same style keywords as border.

solid
dashed
dotted
double
groove
ridge
.solid  { outline: 3px solid #06b6d4; }
.dashed { outline: 3px dashed #ef4444; }
.dotted { outline: 3px dotted #22c55e; }

Example 3: Outline Offset

outline-offset creates space between the element's border and the outline (can be negative).

outline with 12px offset — gap between edge and outline
outline with -5px offset (drawn inside)
.outside { outline: 3px dashed #8b5cf6; outline-offset: 12px; }
.inside  { outline: 3px solid #06b6d4; outline-offset: -5px; }

Example 4: Outline Width and Color

Outline width, style, and color can be set individually or as a shorthand.

2px red
6px green
4px amber
.thin  { outline: 2px solid #ef4444; }
.thick { outline: 6px solid #22c55e; }

/* Individual properties */
.outline {
  outline-width: 4px;
  outline-style: solid;
  outline-color: #f59e0b;
}

Example 5: Focus Outline (Accessibility)

Outlines are essential for keyboard accessibility. The :focus-visible pseudo-class targets keyboard focus only.

Tab through these buttons to see the focus outline:

button:focus-visible {
  outline: 3px solid #06b6d4;
  outline-offset: 3px;
}
/* Never do: outline: none — breaks keyboard navigation */

Example 6: Outline vs Box-Shadow for Visual Effects

Box-shadow can mimic outline but wraps around border-radius. Outline does not follow border-radius.

Outline (square corners)
Box-shadow (follows radius)
.outline { outline: 4px solid #06b6d4; }     /* Square corners */
.shadow  { box-shadow: 0 0 0 4px #ef4444; }  /* Follows border-radius */
