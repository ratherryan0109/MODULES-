Math Functions

calc()

calc(100% - 40px)

Perform arithmetic with mixed units.

width: calc(100% - 40px);

min()

min(100%, 200px)

Selects the smallest value.

width: min(100%, 200px);

max()

max(200px, 50%)

Selects the largest value.

width: max(200px, 50%);

clamp()

clamp(150px, 60%, 400px)

Constrains a value between min and max.

width: clamp(150px, 60%, 400px);
