# Mini Project: Complete Website Navigation System

## Goal
Build a full navigation system for a multi-page website with horizontal nav, dropdown submenus, and a responsive mobile menu.

## Requirements
- Main horizontal navigation bar with 5+ links
- At least one dropdown submenu (3+ items)
- Current page indicator using `.active` class
- Responsive: horizontal on desktop, hamburger menu on mobile
- Skip-to-content link for accessibility
- Sticky header behavior

## Stretch Goals
- Animate the dropdown with CSS transitions
- Add a search input inside the nav bar
- Use `aria-current="page"` for accessibility

## Evaluation Criteria
- Semantic HTML (`<nav>`, `<ul>`, `<li>`, `<a>`)
- Clean, organized CSS (no inline styles)
- Fully responsive design
- All navigation states styled (hover, active, current)
- Accessible markup
