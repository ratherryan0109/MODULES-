# Mini Project: Gradient-Based Landing Page

## Objective
Build a visually stunning landing page that uses CSS gradients as the primary design element. The page should showcase various gradient techniques including text gradients, overlays, and animated backgrounds.

## Requirements

### Sections
1. **Hero** - Full-viewport gradient background with text gradient heading, CTA button with gradient
2. **Features** - 3-4 cards with gradient borders or backgrounds
3. **Testimonials** - Cards with subtle gradient accents
4. **Pricing** - 3 pricing cards with gradient headers
5. **CTA Section** - Gradient background with overlay pattern
6. **Footer** - Dark gradient background

### Gradient Techniques Required
1. Linear gradient backgrounds
2. Radial gradient for spotlight effects
3. Text gradients on headings
4. Gradient overlays on placeholder images
5. Repeating gradient patterns (stripes or dots)
6. Gradient borders using `border-image` or pseudo-elements
7. Gradient buttons with hover transition
8. Animated gradient with `prefers-reduced-motion` fallback
9. Conic gradient in a decorative element
10. Multiple gradient layers combined

### Color Palette
Create a cohesive color palette using CSS custom properties:
- Primary gradient: 2-3 colors
- Secondary gradient: complementary
- Neutral gradients for cards/sections
- Ensure all text has sufficient contrast

### Technical Requirements
1. CSS custom properties for gradient colors
2. Smooth gradient transitions on interactive elements
3. Accessible contrast ratios on all text
4. Respect prefers-reduced-motion
5. Responsive design (mobile-first)
6. No external images (all visual using gradients)

## Deliverables
- `index.html` - landing page

## Bonus Challenges
- Create a gradient loading spinner
- Implement gradient-filled icons/SVGs
- Add a theme switcher that changes gradient colors
- Create a gradient-based progress bar
- Implement a glass-morphism card using gradient backgrounds + backdrop-filter

## Evaluation Criteria
- All 10 gradient techniques used correctly
- Cohesive color palette
- Text is readable over gradient backgrounds
- Animated gradients respect reduced motion
- Responsive at all breakpoints
- No external image dependencies
- Clean, maintainable CSS with custom properties
- Creative and visually appealing design
