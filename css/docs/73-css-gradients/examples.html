CSS Gradients

Example 1: Linear Gradients

Colors transition along a straight line. Use an angle or direction keyword.

to right

135deg

45deg

.ex1 { background: linear-gradient(to right, #06b6d4, #6366f1); }
.ex2 { background: linear-gradient(135deg, #06b6d4, #6366f1, #ec4899); }
.ex3 { background: linear-gradient(45deg, #f59e0b, #ef4444); }

Example 2: Multiple Color Stops

Add many stops for rainbow effects. Stops can have explicit positions.

.rainbow {
    background: linear-gradient(90deg, red, orange, yellow, green, blue, indigo, violet);
}
.stripes {
    /* Hard color stops create sharp transitions — great for striped patterns */
    background: linear-gradient(90deg,
        #06b6d4 0%, #06b6d4 25%,
        #f59e0b 25%, #f59e0b 50%,
        #ef4444 50%, #ef4444 75%,
        #6366f1 75%, #6366f1 100%
    );
}

Example 3: Radial Gradients

Colors radiate from a center point. Shapes can be circle or ellipse.

Circle

Ellipse

Offset center

.circle  { background: radial-gradient(circle, #06b6d4, #0e7490); }
.ellipse { background: radial-gradient(ellipse, #ec4899, #be185d); }
.glow    { background: radial-gradient(circle at top left, #f59e0b, transparent); background-color: #0f172a; }

Example 4: Conic Gradients

Colors rotate around a center point, like a color wheel or pie chart.

Color wheel

Pie chart

.wheel {
    background: conic-gradient(red, yellow, lime, aqua, blue, magenta, red);
}
.pie {
    background: conic-gradient(
        #06b6d4 0% 25%, #f59e0b 25% 50%,
        #ef4444 50% 75%, #6366f1 75% 100%
    );
    border-radius: 50%;
}

Example 5: Repeating Gradients (Patterns)

Gradient patterns that repeat automatically for striped or checkerboard effects.

Diagonal stripes

Dots

Ruled lines

.stripes { background: repeating-linear-gradient(45deg, #06b6d4, #06b6d4 8px, #0e7490 8px, #0e7490 16px); }
.dots    { background: repeating-radial-gradient(circle, #06b6d4, #06b6d4 6px, transparent 6px, transparent 12px); }
.lines   { background: repeating-linear-gradient(0deg, transparent, transparent 14px, rgba(6,182,212,0.15) 14px, rgba(6,182,212,0.15) 15px); }

Example 6: Text Gradient (background-clip: text)

Fill text with a gradient by clipping the background to the text shape.

Gradient Text

Multi-Color Heading

.gradient-text {
    background: linear-gradient(135deg, #06b6d4, #6366f1, #ec4899);
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
}

Example 7: Gradient Overlay

Layer a gradient over a background image or color for readability.

Overlay Text 
.overlay {
    background:
        linear-gradient(to bottom, rgba(0,0,0,0.1), rgba(0,0,0,0.8)),
        url('photo.jpg');
    /* Gradient layers on top of the image */
}
The gradient sits on top of the image. Used for hero banners and card captions.

Example 8: Animated Gradient

Animating the background-position creates a moving gradient effect.

.animated {
    background: linear-gradient(-45deg, #06b6d4, #6366f1, #ec4899, #f59e0b);
    background-size: 300% 300%;
    animation: shift 6s ease infinite;
}
@keyframes shift {
    0%   { background-position: 0% 50%; }
    50%  { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}
Consider prefers-reduced-motion for accessibility. Use @media (prefers-reduced-motion: reduce) to pause the animation.
