How to Style a Blog with Modern CSS

Sarah Miller · June 5, 2026 · 8 min read

Modern CSS gives us incredible tools for styling blogs. From CSS Grid for layouts to custom properties for theming, we can create beautiful reading experiences with minimal code.

Starting with Typography

Typography is the foundation of any blog. The right font choices can make reading effortless and enjoyable. For body text, serif fonts like Merriweather or Georgia provide excellent readability at longer lengths.

"Typography is the craft of endowing human language with a durable visual form." — Robert Bringhurst

Aim for a line length between 45 and 75 characters per line. Use the `ch` unit to set a comfortable max-width on your article container.

Working with Code Blocks

Blogs about code need well-styled code blocks. Here's an example:

:root {
  --ff-serif: 'Merriweather', Georgia, serif;
  --ff-sans: 'Inter', system-ui, sans-serif;
  --fs-body: 1.0625rem;
  --content-width: 65ch;
}

.article-content {
  font-family: var(--ff-serif);
  max-width: var(--content-width);
  line-height: 1.8;
}

Always ensure your code blocks have sufficient contrast against the background and include a monospace font stack.

Responsive Images

Blogs often include images. Make sure they are responsive and optimized. Use the loading="lazy" attribute for images below the fold.

Modern CSS makes blog styling enjoyable. Combine a solid typographic system with a consistent color palette, and you'll create a blog readers love.
