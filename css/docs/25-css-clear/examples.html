CSS Clear - Examples

1. Without Clear (Default Behavior)

.element { /* no clear set - defaults to clear: none */ }
Left
Left
Right
This content sits next to the floated elements because no clear is applied. It wraps around the floats.

2. Clear Left

.element { clear: left; }
Left
Left
Right
clear: left moves me below all left-floated elements, but I still wrap around right floats.
Notice this element cleared left floats but still sits next to the right float.

3. Clear Right

.element { clear: right; }
Left
Left
Right
clear: right moves me below all right-floated elements, but I still wrap around left floats.

4. Clear Both

.element { clear: both; }
Left
Left
Right
clear: both moves me below ALL floated elements (both left and right). This is the most commonly used clear value.

5. Clearfix on Parent Container

.parent::after { content: ''; display: table; clear: both; }

Item 2
Item 3

The parent above has a clearfix. Notice it has a visible background and border because the clearfix properly contains the floated children.
