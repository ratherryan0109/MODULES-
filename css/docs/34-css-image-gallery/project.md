# Mini Project: Portfolio Gallery Page

## Goal
Build a complete portfolio gallery page for a photographer or designer with filtering, lightbox, and responsive layout.

## Requirements
- CSS Grid gallery with `auto-fill`/`minmax` for responsiveness
- At least 12 images in the grid
- Filter buttons (All, Portraits, Landscapes, Events) that show/hide categories
- CSS-only lightbox overlay for full-size image preview
- Smooth hover effects (scale + shadow)
- Lazy loading on all images

## Stretch Goals
- Add caption overlays that slide up on hover
- Use CSS custom properties for theming
- Add a "Load More" button

## Evaluation Criteria
- Fully responsive grid layout
- Working CSS-only filter mechanism
- Accessible lightbox with close functionality
- Clean, organized code
- No JavaScript (unless absolutely necessary for filtering)
