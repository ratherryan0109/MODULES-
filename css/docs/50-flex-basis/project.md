# Mini Project: Responsive Card Grid with Flex Basis

## Overview
Build a responsive card grid using `flex-basis` to control wrapping and sizing.

## Requirements
1. Card grid using `flex: 1 1 300px` pattern
2. Cards auto-wrap as container is resized
3. Some cards with explicit basis, some with auto
4. Show difference between basis: 0 and basis: auto
5. Container has max-width 1200px

## Bonus
- Add basis controls to toggle between modes
- Show computed widths in px

## Evaluation
- Responsive wrapping: 30pts
- Basis usage: 30pts
- Visual demonstration: 20pts
- Code quality: 20pts
