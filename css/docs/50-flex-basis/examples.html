CSS Flex Basis - Examples

Example 1: flex-basis: auto (Default, Content-Sized)

Items size based on their content. flex-basis: auto falls back to width or content size.

Short
Medium text
Longer content here
Each item sized to its content
.item { flex: 0 1 auto; }
/* Each item is only as wide as its content */

Example 2: flex-basis: 0 (Equal Distribution)

Basis starts at 0, so all extra space is distributed equally. All items become the same width.

One
Two
Three
All equal width regardless of content
.item { flex: 1; }
/* Shorthand for flex: 1 1 0 — all items equal */

Example 3: flex-basis: 200px (Fixed + Grow)

Each item starts at 200px, then grows equally to fill remaining space.

200px + share
200px + share
200px + share
flex-basis: 200px each, then grow equally
.item { flex: 1 1 200px; }
/* Starts at 200px, grows to fill container */

Example 4: flex-basis: 30% (Percentage-Based)

Basis can be a percentage of the container (no grow/shrink here).

30%
30%
30%
Each item is 30% of container width
.item { flex: 0 1 30%; }
/* Takes up 30% of container (may wrap if needed) */

Example 5: Responsive Cards with flex-basis

Cards with flex-basis: 250px and flex-wrap: wrap create a responsive grid without media queries.

Card 1
Card 2
Card 3
Card 4
flex: 1 1 220px — shrink to 220px, grow to fill, wrap
.card {
  flex: 1 1 250px;
  /* Grows, shrinks, ideal width 250px */
}

Example 6: flex-basis vs width

When both are set, flex-basis wins in a flex context. width is ignored.

width: 100px (ignored in flex)
flex-basis wins
flex-basis: auto uses width
width used
/* flex-basis overrides width when set to a specific value */
.item { flex: 1 1 200px; width: 100px; }  /* → 200px basis */

/* flex-basis: auto falls back to width */
.item { flex: 0 1 auto; width: 150px; }   /* → 150px width */
