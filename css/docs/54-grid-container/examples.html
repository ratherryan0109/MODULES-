CSS Grid Container

Example 1: display: grid

A block-level grid container creates a grid formatting context. Children become grid items.

1
2
3
4
5
6
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 8px;
}

Example 2: display: inline-grid

An inline-level grid sits inline with text. The grid container only takes the width of its content.

Text before A B C text after — the grid flows inline. 

.container {
    display: inline-grid;
    grid-template-columns: repeat(3, 60px);
    gap: 4px;
}
inline-grid is useful for small UI elements like icon rows or inline toolbars.

Example 3: Explicit Columns with grid-template-columns

Define column widths explicitly. Mixed units (px, fr, %) create flexible layouts.

120px
1fr
2fr
.container {
    display: grid;
    grid-template-columns: 120px 1fr 2fr;
}

Example 4: Explicit Rows with grid-template-rows

Control row heights. The classic auto / 1fr / auto pattern creates a sticky footer.

Header — auto height
Main — 1fr fills remaining space
Footer — auto height
.container {
    display: grid;
    grid-template-columns: 1fr;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
}
The 1fr row absorbs all leftover space, pushing the footer to the bottom.

Example 6: minmax() for Responsive Tracks

Each track has a minimum size and a maximum size. Combined with auto-fit, this creates responsive grids without media queries.

1
2
3
4
5
6
.container {
    grid-template-columns: repeat(auto-fit, minmax(140px, 1fr));
}
Resize the browser. Items wrap automatically. auto-fit collapses empty tracks so items expand to fill space.
