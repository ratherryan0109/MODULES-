# Mini Project: Responsive Image Gallery Grid

## Overview
Build a responsive image gallery using grid container with `auto-fit` and `minmax()`.

## Requirements
1. Grid with `repeat(auto-fit, minmax(200px, 1fr))`
2. 12 image placeholders
3. 15px gap
4. Container max-width 1200px
5. Images should fill their cells

## Bonus
- Add `grid-auto-flow: dense` for masonry-like packing
- Use different sized images (span 2 columns for some)

## Evaluation
- Responsive grid: 30pts
- auto-fit/minmax usage: 25pts
- Visual layout: 25pts
- Code quality: 20pts
