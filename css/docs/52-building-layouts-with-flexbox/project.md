# Mini Project: Full Portfolio Page with Flexbox

## Overview
Build a complete personal portfolio page using multiple Flexbox layout patterns.

## Requirements
1. **Navigation**: Logo left, links center, CTA right
2. **Hero**: Full-width, centered content with background gradient
3. **Skills section**: Cards with equal heights, button at bottom
4. **Projects grid**: Responsive card grid that wraps
5. **Testimonials**: Media objects (avatar + text)
6. **Footer**: Sticky footer with columns
7. All sections use Flexbox (no Grid)
8. Fully responsive without media queries where possible

## Bonus
- Add animations on scroll
- Use auto margins for spacing
- Implement a "back to top" button

## Evaluation
- Correct use of flex patterns: 30pts
- Responsive design: 25pts
- Visual design: 25pts
- Code quality: 20pts
