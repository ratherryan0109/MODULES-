Example 3: Responsive Card Grid

Cards automatically wrap and resize using flex-wrap: wrap + flex: 1 1 220px.

Card One

Short description for the first card.

Card Two

This card has a longer description that takes up more vertical space before the button.

Card Three

Medium length content here.

.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}
.card {
  flex: 1 1 220px;
  display: flex;
  flex-direction: column;
}
.card button { margin-top: auto; }

Example 4: Sticky Footer Layout

The footer stays at the bottom even when content is short.

Site Header

Main content area. The footer stays pinned to the bottom because main { flex: 1 } fills all remaining space.

© 2026 Flexbox Layouts. All rights reserved.
.page {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
.page main { flex: 1; }
/* Footer is pushed to the bottom */

Example 7: Two-Column Sidebar Layout

Fixed sidebar + flexible content area — the most common dashboard pattern.

Sidebar
  • Dashboard
  • Analytics
  • Settings

Dashboard Content

The sidebar stays at 200px while the content fills the rest of the space.

.layout { display: flex; gap: 1rem; }
.sidebar { flex: 0 0 200px; }  /* Fixed 200px */
.content { flex: 1; }           /* Fills remaining space */
