CSS Loading Animations

1. Ring Spinner

A classic circular spinner using border, border-radius: 50%, and a rotating keyframe.

.spinner {
  width: 40px; height: 40px;
  border: 4px solid #e2e8f0;
  border-top-color: #3b82f6;
  border-radius: 50%;
  animation: spin 0.8s linear infinite;
}
@keyframes spin { to { transform: rotate(360deg); } }

2. Bouncing Dots

Three dots that scale up and down. Staggered delays create a cascading wave effect.

.dots span {
  animation: bounce 1.4s ease infinite both;
}
.dots span:nth-child(1) { animation-delay: -0.32s; }
.dots span:nth-child(2) { animation-delay: -0.16s; }
@keyframes bounce {
  0%, 80%, 100% { transform: scale(0); }
  40% { transform: scale(1); }
}

3. Progress Bar

An animated progress bar that fills from left to right. A striped variant adds a shimmer effect.

.progress-bar .fill {
  height: 100%;
  background: linear-gradient(90deg, #3b82f6, #06b6d4);
  animation: progress 2s ease infinite;
}
@keyframes progress {
  0% { width: 0; }
  100% { width: 100%; }
}

4. Skeleton Screen

A content placeholder with a shimmer animation that sweeps across the surface.

.skeleton-line {
  height: 12px;
  background: linear-gradient(
    90deg, #e2e8f0 25%, #f1f5f9 50%, #e2e8f0 75%
  );
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}

5. Pulse Ring

A radial pulse that scales up and fades out, like a radar ping. The second ring is offset by half the duration.

@keyframes pulseRing {
  0% { transform: scale(0.8); opacity: 0.5; }
  100% { transform: scale(1.4); opacity: 0; }
}
.pulse-ring {
  animation: pulseRing 1.5s ease infinite;
}
.pulse-ring-delayed {
  animation: pulseRing 1.5s ease infinite 0.75s;
}

6. Bar Loader

Vertical bars of varying heights that bounce using staggered delays.

.bar { width: 6px; border-radius: 3px; background: #3b82f6;
  animation: bounce 0.6s ease infinite both; }
.bar:nth-child(1) { animation-delay: 0s; height: 30px; }
.bar:nth-child(2) { animation-delay: 0.1s; height: 40px; }
.bar:nth-child(3) { animation-delay: 0.2s; height: 50px; }
.bar:nth-child(4) { animation-delay: 0.3s; height: 40px; }
.bar:nth-child(5) { animation-delay: 0.4s; height: 30px; }

@keyframes bounce {
  0%, 80%, 100% { transform: scaleY(0.4); }
  40% { transform: scaleY(1); }
}
