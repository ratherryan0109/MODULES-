# Mini Project: Loading Screen with Multiple States

## Objective
Build a full-page loading screen with multiple animated states: initial load, progress indicator, and completion transition.

## Requirements
1. Centered loading spinner (ring style)
2. Progress bar showing loading progress (animated width)
3. Status text that changes at intervals (opacity transition)
4. Skeleton preview of content behind loader
5. Smooth fade-out transition when loading completes
6. Accessible with aria-live region for status updates

## Stretch Goals
- Animated logo or brand mark in loader
- Different loading states (connecting, downloading, processing)
- Percentage counter (animated with CSS custom properties)
- Loading complete confetti/celebration

## Evaluation
- GPU-accelerated animations only
- Accessible loading indicators
- Smooth transitions between states
- Performant (no layout thrashing)
- Respects prefers-reduced-motion
