Grid Columns Examples

Interactive examples of grid-template-columns with different sizing strategies

1. Fixed Width Columns

Each column gets an exact pixel width. The total must fit within the container.

150px
200px
150px
.container {
    grid-template-columns: 150px 200px 150px;
}

2. The fr Unit — Flexible Columns

fr distributes remaining space proportionally. Here 1fr 2fr 1fr means the middle column gets twice the space.

1fr (25%)
2fr (50%)
1fr (25%)
.container {
    grid-template-columns: 1fr 2fr 1fr;
}

With a fixed column + fr:

200px fixed
1fr (flexible)
1fr (flexible)
.container {
    grid-template-columns: 200px 1fr 1fr;
}
/* The fr units split whatever is LEFT after the 200px column and gaps */

4. minmax() — Responsive Columns

Each column is at least 120px but can grow to fill available space equally.

min 120px
grows to 1fr
min 120px
grows to 1fr
.container {
    grid-template-columns: repeat(4, minmax(120px, 1fr));
}

5. auto-fill vs auto-fit — The Responsive Powerhouse

auto-fill — keeps empty track space

1
2
3
4
5
.container {
    grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
}
/* Resize the browser: empty track slots remain visible */

auto-fit — collapses empty tracks

1
2
3
4
5
.container {
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
}
/* Resize the browser: items stretch to fill the container */
Try it: Resize your browser horizontally. With auto-fill, you'll see empty track slots. With auto-fit, items expand to fill all space.

7. Named Grid Lines

.item { grid-column: sidebar / end; }

8. Spanning Columns

Items can span multiple columns using grid-column.
