# Mini Project: SCSS Themed Component Library

## Overview
Build a component library using SCSS with full theme support, mixins, and modular architecture.

## Requirements
1. Set up SCSS file architecture with partials
2. Create a theme system with variables (light/dark)
3. Build 4+ components using BEM in SCSS
4. Create responsive mixins
5. Use @each and maps for utility generation
6. Generate sourcemaps in development

## Components
1. Button (.btn) — with mixin for variants
2. Card (.card) — BEM with nesting
3. Navigation (.nav) — responsive with mixin
4. Form elements (.form) — with error states

## Deliverables
1. `scss/` — Complete SCSS file structure
2. `css/` — Compiled CSS output
3. `index.html` — Demo page
4. `package.json` — npm setup with build scripts
