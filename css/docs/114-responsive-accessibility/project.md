# Mini Project: Accessible Responsive Dashboard

## Overview
Build a responsive admin dashboard that remains fully accessible across all viewport sizes (320px to 1920px).

## Requirements
1. Responsive sidebar navigation (collapsible → hamburger on mobile)
2. Dashboard cards grid (auto-fill, responsive)
3. Data table with horizontal scroll on mobile
4. Form with responsive layout
5. Header with search input and user menu

## Accessibility Requirements
- All touch targets ≥ 44×44px
- Navigation keyboard accessible at all sizes
- Skip link present
- 400% zoom reflow test passes
- Relative units throughout
- No user-scalable=no
