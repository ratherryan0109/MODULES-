# Mini Project: Dashboard with Flex Shorthand

## Overview
Build a complete dashboard layout using ONLY the `flex` shorthand (no individual grow/shrink/basis).

## Requirements
1. Sidebar: `flex: 0 0 250px`
2. Main content area: `flex: 1`
3. Right panel: `flex: 0 0 200px`
4. Cards within main use `flex: 1 1 300px`
5. Header/footer: `flex: none`

## Bonus
- Add a toggle that switches between `flex: 1` and `flex: auto`
- Show the computed widths
