CSS Flex Shorthand - Examples

Example 1: flex: initial (0 1 auto)

The default. Items are content-sized, can shrink, but do not grow.

Short
Longer content here
Medium
Content-sized, shrink allowed, no grow
.item { flex: initial; }
/* flex-grow: 0, flex-shrink: 1, flex-basis: auto */

Example 2: flex: auto (1 1 auto)

Items are content-sized but can both grow and shrink to fill the container.

Content
Grows to fill
Takes space
Content-sized base, grows & shrinks
.item { flex: auto; }
/* flex-grow: 1, flex-shrink: 1, flex-basis: auto */

Example 3: flex: none (0 0 auto)

Items are fully inflexible — they stay at their content size regardless of container space.

Fixed
No Grow
No Shrink
No flexibility at all — content-sized, no grow, no shrink
.item { flex: none; }
/* flex-grow: 0, flex-shrink: 0, flex-basis: auto */

Example 4: flex: 1 (1 1 0) — Equal Columns

The most common pattern. All items get equal width regardless of content.

1
2
3
4
All items share equal space
.item { flex: 1; }
/* flex-grow: 1, flex-shrink: 1, flex-basis: 0 */

Example 5: Proportional (flex: 2 vs flex: 1)

Different values create proportional layouts — great for sidebars.

flex: 1
flex: 2
flex: 1
1 : 2 : 1 proportion
.sidebar { flex: 1; }  /* 25% */
.content { flex: 2; }  /* 50% (main area) */

Example 6: flex: 1 1 300px (Responsive Cards)

Combined with flex-wrap: wrap, this creates responsive cards without media queries.

Card
