# CSS Height & Width ‚Äî Cheatsheet
## Properties
- `width: 200px;` ‚Äî fixed width
- `height: 300px;` ‚Äî fixed height
- `min-width: 200px;` ‚Äî minimum width
- `max-width: 800px;` ‚Äî maximum width
- `min-height: 100px;` ‚Äî minimum height
- `max-height: 500px;` ‚Äî maximum height

## Units
- `px`, `em`, `rem`, `%`, `vw`, `vh`, `vmin`, `vmax`
- `auto` ‚Äî browser calculates
- `100%` ‚Äî full parent width/height

## Sizing Behavior
- `box-sizing: content-box` ‚Äî width excludes padding/border (default)
- `box-sizing: border-box` ‚Äî width includes padding/border

## Overflow
- `overflow: visible; hidden; scroll; auto;`
- `overflow-x` and `overflow-y` for individual axes
