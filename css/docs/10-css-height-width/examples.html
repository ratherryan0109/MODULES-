CSS Height & Width - Examples

Example 1: Fixed Width and Height

Exact pixel values give precise control over element dimensions.

width: 200px; height: 100px
.fixed {
  width: 200px;
  height: 100px;
}

Example 2: Percentage Width

Percentage widths are relative to the parent element's width.

width: 60% of parent
.half { width: 60%; }  /* 60% of parent's width */

Example 3: Max-Width (Responsive)

max-width combined with width: 100% creates responsive elements that cap at a maximum size.

max-width: 500px; width: 100% — grows to 500px max, shrinks on small screens
.responsive {
  max-width: 500px;
  width: 100%;  /* Fills container but never exceeds 500px */
}

Example 4: Min-Width and Max-Width Combined

min-width and max-width set a range the element stays within.

min: 150px, max: 400px, width: 50%
.constrained {
  min-width: 150px;
  max-width: 400px;
  width: 50%;
}

Example 5: Min-Height and Max-Height

Height constraints ensure minimum visibility while preventing excessive growth.

min-height
min-height: 100px
max-height
max-height: 80px (overflow hidden)
.min-h { min-height: 100px; }
.max-h {
  max-height: 80px;
  overflow: hidden;
}

Example 6: Viewport Units (vw / vh)

vw and vh are relative to the viewport: 1vw = 1% of viewport width, 1vh = 1% of viewport height.

width: 50vw (50% of viewport)
height: 15vh
.vw { width: 50vw; }  /* Half the viewport width */
.vh { height: 15vh; }  /* 15% of viewport height */

Example 7: Auto Width (Inline and Block)

Block elements default to width: auto (fill container). Inline elements shrink-wrap their content.

Block (auto)
Full width by default
Inline-block
Shrink-wrap
.block { display: block; }   /* width: auto → fills parent */
.inline { display: inline-block; }  /* shrinks to content */

Example 8: Box-Sizing Effect on Width

box-sizing changes how width is calculated. border-box includes padding and border.

content-box
width: 200px + 30px
Total: 230px
border-box
width: 200px inclusive
Total: 200px 
.content-box { box-sizing: content-box; }  /* width excludes padding */
.border-box { box-sizing: border-box; }    /* width includes padding */
