# Mini Project: Accessible Component Library

## Overview
Build a small accessible component library with 3+ interactive components. Each component must be fully keyboard accessible, have proper ARIA attributes, and work with screen readers.

## Required Components
1. **Accordion**: 3+ sections, supports multiple/ single expand
2. **Tab Panel**: 3+ tabs, arrow key navigation
3. **Modal Dialog**: Focus trap, Escape close, focus restoration

## Requirements
- Semantic HTML + ARIA attributes
- Keyboard navigation (Tab, Enter, Escape, Arrow keys)
- Visible focus indicators throughout
- CSS-only states where possible
- Support forced-colors mode
- Test with NVDA or VoiceOver

## Evaluation
- Component functionality: 30 pts
- ARIA usage correctness: 25 pts
- Keyboard navigation: 20 pts
- Focus management: 15 pts
- Code quality: 10 pts
