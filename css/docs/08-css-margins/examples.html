CSS Margins - Examples

Example 1: Margin on All Sides

A single value applies equal margin to all four sides.

margin: 30px (on all sides)
.box { margin: 30px; }  /* All sides */

Example 2: Margin Shorthand (1–4 Values)

Margin shorthand follows the TRBL pattern: top, right, bottom, left.

margin: 20px
All sides: 20px
margin: 10px 30px
Vertical: 10px, Horizontal: 30px
margin: 10px 20px 30px
Top: 10px, Sides: 20px, Bottom: 30px
margin: 10px 20px 30px 40px
Top Right Bottom Left
/* 1 value */  margin: 20px;          /* all sides */
/* 2 values */ margin: 10px 30px;      /* vert / horiz */
/* 3 values */ margin: 10px 20px 30px; /* top / sides / bottom */
/* 4 values */ margin: 10px 20px 30px 40px; /* TRBL */

Example 3: Margin Collapsing

Adjacent vertical margins collapse — the larger margin wins. Horizontal margins do not collapse.

Bottom margin: 30px
Top margin: 20px — collapsed to 30px

The 30px and 20px margins collapse to 30px (the larger value).

.first { margin-bottom: 30px; }
.second { margin-top: 20px; }
/* Actual gap between them: 30px (not 50px) */

Example 4: Auto Margins (Centering)

margin: 0 auto horizontally centers a block element within its container.

margin: 0 auto — centered horizontally
.center {
  max-width: 400px;
  margin: 0 auto;   /* Left/right auto = centering */
}

Example 5: Negative Margins

Negative margins pull an element in the opposite direction, overlapping adjacent elements or extending beyond the parent.

margin: 0 -1rem (wider than parent)
margin-top: -10px (overlaps parent)
.pull-out { margin: 0 -1rem; }     /* Extends beyond parent */
.overlap { margin-top: -10px; }    /* Pulls element upward */

Example 6: Individual Side Margins

Target specific sides with margin-top, margin-right, margin-bottom, margin-left.

margin-top: 20px, margin-left: 40px
margin-right: 30px
margin-bottom: 25px
.specific {
  margin-top: 20px;
  margin-right: 30px;
  margin-bottom: 25px;
  margin-left: 40px;
}

Example 7: Margin Between Elements

Use margins to create consistent spacing between sibling elements in a list or grid.

Item 1
Item 2
Item 3
Item 4
.list-item { margin-bottom: 10px; }
.list-item:last-child { margin-bottom: 0; }
/* Lobotomized owl alternative: */
.item + .item { margin-top: 10px; }
