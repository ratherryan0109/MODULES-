# Mini Project: Complete Company Website

## Goal
Build a complete company website homepage with a hero section, features grid, team section, testimonials, and footer — all responsive using modern CSS.

## Requirements
- Sticky header with logo and navigation links
- Hero section with a background image/gradient and CTA button
- Features section using `auto-fit`/`minmax` grid (4 cards responsive)
- Team section with member photos and info
- Testimonials section with quote cards
- Footer with 3-column layout (links, contact, social)
- Fully responsive (mobile-first or desktop-first)
- Smooth scroll for anchor links

## Stretch Goals
- Add a CSS-only mobile hamburger menu
- Use CSS custom properties for theming
- Add scroll-triggered animations using `@keyframes`

## Evaluation Criteria
- Semantic HTML5 structure
- CSS Grid for page layout, Flexbox for components
- Fully responsive across mobile, tablet, desktop
- Clean, organized CSS with comments
- No JavaScript required
