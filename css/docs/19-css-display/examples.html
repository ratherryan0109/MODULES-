CSS Display - Examples

Example 1: Block Elements

Block elements take up the full width available and stack vertically. They respect width and height.

Block 1 — takes full width
Block 2 — stacks below
Block 3 — with width: 60%
.block {
  display: block;  /* Default for div, p, h1-h6 */
  width: 100%;     /* Fills parent width */
  /* height is respected */
}

Example 2: Inline Elements

Inline elements flow horizontally like text. They ignore width and height.

Inline 1 Inline 2 Inline 3 Inline 4

Notice: width and height settings are ignored — they wrap naturally.

.inline {
  display: inline;   /* Default for span, a, strong */
  /* width and height are IGNORED */
  /* padding/margin only work horizontally */
}

Example 3: Inline-Block

Inline-block combines the best of both — flows inline but respects width and height.

IB 1
IB 2
IB 3
.ib {
  display: inline-block;
  width: 100px;    /* Works! */
  height: 60px;    /* Works! */
  /* Sits inline with text */
}

Example 4: Display: None vs Visibility: Hidden

display: none removes the element completely. visibility: hidden hides it visually but keeps its space.

display: none

You can't see me invisible — I'm removed from the flow.

visibility: hidden

You can't see me invisible — but my space is preserved (see the gap).

.gone  { display: none; }         /* Removed from flow */
.hidden { visibility: hidden; }    /* Invisible but takes space */

Example 5: Flexbox

display: flex creates a flexible layout container. Children align in a row by default, with powerful alignment options.

Flex 1
Flex 2
Flex 3
.flex-container {
  display: flex;
  gap: 0.5rem;
}
.flex-container div {
  flex: 1;  /* Equal distribution */
}

Example 6: CSS Grid

display: grid creates a two-dimensional layout with rows and columns.

Grid 1
Grid 2
Grid 3
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 0.5rem;
}

Example 7: Inline-Flex

display: inline-flex creates a flex container that behaves like an inline element.

This text has tagsinlineflex embedded inline-flex.

.inline-flex {
  display: inline-flex;
  gap: 0.3rem;
}
/* Flex behavior, but inline sizing */
