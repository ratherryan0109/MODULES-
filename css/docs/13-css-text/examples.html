CSS Text - Examples

Example 1: Text Alignment

text-align controls horizontal alignment of inline content within a block.

Left-aligned text is the default.
Center-aligned for headings and short blocks.
Right-aligned for subtle notes or dates.
Justified text stretches each line so both edges are straight. This is common in print but can create uneven spacing on the web when lines are short.
.left    { text-align: left; }
.center  { text-align: center; }
.right   { text-align: right; }
.justify { text-align: justify; }

Example 2: Text Decoration

text-decoration adds lines and effects to text. Shorthand: line, style, color.

underline
overline
line-through
wavy blue underline
.u       { text-decoration: underline; }
.strike  { text-decoration: line-through; }
.fancy   { text-decoration: underline wavy #06b6d4; }

Example 3: Text Transform

text-transform changes the capitalization of text without altering the HTML content.

uppercase
hello world → HELLO WORLD
lowercase
HELLO WORLD → hello world
capitalize
hello world → Hello World
none (default)
Original text preserved
.upper     { text-transform: uppercase; }
.lower     { text-transform: lowercase; }
.capital   { text-transform: capitalize; }

Example 4: Text Shadow

text-shadow adds depth with offset-x, offset-y, blur-radius, and color. Multiple shadows are comma-separated.

Basic Shadow
Glow Effect
.shadow { text-shadow: 2px 2px 4px rgba(0,0,0,0.3); }
.glow   { text-shadow: 0 0 8px rgba(6,182,212,0.6); }
.multi  { text-shadow: 1px 1px 0 #000, 2px 2px 0 #333; }

Example 5: Text Overflow (Ellipsis)

Truncate overflowing text with an ellipsis. Requires three properties working together.

This is a very long text that will be truncated with an ellipsis when it overflows the container.
.truncate {
  white-space: nowrap;      /* Prevent wrapping */
  overflow: hidden;         /* Hide overflow */
  text-overflow: ellipsis;  /* Show "..." */
}

Example 6: Text Indent, Letter Spacing, Word Spacing

Fine-grained spacing controls for typography.

text-indent: 2rem — First line is indented, subsequent lines wrap normally. This mimics traditional paragraph styling in print.
letter-spacing: 4px — spread out for headings
word-spacing: 8px — extra space between words
.indent  { text-indent: 2rem; }
.letters { letter-spacing: 4px; }
.words   { word-spacing: 8px; }

Example 7: Line Height and Vertical Alignment

line-height controls spacing between lines. vertical-align aligns inline elements vertically.

line-height: 1.2 — Tight spacing, good for headings.
Second line here shows the tightness.
line-height: 2 — Loose spacing, good for readability.
Second line here shows the spacious feel.
.tight  { line-height: 1.2; }  /* Compact */
.readable { line-height: 1.8; }  /* Airy, readable */
