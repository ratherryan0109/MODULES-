# Mini Project: Video Gallery Landing Page

## Objective
Build a responsive video gallery landing page that uses `aspect-ratio` for all media elements (video thumbnails, featured video embed, avatar circles, card images). No wrapper elements or padding hacks allowed. The gallery should look professional with uniform thumbnail sizes and a featured hero video.

## Requirements

1. **Featured video section**: Large 16:9 hero video embed at the top, filling the container width
2. **Video grid**: 3-column grid of video thumbnails, each with consistent 16:9 ratio using `aspect-ratio`
3. **Channel avatars**: Circular avatar images next to each video title (1:1 ratio with `border-radius: 50%`)
4. **Category banners**: Decorative banner images using 21:9 and 4:3 ratios at different breakpoints
5. **Loading state**: Skeleton loaders with 16:9 ratio and shimmer animation for when content is loading
6. **Card images**: In "popular channels" section, channel cards with 4:3 header images
7. **Responsive**: Grid changes from 3 columns → 2 columns → 1 column at breakpoints

## Specs

- Use `aspect-ratio` on all media containers (no padding-bottom hacks)
- Combine with `object-fit: cover` on all image/thumbnail elements
- Use `border-radius` with aspect-ratio for avatar circles
- Provide `@supports not (aspect-ratio: ...)` fallback for video grid items
- Include skeleton loaders with `aspect-ratio` and CSS animation
- All images use placeholder gradients (no actual image files needed)
- Logical properties for padding/margin preferred
