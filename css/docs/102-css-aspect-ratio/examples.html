2. Responsive Video Embed

aspect-ratio: 16 / 9 on the video element makes it responsive without any wrapper. Resize the browser to see it adapt.

▶ Video Player (16:9)
video, iframe {\n width: 100%;\n aspect-ratio: 16 / 9;\n}

3. Image Gallery with Consistent Ratios

All images in this grid use aspect-ratio: 4 / 3 with object-fit: cover for a perfectly uniform grid.

.gallery-item {\n width: 100%;\n aspect-ratio: 4 / 3;\n object-fit: cover;\n}
