CSS Skew — Practical Examples

1. SkewX

Skews along the X-axis — top/bottom edges shift horizontally.

skewX(15deg)
skewX(-15deg)
transform: skewX(15deg);
transform: skewX(-15deg);

2. SkewY

Skews along the Y-axis — left/right edges shift vertically.

skewY(15deg)
skewY(-15deg)
transform: skewY(15deg);

3. Skew Both Axes

Skews simultaneously on X and Y.

skew(10, 10)
transform: skew(10deg, 10deg);

4. Ribbon Effect (Counter-Skew)

Parent skewed, child counter-skewed to keep text readable.

Readable Text
.ribbon { transform: skewX(-20deg); }
.ribbon span { transform: skewX(20deg); } /* counter-skew */

5. Combining Skew with Other Transforms

Skew combined with rotate and scale.

Combined
transform: skewX(10deg) rotate(10deg) scale(1.1);

6. Skew with Different Origins

Same skew angle but different transform origins.

bottom right
