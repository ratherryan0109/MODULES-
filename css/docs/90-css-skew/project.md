# Mini Project: Hero Section with Diagonal Dividers

## Objective
Build a hero section with skewed background panels and diagonal section dividers using CSS skew.

## Requirements
1. Hero section with skewed background strip (skewX)
2. Text content counter-skewed for readability
3. Diagonal section divider between hero and content
4. CTA button with subtle skew on hover
5. Multiple panels with alternating skew directions

## Specs
- Use skewX for horizontal diagonal effects
- Counter-skew all text content
- Smooth transitions on hover states
- Responsive — skew should work on mobile

## Stretch Goals
- Add animated skew on scroll using Intersection Observer
- Create a fold/accordion effect using skew
- Use skewY for vertical diagonal sections

## Evaluation Criteria
- Skew effects are visually consistent
- Text remains fully readable
- Counter-skew is correctly applied
- Design is responsive
