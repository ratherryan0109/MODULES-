1. Holy Grail Layout (CSS Grid)

A classic page structure using grid-template-areas with header, nav, main, and footer.

Header
Navigation
Main Content — takes remaining space with 1fr
Footer
.page {\n display: grid;\n grid-template:\n "header header" auto\n "nav main" 1fr\n "footer footer" auto\n / 180px 1fr;\n min-height: 100vh;\n}

4. Card Deck (Grid + Flexbox Hybrid)

Grid for the card container, Flexbox inside each card for content layout. Equal height cards automatically.

Short description.

Card 2

A much longer description that takes more vertical space. This shows how flex:1 on the body keeps cards aligned.

.card-deck { display: grid; grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); gap: 16px; }\n.card { display: flex; flex-direction: column; }\n.card-body { flex: 1; display: grid; gap: 8px; align-content: start; }

5. Centering with Grid

The simplest way to center anything: place-items: center.

Centered!
.container {\n display: grid;\n place-items: center;\n}

6. Flexbox Wrapping Toolbar

Toolbar items that wrap naturally when space is tight. No media queries needed.

File Edit View Insert Format Tools Table Help
.toolbar {\n display: flex;\n flex-wrap: wrap;\n gap: 8px;\n}

7. Sidebar + Content (Flexbox)

Fixed sidebar with fluid main content. Note min-width: 0 on the content to prevent overflow.

Sidebar (200px fixed)
Main content fills remaining space with flex: 1.
.layout { display: flex; gap: 16px; align-items: flex-start; }\n.sidebar { flex: 0 0 200px; }\n.content { flex: 1; min-width: 0; }
