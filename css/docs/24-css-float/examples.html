CSS Float - Examples

1. Text Wrapping (Left Float)

img { float: left; margin-right: 15px; }
📷 Image

This text wraps around the floated image on the left. The float property was originally designed for this exact purpose - allowing text to flow around an element. The image is taken out of the normal flow, but unlike absolute positioning, text still wraps around it. This is great for articles, blog posts, and any content where you want images integrated with text.

2. Text Wrapping (Right Float)

img { float: right; margin-left: 15px; }
📷 Image

This text wraps around the floated image on the right. Floats work equally well on both sides. Right floats are commonly used for pull quotes, sidebars, or images that should appear on the right side of the text. The margin ensures there's always space between the floated element and the wrapping text.

4. Drop Cap with Float

p::first-letter { float: left; font-size: 3.5em; line-height: 0.8; margin-right: 6px; }

Float is perfect for creating drop caps - a decorative typographic effect where the first letter of a paragraph is enlarged and "drops" into the text. By floating the first letter, the rest of the text wraps around its right and bottom sides, creating a professional, magazine-style appearance.

6. Pull Quote with Float

.pull-quote { float: right; width: 200px; border-left: 4px solid #4a90d9; }
"Float is perfect for pull quotes that highlight key excerpts from the text."

Pull quotes are another classic float use case. A pull quote is a key phrase or excerpt pulled from the text and displayed more prominently. Floats allow the pull quote to sit beside the main text while the surrounding content flows around it naturally. This creates a visually engaging layout commonly seen in magazines and editorial designs.
