CSS Clip-Path

Example 1: Basic Shapes — Circle & Ellipse

clip-path cuts an element into a shape. Only the visible area is painted; the rest is hidden.

circle()
circle(45% at center)
ellipse()
ellipse(35% 50%)
.circle  { clip-path: circle(45% at center); }
.ellipse { clip-path: ellipse(35% 50% at center); }

Example 2: Inset Shape with Rounding

inset() clips from each edge. Add round for rounded corners.

inset(10%)
Even padding all sides
inset rounded
inset(10% 20% round 16px)
Pill shape
inset(20px 40px round 50%)
.even    { clip-path: inset(10%); }
.rounded { clip-path: inset(10% 20% round 16px); }
.pill    { clip-path: inset(20px 40px round 50%); }

Example 5: Hover Morphing

Transition clip-path smoothly on hover for interactive shape morphing.

.morph {
    clip-path: circle(40%);
    transition: clip-path 0.4s;
}
.morph:hover { clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%); }

Example 6: Animated Morphing

Keyframe animations can morph between shapes. Both shapes must have the same number of polygon points.

Circle → Diamond → Hexagon → Star → repeat

@keyframes morph {
    0%, 100% { clip-path: circle(50% at center); }
    25%      { clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%); }
    50%      { clip-path: polygon(25% 0%, 75% 0%, 100% 50%, 75% 100%, 25% 100%, 0% 50%); }
}
Only morph between shapes with the same number of polygon points for smooth transitions.

Example 7: Text Wrapping with shape-outside

Combine clip-path with shape-outside to make text flow around a clipped image.

The combination of clip-path and shape-outside creates magazine-style text wrapping. clip-path makes the image visually circular, while shape-outside tells the text where the boundary is. Both use the same shape function (circle(50%)) for consistency. This technique works for circular avatars, hexagonal featured images, and other creative editorial layouts. Text naturally fills the space beside the clipped image.

.wrapped {
    float: left;
    clip-path: circle(50%);
    shape-outside: circle(50%); /* Text follows the same shape */
    margin-right: 16px;
}
