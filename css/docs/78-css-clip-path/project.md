# Project: Creative Clip-Path Portfolio

## Objective
Build a visually engaging portfolio page that uses CSS clip-path extensively for section dividers, image presentations, hover effects, and animated transitions.

## Requirements
- Hero section with a large clipped image (hexagon, polygon, or custom shape)
- At least 3 distinct section dividers using different clip-path shapes
- Portfolio grid where each project card uses a different clip-path shape
- Hover morphing effects on cards (circle → polygon with matching vertex count)
- Animated shape cycling on at least one element
- Text wrapping around a clipped image using shape-outside
- Display the clip-path CSS values used

## Stretch Goals
- Interactive clip-path generator built into the page
- Mouse-following clip-path reveal
- Video background with clip-path
- Clip-path animation on scroll (Intersection Observer)

## Learning Outcomes
- Master all clip-path shape functions
- Create smooth animated morphing effects
- Use shape-outside for text wrapping
- Build creative section layouts with clip-path dividers
