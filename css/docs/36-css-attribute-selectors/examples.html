CSS Attribute Selectors - Examples

Example 1: [attr] — Has Attribute

Selects elements that have a specific attribute (regardless of value).

Hover over me — styled with [title]

No title attribute — not styled

[title] {
  cursor: help;
  border-bottom: 1px dashed gray;
}

Example 2: [attr="value"] — Exact Match

Selects elements with an exact attribute value.

Active — green
Inactive — gray
Pending — yellow
Error — red
[data-status="active"] { background: #10b981; }
[data-status="inactive"] { background: #94a3b8; }
[data-status="pending"] { background: #f59e0b; }

Example 3: [attr^="value"] — Starts With

Targets external links (^="http") and specific file types.

  • External link
  • Internal link
  • GitHub repo
a[href^="http"]::after { content: " ↗"; }
a[href*="github"] { color: #333; }

Example 4: [attr$="value"] — Ends With

Target files by extension — perfect for download links.

  • Download Report (PDF)
  • Download Document (DOCX)
  • Download Sheet (XLSX)
a[href$=".pdf"] { color: red; font-weight: bold; }
a[href$=".docx"] { color: blue; }

Example 5: [attr*="value"] — Contains Substring

Matches attributes containing a substring anywhere in the value.

  • opencode on GitHub
  • Project on GitHub
  • Project on GitLab
a[href*="github"] { color: #333; }

Example 6: [attr|="value"] — Hyphen-Separated

Matches language codes — en matches en-US and en-GB.

Hello! This is American English.

Cheers! This is British English.

Bonjour ! Ceci est du français.

Guten Tag! Das ist Deutsch.

[lang|="en"] { border-left: 4px solid blue; }
[lang|="fr"] { border-left: 4px solid red; }

Example 7: [attr~="value"] — Space-Separated

Matches a whole word within a space-separated list (like classes).

This text is highlighted.

This is marked as important.

This is both highlighted AND important.

[class~="highlight"] { background: yellow; }
[class~="important"] { font-weight: bold; color: red; }

Example 8: Form Input Styling

Attribute selectors make form styling a breeze.

input[required] { border-left: 3px solid red; }
input[type="email"] { border-color: blue; }
input[type="submit"] { background: cyan; color: white; }
