# Mini Project: Smart Link Styling System

## Goal
Build a page that automatically styles all links based on their href attributes — showing icons, colors, and tooltip info without adding any classes.

## Requirements
- External links show an arrow icon and open in new tab
- PDF links show a PDF icon and are styled red
- Document links (.doc, .docx) show a Word icon in blue
- Image links (.jpg, .png, .gif) show an image icon
- Email links (mailto:) show an email icon
- Internal links have no special icon
- Use only attribute selectors (no classes on links)

## Stretch Goals
- Add a counter showing how many of each link type exist
- Style visited links differently
- Add `title` attribute tooltips automatically using CSS `content`

## Evaluation Criteria
- No JavaScript used
- No classes on links — only attribute selectors
- All 6 link types are distinguished
- Clean, organized CSS with comments
- Responsive and readable
