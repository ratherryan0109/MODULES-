# Mini Project: Action Button Toolbar

## Objective
Create a toolbar with action buttons using inline-flex that can sit within a content area.

## Requirements
1. Use `display: inline-flex` for the toolbar
2. Include buttons: Save, Edit, Delete, Share
3. Add proper `gap` spacing between buttons
4. Style buttons with hover effects
5. Place the toolbar after a paragraph of text (it should sit inline or below)

## Layout
```
Lorem ipsum dolor sit amet, consectetur adipiscing elit.

[Save] [Edit] [Delete] [Share]

More content continues here...
```

## Specifications
- Buttons should have consistent padding
- Each button should have a distinct color (green, blue, red, gray)
- Toolbar should have a subtle background and border
- Gap between buttons: 0.5rem

## Bonus Challenges
- Add icons to buttons (use emoji or SVG)
- Make the toolbar sticky on scroll
- Add a responsive breakpoint where buttons stack vertically

## Evaluation Criteria
- Correct use of inline-flex
- Proper gap spacing
- Clean button styling
- Responsive behavior
- Hover/focus states for accessibility
