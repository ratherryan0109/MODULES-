Example 5

Multi-Column Layout

Multi-column layout makes text flow naturally between columns, like a newspaper. This is ideal for article content.

The number of columns changes with viewport width: 1 column on mobile, 2 on tablet, 3 on desktop.

Use column-gap and column-rule to style the space between columns.

Multi-column is great for long-form content where you want to maximize readability.
