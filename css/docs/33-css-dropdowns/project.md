# Mini Project: Multi-Level Navigation System

## Goal
Build a complete website navigation bar with multi-level dropdowns, responsive behavior, and smooth animations.

## Requirements
- Horizontal nav bar with 5 top-level items
- At least 2 items have dropdown submenus (with 4+ items each)
- One submenu has a second-level nested dropdown
- Animated transitions (fade + slide) on all dropdowns
- Responsive: dropdowns become accordion-style on mobile
- Active/current page indicator

## Stretch Goals
- Add a mega menu dropdown
- Use `@media` queries to switch between hover and click behavior on touch devices
- Add keyboard navigation support with `:focus-within`

## Evaluation Criteria
- Semantic, accessible HTML structure
- Clean CSS with no JavaScript
- Smooth animations
- Responsive across device sizes
- All dropdown states work reliably
