# Mini Project: Simple Grid Layout

## Overview
Build a simple page layout using CSS Grid with header, sidebar, main, and footer.

## Requirements
1. 3-column grid: 200px sidebar, 1fr main, 200px sidebar
2. Header spans all columns
3. Footer spans all columns
4. Min-height 100vh
5. Responsive: sidebars stack on mobile

## Bonus
- Use named grid areas
- Add a nested grid in the main content

## Evaluation
- Grid structure: 30pts
- Spanning items: 25pts
- Responsive: 25pts
- Code quality: 20pts
