# Mini Project: BEM Component Library

## Overview
Build a small UI component library using strict BEM naming methodology. The library should include documentation showing proper BEM usage.

## Required Components
1. Button (.btn) — primary, secondary, small, large, disabled
2. Card (.card) — with title, body, actions, featured modifier
3. Navigation (.nav) — with items, links, active state
4. Form (.form) — with inputs, labels, error states
5. Modal (.modal) — overlay, content, title, actions, close button

## Requirements
- All components use BEM naming
- No element-of-element nesting
- Independent blocks composed together
- Documentation showing HTML structure with BEM classes
- Preprocessor nesting (Sass) for clean source

## Deliverables
1. `components/` — CSS files for each component
2. `index.html` — Demo page with all components
3. `README.md` — BEM usage guide for the library

## Evaluation
- Correct BEM naming: 30 pts
- No element-of-element errors: 20 pts
- Proper block composition: 20 pts
- Component documentation: 15 pts
- Code quality: 15 pts
