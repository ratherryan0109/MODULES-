CSS Position - Examples

1. Static Positioning

Default position. Elements flow naturally in the document.

.static-box { position: static; }
I am static (default)
I am also static
Static elements stack vertically

2. Relative Positioning

Offset from normal position. Original space is preserved.

.relative-box { position: relative; top: 20px; left: 30px; }
Normal element
I'm shifted 20px down, 30px right
Notice the gap where relative box was

3. Absolute Positioning

Positioned relative to nearest positioned ancestor.

.container { position: relative; }
.box { position: absolute; top: 20px; right: 20px; }
This is the container (position: relative)
Absolutely positioned
The absolute box is removed from normal flow and positioned inside its positioned ancestor.

4. Fixed Positioning

Positioned relative to the viewport. Stays in place during scroll.

.fixed-header { position: fixed; top: 0; left: 0; width: 100%; }
📌 I'm a fixed header - scroll to see me stay!
Click the button above to see the fixed header. The page content scrolls behind it.

5. Sticky Positioning

Hybrid of relative and fixed. Sticks when scroll reaches threshold.

.sticky-header { position: sticky; top: 0; }

Content for Section A

Content for Section B

Content for Section C
Each header sticks until the next one pushes it.

6. Overlay / Modal with Fixed + Absolute

Common pattern using fixed overlay and absolute positioned content.

.overlay { position: fixed; }
.overlay-content { position: relative; }
