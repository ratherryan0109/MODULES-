3. place-items: center (Center Everything)

Shortcut to center all items both horizontally and vertically in one line.

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 80px);
    place-items: center; /* Shorthand for align-items + justify-items */
}

4. justify-self (Per-Item Override)

Individual items can override the container's alignment. Here items 1 and 3 override the center alignment.

.container { place-items: center; }
.item-1 { justify-self: start; }
.item-2 { /* Uses container's center */ }
.item-3 { justify-self: end; }

6. place-content: center (Center Whole Grid)

Center the entire grid within the container both horizontally and vertically.

1
2
3
4
.container {
    grid-template-columns: repeat(2, 100px);
    grid-template-rows: repeat(2, 60px);
    place-content: center; /* Centers grid in container */
    min-height: 250px;
}
💡 place-content: center on the container is the simplest way to center a grid within its container.
