CSS Combinators - Examples

1. Descendant Combinator (space)

article p { color: #4a90d9; } /* Any p inside article */

This paragraph is inside an article. It's styled by the descendant combinator.

This paragraph is nested inside a div within the article. It's also a descendant.

This paragraph is outside the article and is NOT styled.

2. Child Combinator (>)

.parent > p { background: #c8e6c9; } /* Direct children only */

I'm a direct child - styled!

I'm nested inside a div - NOT a direct child, not styled.

I'm also a direct child - styled!

3. Adjacent Sibling Combinator (+)

h3 + p { color: #e94560; font-weight: bold; }

Heading

I'm immediately after h3 - styled with adjacent sibling!

I'm the second paragraph - NOT styled.

Another Heading

I'm also immediately after an h3 - styled!

4. General Sibling Combinator (~)

h3 ~ p { border-left: 3px solid #6c5ce7; }

Section Title

First sibling paragraph - styled with border.

Second sibling paragraph - also styled!

Some div in between

Third paragraph after the div - still styled as general sibling!

5. Combined Combinators

.card > .header + .content { background: #f0f0f0; }
Card Header
This content is adjacent to header within card - styled!
Footer - not styled.
This combines child (>) and adjacent sibling (+) for precise targeting.
