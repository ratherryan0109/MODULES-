CSS Pseudo-elements - Examples

1. ::before and ::after (Content Insertion)

.link::before { content: "→ "; }
.link::after { content: " ←"; }
Clickable Link

2. ::before and ::after (Decorative Elements)

.box::before, .box::after { content: ''; position: absolute; ... }
I have decorative circles in corners!

3. ::first-letter (Drop Cap)

p::first-letter { font-size: 3.5em; float: left; }

This is a paragraph with a decorative drop cap. The first letter is enlarged and floats to the left, creating a classic typographic effect often seen in books and magazines.

4. ::first-line

p::first-line { font-size: 1.3em; font-weight: bold; }

This is the first line of a paragraph. It is styled differently using the ::first-line pseudo-element, which applies styles only to the first line based on the current viewport width. When the text wraps, only the first line is affected.

5. ::selection (Custom Highlight)

::selection { background: #4a90d9; color: white; }

Select (highlight) this text to see custom selection colors! The background becomes blue and text becomes white.

6. ::placeholder

input::placeholder { color: #999; font-style: italic; }

7. ::marker (List Style)

li::marker { color: #e94560; font-size: 1.2em; }
  • Custom bullet color and size
  • Each marker is styled individually
  • No extra HTML elements needed!

8. Blockquote with Decorative Quotes

blockquote::before { content: '"'; }
blockquote::after { content: '"'; }

Pseudo-elements are one of the most powerful features in CSS, allowing you to add decorative content without modifying your HTML.
