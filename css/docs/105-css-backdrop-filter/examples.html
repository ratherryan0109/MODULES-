CSS Backdrop Filter — Interactive Examples

🍷 Frosted Navbar
Home Products About

0. Frosted Navigation Bar

A fixed navbar with backdrop-filter: blur(10px). Scroll the page to see the blur effect as content passes behind.

.navbar {\n background: rgba(30, 30, 46, 0.7);\n backdrop-filter: blur(10px);\n -webkit-backdrop-filter: blur(10px);\n}

1. Glassmorphism Card

The classic glass effect with backdrop-filter: blur(12px) over a gradient background.

Glass Card

This card has a frosted glass effect. The background behind it is blurred, creating depth and visual interest.

.glass-card {\n background: rgba(255, 255, 255, 0.1);\n backdrop-filter: blur(12px);\n border: 1px solid rgba(255, 255, 255, 0.2);\n}

2. Dark Glass (Dark Mode)

Dark theme glass with brightness(0.8) added for extra depth.

Dark Glass

Perfect for dark mode interfaces. The brightness filter adds depth to the frosted effect.

.dark-glass {\n background: rgba(0, 0, 0, 0.3);\n backdrop-filter: blur(12px) brightness(0.8);\n}

3. Modal Overlay with Blur

Click to show a modal with backdrop-filter: blur(4px) on the overlay.

Modal Dialog

Background is blurred with backdrop-filter, maintaining context.

.modal-overlay {\n background: rgba(0, 0, 0, 0.3);\n backdrop-filter: blur(4px);\n}

4. Vibrant Glass with Saturation

Extra vibrancy with saturate(1.8) combined with blur.

Vibrant Glass

The saturate filter makes background colors more vivid, creating a striking visual effect.

.vibrant-glass {\n backdrop-filter: blur(16px) saturate(1.8);\n}

5. Hover-Activated Glass

Hover over the box to activate the backdrop-filter with a smooth transition.

Hover me

The glass effect activates on hover. Note: animating backdrop-filter can be performance-intensive.

.glass {\n transition: backdrop-filter 0.3s;\n}\n.glass:hover {\n backdrop-filter: blur(8px);\n}
