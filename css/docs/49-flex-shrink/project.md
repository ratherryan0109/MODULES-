# Mini Project: Responsive Dashboard Skeleton

## Overview
Build a dashboard skeleton with fixed-width sidebars that don't shrink.

## Requirements
1. Left sidebar: 250px fixed (flex-shrink: 0)
2. Right sidebar: 200px fixed (flex-shrink: 0)
3. Main content: flexible (flex: 1)
4. Header and footer: full width, fixed height
5. Responsive: at 768px, sidebars become top/bottom bars

## Bonus
- Add content that triggers flex-shrink behavior
- Show min-width: 0 effect on shrinking

## Evaluation
- Fixed sidebars: 25pts
- Flexible main area: 25pts
- Responsive: 25pts
- Code quality: 25pts
