Example 1: All flex-shrink: 1 (Default, Equal Shrink)

Each item has flex-basis: 200px (600px total) but the container is only 450px. All items shrink equally.

Example 2: Mixed Shrink (Some Resist, Some Yield)

Item 1 has flex-shrink: 0 and stays at 200px. Others shrink more to compensate.

Example 4: Practical Fixed Sidebar

A common pattern: fixed-width sidebar that never shrinks + flexible content area.

Fixed Sidebar
200px
Flexible Content
.sidebar { flex: 0 0 200px; }  /* flex-shrink: 0 — never shrinks */
.content { flex: 1; }           /* flex-shrink: 1 — can shrink */

Example 5: Preventing Image Shrink

Images in flex items can get squished. Use flex-shrink: 0 or min-width: 0.

No shrink
This long text will wrap because the item can shrink
.no-shrink { flex: 0 0 auto; }        /* Content-sized, no shrink */
.can-shrink { flex: 1 1 200px; }      /* Can shrink if needed */
Tip: Use flex-shrink: 0 on elements that should maintain their size (sidebars, images, icons).
