# Mini Project: Technical Documentation Page

## Goal
Build a technical documentation page with auto-numbered sections, subsections, a table of contents, and numbered code examples.

## Requirements
- Document with 3+ main chapters, each with 2-3 subsections
- Auto-numbered chapters (1, 2, 3) and subsections (1.1, 1.2, 2.1, etc.)
- Table of contents that mirrors the numbering
- Auto-numbered code examples (Listing 1, Listing 2, etc.)
- Auto-numbered figures

## Stretch Goals
- Use `counters()` for 3-level nesting
- Add cross-references using CSS content
- Style the table of contents as a sticky sidebar

## Evaluation Criteria
- CSS counters used for all numbering (no hardcoded numbers)
- Nested numbering works correctly
- Clean, semantic HTML structure
- Responsive layout
- No JavaScript for numbering
