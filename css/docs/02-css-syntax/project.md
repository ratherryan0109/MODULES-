# Mini Project: CSS Syntax Reference Page

## Project Overview
Create a beautifully designed CSS syntax reference page that visually demonstrates all the syntax rules you've learned. This page will serve as a quick reference guide for yourself and others.

## Requirements

### Content Sections
1. **CSS Rule Anatomy** — Annotated diagram of a CSS rule
2. **Selector Types** — Examples of element, class, ID, attribute, and pseudo-class selectors
3. **Declaration Examples** — Common property-value pairs organized by category
4. **Shorthand vs Longhand** — Comparison table with examples
5. **Common Mistakes** — A "Don't Do This" section with error examples
6. **Vendor Prefixes** — When and how to use them

### Design Requirements
- Use CSS to style the page attractively
- Code examples must be in styled code blocks with proper syntax highlighting
- Use a color-coded system (e.g., blue for selectors, green for properties, red for values)
- Include a responsive layout

### HTML Structure
```
syntax-reference/
├── index.html
└── style.css
```

## Grading Criteria
- Correct CSS syntax in your stylesheet (25%)
- Visual design and layout quality (25%)
- Code examples are properly formatted and color-coded (25%)
- Content accuracy and completeness (25%)

## Stretch Goals
- Add interactive tabs for different categories
- Include a "Copy to Clipboard" button for code examples
- Add a search/filter functionality
- Create a dark mode toggle
