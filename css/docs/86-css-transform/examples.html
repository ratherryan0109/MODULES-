CSS Transform — Practical Examples

1. Translate

Moves the element from its original position.

translateX
translate(x,y)
transform: translateX(60px);
transform: translate(40px, 40px);

2. Rotate

Rotates around the transform-origin (default: center).

rotate 45°
rotate 180°
origin: top left
transform: rotate(45deg);
transform-origin: top left;

3. Scale

Scales the element in size.

scale(1.2)
scale(0.8)
scale(1.5, 0.5)
transform: scale(1.2);
transform: scale(1.5, 0.5); /* X, Y */

4. Skew

Skews the element along X and Y axes.

skewX
skewY
skew(x, y)
transform: skewX(15deg);
transform: skew(10deg, 10deg);

5. Combined Transforms

Multiple functions in one transform property. Applied right-to-left.

Combined
transform: translateX(40px) rotate(45deg) scale(1.2);

Same rotation (45deg) with different transform-origin values.
