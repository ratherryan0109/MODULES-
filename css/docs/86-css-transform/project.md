# Mini Project: Interactive Portfolio Card

## Objective
Build an interactive portfolio/identity card that uses CSS transforms for multiple visual effects.

## Requirements
1. Create a card with: avatar/photo placeholder, name, title, social icons, and a bio section
2. Hover effect: card lifts and rotates slightly (translateY + rotate)
3. Avatar scales up on hover with a different transform-origin
4. Social icons slide up from below on card hover (translateY)
5. Bio text reveals with a combination of scale and opacity
6. Click effect on the card using :active

## Specs
- Use at least 4 different transform functions
- Demonstrate transform-origin change
- Combine transforms with transition for smooth animation
- Must work on both desktop and mobile

## Stretch Goals
- Add a 3D card tilt effect using rotateX/rotateY with mouse movement (JS)
- Create a flip animation to reveal a back side
- Add a ripple effect on button click using scale

## Evaluation Criteria
- Correct transform function usage
- Smooth transitions
- Creative combination of transforms
- Proper transform-origin usage
