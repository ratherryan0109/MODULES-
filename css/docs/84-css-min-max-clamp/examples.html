CSS min(), max(), and clamp()

Responsive, constraint-based CSS without media queries

Basic Comparison Functions

min() — Selects Smallest

min(100%, 250px)

Fills container but never exceeds 250px.

width: min(100%, 250px);

max() — Selects Largest

max(200px, 50%)

At least 200px wide, grows with container.

width: max(200px, 50%);

clamp() — Range Constraint

clamp(150px, 60%, 400px)

Stays between 150px and 400px, prefers 60%.

width: clamp(150px, 60%, 400px);

Fluid Typography with clamp()

Resize the browser to see the typography scale smoothly.

Heading 2 — clamp(1.2rem, 4vw, 2.4rem)

Heading 3 — clamp(1rem, 3vw, 1.8rem)

Body text — clamp(0.875rem, 1.5vw, 1.125rem)

Small text — clamp(0.75rem, 1vw, 0.875rem)

font-size: clamp(minSize, preferredSize, maxSize);

Responsive Container

width: min(90%, 1200px); — Responsive container
.container { width: min(90%, 1200px); margin: 0 auto; }

Dynamic Spacing

This box uses clamp() for padding and font-size. All values scale fluidly with the viewport.

.box { padding: clamp(0.5rem, 3vw, 2rem); }
