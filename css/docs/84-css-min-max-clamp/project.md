# Project: Media-Query-Free Layout with clamp()

## Objective
Build a complete landing page that achieves full responsiveness using only min(), max(), clamp(), and calc() — without any media queries. The page should adapt fluidly from mobile to desktop.

## Requirements
- Fully responsive layout without any @media queries
- Fluid typography using clamp() for all text (h1-h6, body, small)
- Responsive container using min() (width: min(90%, 1200px))
- Card grid using clamp() for column sizing
- All spacing (padding, margin, gap) uses clamp() or max()
- Navigation adapts using min() for link spacing
- At least one section with a full-bleed background

## Stretch Goals
- Animated elements that respond to viewport size
- Dark/light theme toggle using CSS variables
- Performance comparison showing layout shift improvements
- Component library documented with clamp() patterns

## Learning Outcomes
- Master min(), max(), and clamp() for responsive design
- Build fluid layouts without media queries
- Create design systems with constraint-based sizing
- Reduce CSS code while improving responsiveness
