# Mini Project: Image Gallery with Parallax Depth

## Objective
Build an image gallery where cards appear at different depths using translate Z, with interactive movement controls.

## Requirements
1. Create 3-5 cards with placeholder images at different Z depths
2. Parent container has perspective for 3D effect
3. Cards slide in from different directions on page load using translate X/Y
4. Click a card to bring it to front (translate Z to max value)
5. Navigation arrows to scroll through cards

## Specs
- Use standalone `translate` property (not transform)
- Minimum 3 translate values (X, Y, Z combinations)
- Smooth transitions on all movements
- Responsive design

## Stretch Goals
- Add mouse-tracking parallax where cards move based on cursor position
- Add a depth-of-field blur effect on cards further back
- Create an auto-rotate carousel using Z translations

## Evaluation Criteria
- Correct translate usage across all axes
- 3D perspective effect works correctly
- Smooth animations and transitions
- Interactive controls function properly
