CSS Translate — Practical Examples

1. Basic Translate X

Moving element along X-axis.

translateX
translate: 100px 0;

2. Translate XY

Moving element along both axes.

translate(x,y)
translate: 60px 60px;

3. Percentage Translate (Centering)

Using percentage to center an element.

Centered
position: absolute; top: 50%; left: 50%; translate: -50% -50%;

4. Negative Translate

Moving element in negative direction.

negative
translate: -60px -40px;

5. 3D Translate Z

Moving element along Z-axis (requires perspective on parent).

translateZ
translate: 0 0 200px; /* parent needs perspective */

6. Translate vs Transform translate

Standalone translate (left) vs transform translate (right). Same visual result.

translate: 80px
transform
/* Standalone */ translate: 80px 0;
/* Transform */ transform: translate(80px, 0);
