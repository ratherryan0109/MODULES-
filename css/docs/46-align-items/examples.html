Example 2: Perfect Centering

Combine justify-content: center with align-items: center for perfect center alignment.

Perfectly Centered
Horizontally & Vertically
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Example 3: Align-Self Override

Individual items can override the container's align-items with align-self.

flex-start
center
flex-end
stretch
.container { align-items: flex-start; }
.item:nth-child(2) { align-self: center; }
.item:nth-child(3) { align-self: flex-end; }
.item:nth-child(4) { align-self: stretch; }

Example 4: align-items in a Column Layout

When flex-direction: column, align-items works horizontally (cross axis).

Item 1
Item 2
Item 3
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
}

Example 5: Vertical Navigation (align-items: stretch)

Sidebar navigation items stretch to fill width with align-items: stretch.

Home
Profile
Settings
Logout
.sidebar {
  display: flex;
  flex-direction: column;
  align-items: stretch;
}

Example 6: Baseline Alignment for Text

Aligns items by the baseline of their text content — useful for mixed font sizes.

Big
Normal
Small
.container {
  display: flex;
  align-items: baseline;
}
/* All text sits on the same baseline */
