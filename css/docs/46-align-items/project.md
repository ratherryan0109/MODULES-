# Mini Project: Dashboard Card Grid

## Overview
Build a dashboard card grid where all cards have equal height using `align-items: stretch` and a centered login modal.

## Requirements
1. 4 dashboard cards in a row, all equal height
2. Cards have: icon, title, value, description
3. Some cards have more content than others
4. All cards are same height regardless of content
5. A centered modal overlay (centered both axes)
6. Responsive: wraps to 2 columns, then 1

## Bonus
- Use `align-self` on one card to differentiate it
- Add baseline alignment in card footer rows

## Evaluation
- Equal height layout: 25pts
- Centered modal: 25pts
- Responsiveness: 25pts
- Visual design: 25pts
