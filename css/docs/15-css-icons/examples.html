CSS Icons - Examples

Example 1: Inline SVG Icons

SVG icons can be embedded directly in HTML and styled with CSS (color, size, stroke-width).

Home
Heart
Arrow
Clock
Mail
<svg width="32" height="32" viewBox="0 0 24 24"
     fill="none" stroke="#06b6d4" stroke-width="2">
  <path d="M3 12l2-2..." />
</svg>

Example 2: Icon Size Variation

SVG icons scale cleanly at any size because they are vector-based.

<!-- Same icon at different sizes -->
<svg width="16" height="16" ...>...</svg>
<svg width="32" height="32" ...>...</svg>
<svg width="48" height="48" ...>...</svg>

Example 3: Icons with currentColor

Using stroke="currentColor" lets the icon inherit its parent's text color.

Red theme
Green theme 
.icon { stroke: currentColor; }  /* Inherits parent color */

Example 4: Icons in Buttons

Icons inside buttons improve visual communication. Use display: inline-flex with gap for alignment.

Save Delete Back Info 
.btn {
  display: inline-flex;
  align-items: center;
  gap: 0.5rem;
}

Example 5: Icons with Labels

Icons paired with text labels improve scannability in UIs.

5 minutes Due today New York, NY 
.labeled-icon {
  display: inline-flex;
  align-items: center;
  gap: 0.4rem;
}

Example 6: Animated Icons

CSS animations can add motion to SVG icons for loading states or attention-grabbing effects.

spin
pulse
@keyframes spin { to { transform: rotate(360deg); } }
.spin { animation: spin 2s linear infinite; }

@keyframes pulse { 50% { opacity: 0.4; } }
.pulse { animation: pulse 1.5s ease-in-out infinite; }
