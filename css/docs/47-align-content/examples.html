Example 7: Align-Content vs Align-Items

align-content distributes entire lines of wrapped items. align-items aligns individual items within a single line.

align-items: center (one line)
Short
Tall
Short
align-content: center (wrapped lines)
Row 1
Row 1
Row 2
Row 2
/* Aligns items within a line */
align-items: center;

/* Distributes entire lines of content */
align-content: center;  /* requires flex-wrap: wrap */
Important: align-content only works when flex-wrap: wrap is set and there is extra space on the cross axis. It distributes lines of items, not individual items.
