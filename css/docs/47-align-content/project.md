# Mini Project: Multi-Line Content Layout

## Overview
Build a layout with multiple lines of content where line distribution is controlled with `align-content`.

## Requirements
1. A container with 10+ items wrapped to multiple lines
2. Container has extra height (500px) to demonstrate align-content
3. Toggle between different align-content values with CSS classes
4. Include 4 buttons to switch: center, space-between, flex-start, flex-end
5. Use JavaScript (or checkbox hacks) to switch alignment

## Bonus
- Add align-items toggle to compare with align-content
- Visual indicators showing line distribution
- Animate transitions between values

## Evaluation
- Multi-line flex layout: 25pts
- align-content usage: 25pts
- Interactive controls: 25pts
- Visual demonstration: 25pts
