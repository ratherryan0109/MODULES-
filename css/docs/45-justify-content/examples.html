📊 Visualizing Space Distribution

Note how the free space is distributed differently:

space-between: [1]____[2]____[3]  (space between items only)
space-around:  __[1]____[2]____[3]__ (space before, between, after)
space-evenly:  _[1]__[2]__[3]_ (identical spacing everywhere)
