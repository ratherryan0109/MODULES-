# Mini Project: Personal Portfolio Landing Page

## Objective
Build a fully responsive personal portfolio landing page that looks great on mobile, tablet, and desktop devices.

## Requirements

### Layout
- Header with logo and navigation links
- Hero section with headline, subtext, and CTA button
- Skills/Expertise section with 3-6 cards
- Projects section with a grid of project cards
- Contact section with a simple form
- Footer with links and social icons

### Technical Requirements
1. **Viewport meta tag** must be present
2. **Mobile-first CSS** approach with min-width media queries
3. At least **3 breakpoints** (mobile, tablet, desktop)
4. **CSS Grid** for main layout
5. **Flexbox** for navigation and card alignment
6. **Responsive images** using max-width: 100%
7. **Fluid typography** using clamp()
8. **CSS custom properties** for colors and spacing
9. **Smooth transitions** on interactive elements
10. **Form validation** feedback (HTML5 or custom)

### Breakpoint Specifications
- Mobile: < 640px (single column)
- Tablet: 640px - 1024px (2 columns)
- Desktop: > 1024px (3 columns, max-width container)

### Styling Requirements
- Color scheme using CSS variables
- Professional typography (system font stack or Google Font)
- Card hover effects with transforms
- Responsive navigation (hamburger menu on mobile)
- Sticky header on desktop
- Footer stays at bottom

## Deliverables
Submit a single `index.html` file with embedded CSS (or linked `style.css`).

## Bonus Challenges
- Add a dark/light theme toggle using CSS custom properties
- Implement smooth scrolling for anchor links
- Add scroll-triggered animations (CSS only or with Intersection Observer)
- Use CSS Grid named areas for the layout
- Add a loading skeleton or transition

## Evaluation Criteria
- Responsive layout works at all specified breakpoints
- Code is clean, well-commented, and follows best practices
- CSS uses modern features (Grid, Flexbox, Custom Properties, clamp)
- Navigation is functional and accessible
- Form has basic validation
- Visual design is cohesive and professional

## Suggested Tools
- Chrome DevTools for testing
- Lighthouse for performance audit
- WAVE or axe for accessibility audit
