# Mini Project: Parallax Landing Page

## Objective
Build a landing page with a multi-layer parallax scrolling effect using pure CSS (perspective method).

## Requirements
1. Full-screen hero with parallax background
2. Three layers: far background (mountains/sky), mid (trees/buildings), foreground (content)
3. Content sections below with normal scroll
4. Smooth performance across devices
5. Fallback for browsers/devices that don't support 3D transforms
6. Respect prefers-reduced-motion

## Stretch Goals
- Add floating elements with different parallax speeds
- Parallax on multiple sections (not just hero)
- Mouse-driven parallax tilt (responds to cursor)
- Reveal animations triggered by scroll position

## Evaluation
- Pure CSS parallax (no JS scroll listeners)
- Proper perspective and scale compensation
- Smooth 60fps scrolling
- Working fallback for unsupported browsers
- Accessible reduced-motion support
