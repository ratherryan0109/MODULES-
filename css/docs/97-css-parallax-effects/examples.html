CSS Parallax Effects

1. Perspective Parallax (Pure CSS)

Uses perspective and translateZ to create a depth effect. Scroll inside the box — the background moves slower than the foreground.

Foreground Card

This content scrolls at normal speed.

Mid Section

Notice how the background gradients drift at a different rate.

Bottom Card

The parallax effect creates a sense of depth.

.scene { perspective: 2px; overflow-y: auto; }
.bg {
  transform: translateZ(-1px) scale(1.5);
  /* scale(1.5) cancels the shrink from translateZ(-1px) */
}
.content { transform: translateZ(0); }

3. Multi-Layer Parallax

Three layers at different Z-depths create a rich sense of depth. Each layer has a different translateZ value.

Front Layer

Scroll to see each layer move at its own speed.

Mid Layer

Contains the grid pattern (mid speed).

Back Layer

The gradient background moves the slowest.

.layer-back { transform: translateZ(-0.5px) scale(1.5); }
.layer-mid  { transform: translateZ(-0.2px) scale(1.2); }
.layer-front { transform: translateZ(0); }
/* Each layer has a different scroll speed */

4. Mouse-Tracking Parallax

Move your mouse inside the box. The blurred circles shift position in response to mouse movement using JavaScript.

Move mouse
Layers track cursor position
container.addEventListener('mousemove', (e) => {
  const x = (e.clientX / rect.width - 0.5) * 20;
  const y = (e.clientY / rect.height - 0.5) * 20;
  layer.style.transform =
    `translate(${x}px, ${y}px)`;
});

5. Sticky Clip Parallax

A position: sticky element with clip creates a reveal effect as the user scrolls.

Sticky Hero

Scroll down ↓ The hero section sticks and content scrolls over it.

This creates a parallax-like reveal effect as the hero is gradually covered.

Using position: sticky is simpler than perspective-based parallax and works well on mobile.

.hero { position: sticky; top: 0; height: 200px; }
.content { position: relative; }
/* Hero stays visible while content scrolls over it */

6. Scroll-Driven Parallax (transform)

A JavaScript-free parallax using scroll-behavior and CSS transforms on the background element within a scroll container.

Scroll Inside

Background moves at a different speed using the same perspective + translateZ technique.

Smooth & Performant

CSS transforms are GPU-accelerated, making this approach very efficient.

/* Same perspective technique as Example 1 */
.scene { perspective: 2px; overflow-y: auto; }
.bg { transform: translateZ(-1px) scale(1.5); }
