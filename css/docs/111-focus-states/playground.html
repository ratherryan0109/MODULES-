📋 Focus Checklist

✅ Do

  • ✓ Use :focus-visible (not :focus)
  • ✓ Minimum 2px ring width
  • ✓ 3:1 contrast against background
  • ✓ Test with keyboard Tab
  • ✓ Support forced colors mode

❌ Don't

  • ✗ outline: none without replacement
  • ✗ Remove focus for mouse clicks only
  • ✗ Use low-contrast focus colors
  • ✗ Trap focus without escape
  • ✗ Forget custom component focus
