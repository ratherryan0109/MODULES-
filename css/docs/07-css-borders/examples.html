Example 4: Individual Sides

Target specific sides with border-top, border-right, border-bottom, border-left.

border-top only
border-right only
border-bottom only
border-left only
.top { border-top: 5px solid #ef4444; }
.right { border-right: 5px solid #22c55e; }
.bottom { border-bottom: 5px solid #f59e0b; }
.left { border-left: 5px solid #8b5cf6; }

Example 5: Border Color

Set individual side colors, or use the shorthand border-color with up to four values.

top right bottom left
.multi-color {
  border: 5px solid;
  border-color: #ef4444 #22c55e #f59e0b #8b5cf6;
  /*         top    right  bottom  left  */
}

Example 6: Border Shorthand

The border shorthand combines width, style, and color in one declaration.

border: 4px dashed #8b5cf6
.shortcut {
  border: 4px dashed #8b5cf6;
  /* Same as:
     border-width: 4px;
     border-style: dashed;
     border-color: #8b5cf6;
  */
}

Example 7: Transparent Borders

Transparent borders are useful for spacing that changes on hover without layout shift.

Transparent
Visible
.default {
  border: 4px solid transparent;  /* No visible border */
}
.hover-effect:hover {
  border-color: #06b6d4;          /* Appears on hover — no layout shift */
}
