Using nested selectors to style child elements and modifier classes inside a card block.

.card {

    border-color: #3b82f6; border-width: 2px;

  }
}

2. Nested Pseudo-Classes & Compound Selectors

Buttons with &:hover and &.outline demonstrate nesting with the parent selector.

.btn {
  padding: 10px 20px; border-radius: 8px; border: none;
  font-weight: 600; transition: all 0.2s;

  &:hover { /* & = .btn */ }
}

.btn-primary {
  background: #3b82f6; color: white;
  &:hover { background: #1d4ed8; }
}

.btn-outline {
  background: transparent; color: #8b5cf6;
  border: 2px solid #8b5cf6;
  &:hover { background: #8b5cf6; color: white; }
}

3. Nested Pseudo-Elements

Using &::before and &:hover inside a list component creates clean, scoped styling.

  • Responsive design
  • CSS Grid & Flexbox
  • Modern animations
  • Cross-browser support
ul {
  list-style: none; padding: 0;

  li {
    padding: 10px 14px;
    border-bottom: 1px solid #e2e8f0;

    &::before { content: '\2713'; color: #10b981; }
    &:last-child { border-bottom: none; }
    &:hover { background: #f1f5f9; }
  }
}

4. Nested at-rules (@media)

Media queries can be nested directly inside a selector, keeping responsive styles colocated.

Resize browser to see grid collapse.

Card B

Featured item with nested media query support.

Card C

Last item in the responsive grid.

.media-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 12px;

  .item {
    background: #f1f5f9; border-radius: 8px; padding: 20px;

    &.featured {
      background: #dbeafe; border: 1px solid #3b82f6;
    }
  }

  @media (max-width: 600px) {
    grid-template-columns: 1fr; /* nested media query */
  }
}
