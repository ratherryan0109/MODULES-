# Project: Animated Design System with @property

## Objective
Build a design system component library that uses registered custom properties (@property) for smooth animations and transitions. The system should demonstrate the power of typed custom properties for creating fluid, performant UI animations.

## Requirements
- Register at least 5 custom properties with different syntax types (color, number, length, angle, percentage)
- Build 3-4 reusable components (card, button, badge, progress bar) that use these registered properties
- Animated theme switcher with smooth color transitions
- At least one component with a continuous keyframe animation using a registered property
- Interactive controls to adjust property values on a live preview
- Display the @property declarations used

## Stretch Goals
- Gradient rotation animation using registered angle
- Morphing shape using registered length properties
- Performance comparison with unregistered properties
- Export the @property declarations as copyable code

## Learning Outcomes
- Master the @property rule and its descriptors
- Create smooth animations with typed custom properties
- Build animation-friendly design systems
- Understand the CSS Properties and Values API
