CSS Overflow - Examples

1. Overflow Values Comparison

overflow: visible

This long text overflows the container box visibly because overflow is set to visible (default).

overflow: hidden

This long text is clipped and hidden because overflow is set to hidden.

overflow: scroll

Scrollbars always appear even if content fits, because overflow is scroll.

overflow: auto

Scrollbars appear only when needed with overflow: auto.

2. Text Overflow with Ellipsis

.ellipsis { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }
This is a very long text that will be truncated with an ellipsis at the end because it exceeds the container width.
Ellipsis requires three properties: white-space: nowrap, overflow: hidden, and text-overflow: ellipsis.

3. Overflow Wrap (Long Words)

.word-break { overflow-wrap: break-word; }
Without overflow-wrap:
This text contains a verylongwordthatwilloverflowthecontainerwithoutbreaking
With overflow-wrap: break-word:
This text contains a verylongwordthatwillbreakproperlyinsidethecontainer

4. Scrollable Container with Custom Scrollbar

.custom-scroll::-webkit-scrollbar { width: 8px; }
.custom-scroll::-webkit-scrollbar-thumb { background: #6c5ce7; }
Custom scrollbars enhance visual design. Use ::-webkit-scrollbar for WebKit browsers and scrollbar-width for Firefox.
