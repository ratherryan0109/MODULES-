Design Token Demo

This card is styled entirely through CSS custom properties. Changing the tokens on the left updates this card in real time.

:root {\n --card-bg: #313244;\n --card-border: #45475a;\n --card-radius: 12px;\n --card-padding: 24px;\n --btn-bg: #667eea;\n --btn-color: #fff;\n --btn-radius: 8px;\n}

Why This Matters

By centralizing design tokens as CSS custom properties, you can theme entire systems by changing only the token values. This is the foundation of CSS Architecture — maintainable, scalable, predictable stylesheets.
