# Mini Project: Utility-First Landing Page

## Overview
Build a marketing landing page using only utility classes — no custom CSS allowed (except for the utility framework itself).

## Requirements
1. Build a utility CSS framework with at least 30 utility classes
2. Create a landing page with: header, hero, features grid, testimonials, CTA, footer
3. Fully responsive using responsive utility prefixes
4. Include hover/focus state variants
5. No custom component CSS — all styling via utilities

## Technical Requirements
- Your own mini utility framework OR Tailwind CSS
- Responsive design (mobile-first)
- Interactive states (hover, focus)

## Deliverables
1. `framework.css` — Your utility CSS framework
2. `index.html` — Landing page built with utilities
3. `README.md` — Framework documentation

## Evaluation
- Framework completeness: 25 pts
- Utility-only HTML (no custom CSS): 25 pts
- Responsive implementation: 20 pts
- Visual quality: 15 pts
- Documentation: 15 pts
