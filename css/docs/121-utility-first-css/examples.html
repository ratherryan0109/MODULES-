Example 2: Spacing Scale

Utility classes use a consistent 4px scale: p-1 = 4px, p-2 = 8px, p-4 = 16px, etc.

p-1
p-2
p-4
p-6
p-8
:root { --scale: 4px; }
.p-1 { padding: calc(var(--scale) * 1); }  /* 4px */
.p-2 { padding: calc(var(--scale) * 2); }  /* 8px */
.p-4 { padding: calc(var(--scale) * 4); }  /* 16px */
.p-6 { padding: calc(var(--scale) * 6); }  /* 24px */
.p-8 { padding: calc(var(--scale) * 8); }  /* 32px */

Example 5: State Variants

Use the hover: prefix for state-specific styles. These only apply on hover.

.hover\:bg-primary-dark:hover {
  background: #0891b2;
}
.hover\:shadow-lg:hover {
  box-shadow: 0 10px 15px -3px rgba(0,0,0,0.1);
}

Right Panel
