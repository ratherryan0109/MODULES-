# Project: CSS Variables Theme System

## Objective
Build a comprehensive design system powered by CSS custom properties. The system should include theme switching, component customization, and responsive scaling using CSS variables.

## Requirements
- Complete design token system with colors, spacing, typography, shadows, and borders defined as variables
- At least 3 themes (light, dark, and a color theme) switchable via JavaScript
- A set of reusable components (cards, buttons, badges) that use the design tokens
- Scoped variable overrides on individual component instances
- Responsive scaling where spacing/font variables change at breakpoints via media queries
- Live preview showing the current variable values

## Stretch Goals
- Variable inspector panel showing all computed custom properties
- Color palette generator that derives shades from a primary hue variable
- Export theme as CSS code
- CSS variable animation (transition between themes)

## Learning Outcomes
- Master CSS custom property declaration, inheritance, and fallbacks
- Build a theme system with runtime switching
- Use scoped variables for component customization
- Create responsive design systems with variables
