.gallery {
    grid-template-columns: repeat(4, 1fr);
    grid-auto-flow: dense;
    grid-auto-rows: 60px;
}
.big   { grid-column: span 2; grid-row: span 2; }
.wide  { grid-column: span 2; }
.tall  { grid-row: span 2; }
💡 Notice how items are packed tightly with no wasted space. The DOM order may not match visual order.

.container {
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 50px);  /* Row 1-2: 50px */
    grid-auto-rows: 40px;                 /* Row 3+: 40px */
    grid-auto-flow: row;
}
