CSS Backgrounds - Examples

Example 1: Background Color

Use background-color to fill an element with a solid color.

background-color: #06b6d4
.panel { background-color: #06b6d4; }

Example 2: Background Image & Repeat

Images can repeat, tile, or be placed once using background-repeat.

repeat (default)
no-repeat center
.tile { background-image: url('pattern.png'); background-repeat: repeat; }
.single { background-image: url('logo.svg'); background-repeat: no-repeat; background-position: center; }

Example 3: Background Position

Position the background image with keywords, percentages, or pixel values.

bottom right
center left
30% 50%
.pos1 { background-position: bottom right; }
.pos2 { background-position: 30% 50%; }

Example 4: Linear Gradient

linear-gradient() creates smooth color transitions along a straight line.

linear-gradient(135deg, #667eea, #764ba2)
to right: orange → red
.gradient { background: linear-gradient(135deg, #667eea, #764ba2); }
.horizontal { background: linear-gradient(to right, #f97316, #ef4444); }

Example 5: Radial Gradient

radial-gradient() creates circular or elliptical gradients radiating from a center point.

radial-gradient(circle, #ffecd2, #fcb69f)
.radial { background: radial-gradient(circle at center, #ffecd2, #fcb69f); }

Example 6: Background Size (Cover & Contain)

background-size: cover scales the image to fill the element; contain keeps it fully visible.

cover
contain
.cover { background-size: cover; }
.contain { background-size: contain; }

Example 7: Multiple Backgrounds

Layer multiple backgrounds by comma-separating values. The first value is the topmost layer.

Overlay pattern + gradient
.multi {
  background:
    url('overlay.png') repeat,
    linear-gradient(135deg, #f093fb, #f5576c);
}

Example 8: Background Shorthand

The background shorthand combines color, image, repeat, position, and size.

Shorthand: color image repeat position
.shorthand {
  background: #3069fe url('dots.png') repeat-x center;
}

/* Same as:
background-color: #3069fe;
background-image: url('dots.png');
background-repeat: repeat-x;
background-position: center;
*/
