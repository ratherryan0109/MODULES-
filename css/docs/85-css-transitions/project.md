# Mini Project: Interactive Pricing Card

## Objective
Build an interactive pricing card with CSS transitions for hover effects, selection states, and smooth visual feedback.

## Requirements
1. Create a pricing card with: title, price, features list, and CTA button
2. On hover: card lifts (translateY), shadow deepens, CTA button changes color
3. On click/active: button scales down slightly
4. Feature items fade in with staggered delay on card hover
5. Price has a transition on color and size when toggling between monthly/yearly

## Specs
- Use `transition` properties only (no @keyframes)
- Minimum 4 different properties transitioning
- Include at least one custom cubic-bezier
- Respect prefers-reduced-motion

## Stretch Goals
- Add a toggle switch between monthly/yearly pricing
- Animate the price change with a flip effect
- Add a "Popular" badge with transition

## Evaluation Criteria
- Smooth, performant transitions
- Logical timing and easing choices
- Staggered animations work correctly
- Accessible (reduced motion)
