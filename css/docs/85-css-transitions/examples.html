CSS Transitions — Practical Examples

1. Basic Hover Transition

Background color and scale on hover with 0.3s ease.

Hover me
transition: background-color 0.3s ease, transform 0.3s ease;

2. Timing Functions Comparison

Each box moves 200px on hover with a different timing function (1s duration).

linear
ease
ease-in
ease-out
ease-in-out

3. Custom Cubic Bezier (Bounce Effect)

Using cubic-bezier(0.68, -0.55, 0.265, 1.55) for an elastic bounce.

Click/Hover
cubic-bezier(0.68, -0.55, 0.265, 1.55)

4. Transition Delay

Fades out with a 0.2s delay using transition-delay: 0.2s.

Click me
transition: opacity 0.3s ease 0.2s;

5. Multiple Properties, Different Timings

Background, border-radius, and transform all transition independently.

Hover me
transition:
  background-color 0.3s ease,
  border-radius 0.5s ease-in-out,
  transform 0.4s cubic-bezier(0.68, -0.55, 0.27, 1.55);
