Grid Rows Examples

Interactive examples of grid-template-rows with different sizing strategies

1. Fixed Height Rows

Exact pixel heights for specific row tracks.

Row 1: 80px
Row 2: 120px
Row 3: 60px
.container {
    grid-template-rows: 80px 120px 60px;
    grid-template-columns: 1fr;
}

2. The Classic: auto 1fr auto (Sticky Footer)

Header and footer size to their content; main fills all remaining space.

Header (auto height)
Main Content — 1fr fills all remaining vertical space. Resize me!
Footer (auto height) — sticks to bottom
.container {
    display: grid;
    grid-template-rows: auto 1fr auto;
    grid-template-columns: 1fr;
    min-height: 400px; /* Or 100vh */
}
/* Auto rows fit content; 1fr fills leftover space */

3. Equal Height Rows with repeat()

Using repeat(3, 1fr) to create 3 equal-height rows.

Row 1: 1fr
Row 2: 1fr
Row 3: 1fr
.container {
    grid-template-rows: repeat(3, 1fr);
    height: 300px; /* Required: fr needs a reference height */
}

5. minmax() in Rows

Rows with a minimum height that can grow to 1fr.

minmax(60px, 1fr)
At least 60px, grows to fill
minmax(100px, 2fr)
At least 100px, gets double space
minmax(50px, 1fr)
At least 50px, grows to fill
.container {
    grid-template-rows: minmax(60px, 1fr) minmax(100px, 2fr) minmax(50px, 1fr);
    min-height: 400px;
}

6. Implicit Rows with grid-auto-rows

Items beyond the explicit rows get sized by grid-auto-rows.

.container {
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 60px);  /* 2 explicit rows */
    grid-auto-rows: 40px;                 /* All implicit rows */
}
/* First 2 rows: 60px. Remaining rows: 40px each */

7. Mixed Row Layout with Spanning

Real-world pattern: banner (auto), content grid (1fr), metadata row.

Banner — auto height (sizes to content)
Main Content — fills remaining space
Uses 1fr
Side Content — also fills space
Also 1fr
Metadata footer — auto height
.container {
    grid-template-columns: 1fr 1fr;
    grid-template-rows: auto 1fr auto;
    min-height: 350px;
}
