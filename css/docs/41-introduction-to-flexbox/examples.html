Flexbox Introduction - Examples

📦 Block Layout (Default)

Item 1
Item 2
Item 3
display: block; /* default */

Blocks stack vertically, each taking full width.

📐 Flex Layout

Item 1
Item 2
Item 3
display: flex;

Items align horizontally (along the main axis) by default.

📏 Inline-Flex

This text is inline with the container:

A
B
C
display: inline-flex;

The container shrinks to fit its content, like inline-block.
