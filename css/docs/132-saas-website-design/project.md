# Project Specification: SaaS Website Design

## Overview
Build a conversion-optimized SaaS website called "CloudPro" with hero, feature grid, pricing comparison, testimonials, and CTA sections.

## Requirements
- Fixed nav with transparent-to-solid background
- Hero section with headline, subtext, CTA, and preview image
- Feature grid with 6 cards (icon, title, description)
- Pricing comparison with 3 tiers (one featured)
- Testimonial section with quote cards
- CTA banner with gradient background
- Footer with links

## CSS Features
- CSS Grid for features and pricing sections
- Flexbox for nav, hero stats, testimonial layout
- Glassmorphism header with backdrop-filter
- Gradient backgrounds (hero, CTA)
- Hover effects (card lift, button glow)
- Scale transform on featured pricing card

## Accessibility
- Skip link and semantic structure
- ARIA labels on CTA buttons
- Focus-visible outlines
- Proper heading hierarchy
- Alt text on all images

## Project Structure

```
saas-website/
├── index.html
├── css/style.css
└── assets/images/
```
