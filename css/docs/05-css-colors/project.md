# Mini Project: Brand Style Guide - Colors

## Project Overview
Create a brand color style guide page that showcases a company's complete color palette with all technical specifications (HEX, RGB, HSL), usage guidelines, and accessibility information.

## Requirements
1. **Primary Colors**: 3-5 main brand colors with all format conversions
2. **Secondary Colors**: 3-4 accent/supporting colors
3. **Neutral/Grayscale**: 5-step gray scale from light to dark
4. **Semantic Colors**: Success, warning, error, info colors
5. **Accessibility Section**: Show contrast ratios between primary text/background pairs
6. **Usage Guidelines**: When to use each color

## Design
- Professional color swatch cards with all format values
- Color contrast verification badges
- Interactive hover effects on swatches
- Clean, accessible layout

## Stretch Goals
- Add a color-blindness simulation toggle
- Include CSS custom property definitions
- Add a "Copy to Clipboard" feature for color values
