CSS Colors - Examples

Example 1: Named Colors

CSS supports 140 named colors like red, teal, and coral.

red
coral
tomato
gold
teal
navy
rebeccapurple
mediumseagreen
.one { color: red; }
.two { color: teal; }
.three { color: rebeccapurple; }

Example 2: HEX Colors

HEX notation uses #RRGGBB or shorthand #RGB, with optional alpha #RRGGBBAA.

#ff0000
#00cc66
#3069fe
#f0f (shorthand)
#ff000080
#0f172a
.hex-full { color: #ff0000; }         /* Full notation */
.hex-short { color: #f00; }            /* Shorthand */
.hex-alpha { color: #ff000080; }       /* 50% alpha */

Example 3: RGB / RGBA

RGB defines colors by red, green, blue channels (0–255 each). RGBA adds an alpha channel for opacity.

rgb(255,0,0)
rgb(0,200,100)
rgb(48,105,254)
rgba(..,0.5)
1.0
0.8
0.6
0.4
0.2
0.1
.solid { background: rgb(255, 0, 0); }
.translucent { background: rgba(48, 105, 254, 0.5); }

Example 4: HSL / HSLA

HSL uses hue (0–360°), saturation (0–100%), and lightness (0–100%). HSLA adds alpha.

hsl(0,100%,50%)
hsl(120,60%,50%)
hsl(240,80%,60%)
hsl(0,80%,70%)
hsl(0,80%,30%)
hsla(..,0.5)
.vivid { background: hsl(0, 100%, 50%); }      /* Pure red */
.muted { background: hsl(0, 50%, 70%); }        /* Washed-out red */
.translucent { background: hsla(240, 80%, 60%, 0.5); }

Example 5: currentColor

currentColor refers to the element's computed color value. Change it once and all uses update.

Red theme
Border uses currentColor — matches this text color.
Cyan theme
Same CSS, different color — border updates automatically. 
.card {
  color: #ef4444;
  border: 3px solid currentColor;  /* Matches text color */
}

Example 6: Opacity

The opacity property makes the entire element (and its children) translucent.

100% 80% 60% 40% 20% 
.faded { opacity: 0.5; }  /* Element and children at 50% */

Example 7: Modern Color Spaces (OKLCH, P3)

Modern CSS supports wider gamut color spaces like oklch() and color() for more vibrant colors.

oklch()
color(srgb)
color(p3)
.oklch { background: oklch(0.6, 0.2, 25); }
.p3 { background: color(display-p3 1 0.2 0.3); }
