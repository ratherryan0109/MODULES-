2. Combinators

Descendant (space)

I'm inside a div inside .descendant-demo — red via descendant selector.

Child (>)

I'm a direct child — green background.

I'm nested — NOT styled by child selector.

Adjacent Sibling (+)

I'm immediately after h3 — bold blue.

I'm second — not styled.

General Sibling (~)

I'm first after h3 — italic.

I'm also after h3 — also italic.

.descendant-demo p { color: #e53e3e; } /* All p descendants */
.child-demo > p { background: #c6f6d5; } /* Direct children only */
h3 + p { font-weight: bold; } /* Immediately after h3 */
h3 ~ p { font-style: italic; } /* All after h3 */

3. Attribute Selectors

Secure link (https) | Regular link (http) | PDF file

a[href^="https"] { color: #38a169; } /* Starts with */
a[href$=".pdf"]::after { content: " 📄"; } /* Ends with */
input[type="text"] { border: 2px solid #4299e1; } /* Exact value */
