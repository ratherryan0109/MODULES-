# Mini Project: Contact Form with Validation

## Goal
Build a fully styled, accessible contact form with real-time validation indicators and custom-styled form controls.

## Requirements
- Name, Email, Subject (select), Message (textarea), and file upload fields
- Labels above each field
- Custom-styled checkboxes (agree to terms)
- Real-time validation: green border on valid, red on invalid
- Submit button with hover and active states
- Responsive form layout (single column on mobile)
- Clear focus indicators on all interactive elements

## Stretch Goals
- Add floating label animation
- Style the file input with a custom button
- Add a character counter for the message field
- Use `:focus-within` for group highlighting

## Evaluation Criteria
- No JavaScript for styling (HTML5 validation only)
- Consistent spacing and typography
- All form elements are styled
- Accessible (labels, focus, error messages)
- Cross-browser compatible approach
