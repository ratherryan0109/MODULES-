🎨 Color Accessibility Examples

Demonstrating accessible vs inaccessible color usage in CSS

1. Status Indicators

❌ Bad: Color Only

Online

Offline

Away

A color-blind user cannot distinguish these statuses.

✅ Good: Color + Icon + Label

✓ Online
✗ Offline
~ Away

Multiple indicators ensure information is accessible to everyone.

2. Form Validation

❌ Bad: Red Border Only

✅ Good: Red Border + Icon + Error Text

Please enter a valid email address.
