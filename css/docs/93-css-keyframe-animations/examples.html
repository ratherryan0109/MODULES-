CSS Keyframe Animations

1. Bounce

The element bounces up and down using a two-keyframe animation at 0%, 50%, and 100%.

bounce
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-40px); }
}
.box { animation: bounce 1s ease infinite; }

2. Shake

Multiple keyframes create a back-and-forth horizontal shake effect.

shake
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20%, 60% { transform: translateX(-8px); }
  40%, 80% { transform: translateX(8px); }
}
.box { animation: shake 0.6s ease infinite; }

3. Pulse

Smooth scaling combined with opacity change for a breathing effect.

pulse
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.2); opacity: 0.7; }
}
.box { animation: pulse 1.5s ease infinite; }

4. Spin

Continuous 360-degree rotation. Using to is shorthand for 100%.

spin
@keyframes spin {
  to { transform: rotate(360deg); }
}
.box { animation: spin 1s linear infinite; }

5. Fade In Up

A one-shot entrance animation (not infinite). Uses animation-fill-mode: both to retain start/end states.

fade in
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(24px); }
  to { opacity: 1; transform: translateY(0); }
}
.box { animation: fadeInUp 0.8s ease both; }

6. Wiggle

Alternating rotation keyframes create a playful wiggling motion.

wiggle
@keyframes wiggle {
  0%, 100% { transform: rotate(0); }
  25% { transform: rotate(-12deg); }
  75% { transform: rotate(12deg); }
}
.box { animation: wiggle 0.8s ease infinite; }

7. Color Cycle

Multiple keyframes cycle through a series of background colors over 4 seconds.

color
@keyframes colorCycle {
  0% { background: #3b82f6; }
  25% { background: #8b5cf6; }
  50% { background: #10b981; }
  75% { background: #f59e0b; }
  100% { background: #3b82f6; }
}
.box { animation: colorCycle 4s ease infinite; }

8. Heartbeat

Rapid sequential scale changes create a realistic heartbeat rhythm.

heart
@keyframes heartbeat {
  0%, 100% { transform: scale(1); }
  14% { transform: scale(1.3); }
  28% { transform: scale(1); }
  42% { transform: scale(1.3); }
  70% { transform: scale(1); }
}
.box { animation: heartbeat 1.2s ease infinite; }
