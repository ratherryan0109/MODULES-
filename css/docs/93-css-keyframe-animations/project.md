# Mini Project: Animated Landing Page Hero

## Objective
Build a landing page hero section with multiple coordinated CSS keyframe animations for an engaging first impression.

## Requirements
1. Hero heading: fade-in-up entrance animation
2. Subtitle: slide-in from left with delay
3. CTA button: pulse animation to draw attention
4. Background: slow floating/parallax shapes
5. Hero image: gentle float animation (translateY alternate)
6. All animations staggered for a sequenced reveal
7. Respect prefers-reduced-motion

## Stretch Goals
- Typewriter text animation using steps()
- Gradient animation shifting through colors
- Particle-like floating elements with different delays
- Scroll-triggered entrance (with Intersection Observer)

## Evaluation
- Smooth 60fps (transform/opacity only)
- Proper animation-fill-mode usage
- Coordinated timing and delays
- Responsive and accessible
- Clean, well-organized code
