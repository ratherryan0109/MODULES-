# Mini Project: CSS Architecture Design

## Overview
Design and implement a complete CSS architecture for a medium-sized web application (e-commerce site).

## Requirements
1. Write an architecture document explaining your approach
2. Implement the full file structure with @layer
3. Create at least 5 reusable components following naming conventions
4. Create at least 3 layout objects
5. Include utility classes for common adjustments
6. All design tokens in custom properties

## Deliverables
1. `architecture.md` — Design document
2. `styles/` — Complete file structure with all CSS files
3. `index.html` — Demo page showcasing the architecture

## Evaluation
- Architecture document clarity: 20 pts
- Layer usage correctness: 20 pts
- Object/component separation: 20 pts
- Naming convention consistency: 15 pts
- File structure organization: 15 pts
- Demo page implementation: 10 pts
