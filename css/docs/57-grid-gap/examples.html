row-gap: 20px (vertical) vs column-gap: 40px (horizontal). Same across all sizes.

    column-gap: 40px;   /* Horizontal gap between columns */
}

4. Gap + Padding for Complete Spacing

Combining gap: 16px with padding: 20px for symmetrical internal AND external spacing.

1
2
3
4
5

.container {
    gap: 16px;
    padding: 20px;   /* Adds space at outer edges */
}
The purple area shows the padding. Items have 16px gap between them. Total internal + external spacing = 36px from any item to container edge.
