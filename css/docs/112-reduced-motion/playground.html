📊 Animation Classifications

❌ Remove When Reduced

  • ✦ Decorative bounce/float
  • ✦ Parallax scrolling
  • ✦ Confetti effects
  • ✦ Hover scale transforms
  • ✦ Page-load splash animations
  • ✦ Shimmer/Skeleton pulse

⚠️ Reduce Intensity

  • ✦ Slide → Fade transitions
  • ✦ Scale → Color hover effects
  • ✦ Page transitions (simplify)
  • ✦ Scroll-triggered reveals

✅ Keep Functional

  • ✦ Loading spinners
  • ✦ Progress bars
  • ✦ Typing indicators
  • ✦ Focus indicators
  • ✦ Essential UI transitions

💡 Best Practices

  • ✦ Use prefers-reduced-motion
  • ✦ Provide manual toggle
  • ✦ Animate transform/opacity
  • ✦ Avoid infinite animations
  • ✦ Test with OS setting on
