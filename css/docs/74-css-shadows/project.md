# Project: CSS Shadow Showcase

## Objective
Build a visually interactive "Shadow Gallery" page that demonstrates the full range of CSS shadow capabilities. The gallery should display 8-12 cards, each highlighting a different shadow technique with live labels showing the CSS code beneath each card.

## Requirements
- Minimum 8 cards, each with a distinct shadow style (basic drop, layered, inset, neon glow, 3D text, drop-shadow on SVG, soft elevation, outline/ring)
- Each card must display the CSS code used to create its shadow effect
- Hover state that transforms the shadow (increase elevation, change color, animate glow)
- A responsive grid layout (auto-fill, minmax(280px, 1fr))
- Semantic HTML with a dark theme background to make shadows pop
- A toggle or filter to show only box-shadow cards or text-shadow cards

## Stretch Goals
- Interactive controls to adjust blur, offset, and color on a live preview card
- CSS-only animation that cycles through different shadow styles
- Export the current shadow as copyable CSS code

## Learning Outcomes
- Master box-shadow parameters including spread and inset
- Understand the difference between box-shadow and drop-shadow()
- Create layered and realistic shadow effects
- Use shadows for visual hierarchy and depth
