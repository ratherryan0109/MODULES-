CSS Filters

Example 1: Basic Filter Functions

Filters apply visual effects like blur, brightness, and contrast to elements.

No filter
blur(3px)
brightness(1.5)
contrast(2)
.blur      { filter: blur(3px); }
.bright    { filter: brightness(1.5); }
.contrast  { filter: contrast(2); }

Example 2: Color Manipulation Filters

Transform colors with grayscale, sepia, hue-rotate, saturate, and invert.

grayscale(1)
sepia(0.8)
hue-rotate(180deg)
saturate(3)
invert(1)
.grayscale  { filter: grayscale(1); }
.sepia      { filter: sepia(0.8); }
.hue-rotate { filter: hue-rotate(180deg); }
.saturate   { filter: saturate(3); }
.invert     { filter: invert(1); }

Example 3: Combined Filters

Multiple filter functions can be combined in a space-separated list.

Original
brightness + contrast + saturate
.enhanced {
    filter: brightness(1.2) contrast(1.2) saturate(1.3);
}
This "enhance" combo is great for product photos and hero images.

Example 4: drop-shadow()

drop-shadow() creates a shadow that follows the visible alpha channel, unlike box-shadow on the bounding box.

drop-shadow follows shape
box-shadow on bounding box
.drop { filter: drop-shadow(4px 6px 8px rgba(0,0,0,0.3)); }
/* Captures the triangle shape; box-shadow casts on the rectangle */

Example 5: backdrop-filter (Glassmorphism)

backdrop-filter applies filters to the area behind the element, creating glass-like effects.

Glass Panel

backdrop-filter: blur(12px)

.glass {
    background: rgba(255,255,255,0.15);
    backdrop-filter: blur(12px) saturate(1.5);
    border: 1px solid rgba(255,255,255,0.2);
}

Example 6: Hover Filter Transitions

Smoothly transition between filter states on hover.

Grayscale on/off
Brighten on hover
.grayscale-hover {
    filter: grayscale(1);
    transition: filter 0.4s;
}
.grayscale-hover:hover {
    filter: grayscale(0);
}

Example 7: Animated Filters

Filters can be animated with keyframes for dynamic visual effects.

Pulsing saturation + brightness

@keyframes pulse {
    0%, 100% { filter: saturate(1) brightness(1); }
    50%      { filter: saturate(2) brightness(1.3); }
}
.animated {
    animation: pulse 2s ease-in-out infinite;
}
Use @media (prefers-reduced-motion: reduce) to disable filter animations.
