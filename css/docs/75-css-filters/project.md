# Project: CSS Filter Studio

## Objective
Build an interactive "Filter Studio" application where users can apply, combine, and customize CSS filters on sample images. The studio should include preset filters, manual controls, and the ability to see the resulting CSS code.

## Requirements
- A preview area showing an image with live filter application
- Slider controls for at least 6 filter functions (blur, brightness, contrast, grayscale, sepia, hue-rotate, saturate)
- At least 5 preset filter combinations (e.g., Vintage, Noir, Vibrant, Sepia, Cool Blue)
- Display the generated CSS filter code that users can copy
- A reset button to clear all filters
- Responsive layout with a dark theme

## Stretch Goals
- Allow users to upload their own image
- Save favorite filter combinations to local storage
- Add a comparison slider to show before/after
- Include a backdrop-filter demo panel

## Learning Outcomes
- Master CSS filter functions and their parameters
- Understand combined filter effects
- Learn about backdrop-filter for glass effects
- Apply filters in interactive user interfaces
