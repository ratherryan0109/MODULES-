CSS Links - Examples

Example 1: Link States (LVHA Order)

Links have four states that must be styled in the correct order: :link, :visited, :hover, :active (LVHA).

Normal link (try hovering and clicking) Another link to test hover effect 
a:link    { color: #2563eb; }   /* Unvisited */
a:visited { color: #7c3aed; }   /* Visited */
a:hover   { background: #dbeafe; }  /* Hover */
a:active  { color: #dc2626; }   /* While clicking */
/* Order matters: LoVe HAte (link, visited, hover, active) */

Example 2: Removing Underline

The default underline on links can be removed with text-decoration: none and replaced with custom styling.

This link has no underline

This link has an animated underline on hover

.no-underline { text-decoration: none; }

/* Animated underline */
.fancy-underline {
  text-decoration: none;
  position: relative;
}
.fancy-underline::after {
  content: '';
  position: absolute;
  bottom: -2px; left: 0;
  width: 100%; height: 2px;
  background: #2563eb;
  transform: scaleX(0);
  transition: transform 0.2s;
}
.fancy-underline:hover::after {
  transform: scaleX(1);
}

Example 5: Different Link Types

Links can target different protocols using specific URL schemes.

https://example.com (external)

mailto:user@example.com

tel:+1 (234) 567-890

<a href="https://example.com">Web link</a>
<a href="mailto:user@example.com">Email link</a>
<a href="tel:+1234567890">Phone link</a>

Example 6: Styling by Link Type with Attribute Selectors

Use attribute selectors to style links differently based on their destination.

Secure link (https) — green

Insecure link (http) — red

a[href^="https"] { color: #22c55e; }  /* Secure links */
a[href^="http:"] { color: #ef4444; }  /* Insecure links */
a[href$=".pdf"]::after { content: ' (PDF)'; }  /* File type */

Example 7: Focus Styles for Accessibility

Never remove focus outlines entirely. Use :focus-visible to show outlines only during keyboard navigation.

Tab to me — focus ring appears

/* ❌ Bad — removes focus for everyone */
a:focus { outline: none; }

/* ✅ Good — hides focus for mouse, shows for keyboard */
a:focus-visible {
  outline: 3px solid #06b6d4;
  outline-offset: 2px;
}
