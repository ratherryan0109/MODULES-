# Mini Project: Interactive Link Styles

Create a page showcasing different link styles:
- Text links with animated underline (underline slides in on hover)
- Button-style links with multiple color variants
- Card links (entire card is clickable)
- Breadcrumb navigation links
- Social media icon links with hover effects
- All links must have proper focus styles
- Use attribute selectors to style external links, emails, and phone links differently
