CSS Alignment - Examples

1. Text Alignment

text-align: left | center | right | justify;
Left aligned text
Center aligned text
Right aligned text
Justified text stretches each line to fill the width. This creates even margins on both sides but can cause uneven spacing between words.

2. Block Centering with margin: auto

.block { width: 300px; margin: 0 auto; }
Centered Block

3. Vertical Alignment (inline-block)

vertical-align: baseline | top | middle | bottom;
baseline top middle bottom

4. Flexbox Centering

display: flex; align-items: center; justify-content: center;
Centered

5. Grid Centering

display: grid; place-items: center;
Centered

6. Absolute Centering with Transform

position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);
Absolutely Centered

7. Line Height Vertical Centering

.box { height: 80px; line-height: 80px; text-align: center; }
Vertically centered text
Only works for single lines of text.
