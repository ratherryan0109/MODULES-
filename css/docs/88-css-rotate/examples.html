CSS Rotate — Practical Examples

1. Basic 2D Rotation (Z-axis)

Rotates the element flat on the screen.

rotate(45deg)
0.25turn
rotate: 45deg;
rotate: 0.25turn;

2. Full Rotation (360°)

Element completes a full spin on hover.

360deg
1turn

3. 3D Rotation - X Axis

Flips forward around X-axis (requires perspective).

X 180deg
rotate: X 180deg;
/* parent: perspective: 600px */

4. 3D Rotation - Y Axis

Flips sideways around Y-axis (door-opening effect).

Y 180deg
rotate: Y 180deg;
