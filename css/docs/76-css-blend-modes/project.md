# Project: CSS Blend Mode Art Gallery

## Objective
Create an interactive art gallery that showcases CSS blend modes through a series of visually striking compositions. Each gallery piece should demonstrate a different blend mode or combination of blend modes.

## Requirements
- Minimum 8 gallery pieces, each demonstrating a different blend mode or combination
- At least one duotone hero section using background-blend-mode
- At least one text piece using mix-blend-mode: difference or exclusion
- At least one interactive piece where hover changes the blend mode
- A color picker that lets visitors change the overlay color
- Display the CSS code for the currently visible piece
- Responsive grid layout

## Stretch Goals
- Allow visitors to upload their own image and apply blend modes
- Animated blend mode transitions
- Side-by-side comparison slider (original vs blended)
- Export the current composition as CSS code

## Learning Outcomes
- Master mix-blend-mode and background-blend-mode
- Create duotone and artistic overlay effects
- Understand isolation and stacking context with blend modes
- Build accessible visual effects with blend modes
