CSS Blend Modes

Example 1: mix-blend-mode on Elements

mix-blend-mode blends an element with its background. Each element sits on a gradient background.

multiply
screen
overlay
difference
.multiply   { mix-blend-mode: multiply; }
.screen     { mix-blend-mode: screen; }
.overlay    { mix-blend-mode: overlay; }
.difference { mix-blend-mode: difference; }
multiply darkens, screen lightens, overlay increases contrast, difference inverts colors.

Example 3: Text with mix-blend-mode

Blend text directly into the background for dramatic hero sections.

BLEND MODE

.hero-text {
    color: white;
    mix-blend-mode: difference;
    /* Text inverts the background colors beneath it */
}
difference blend mode inverts whatever is behind the text, creating a striking effect on gradient backgrounds.

Example 5: isolation: isolate

isolation: isolate creates a new stacking context, preventing blend modes from affecting elements outside the container.

The blended text below only blends within this box:

Blended Text

.container {
    isolation: isolate;
    /* mix-blend-mode on children won't bleed outside */
}
.child { mix-blend-mode: difference; }
Without isolation, the child's mix-blend-mode would blend with elements behind the container too.
