5. prefers-reduced-motion Override

When users prefer reduced motion, smooth scrolling should be disabled.

@media (prefers-reduced-motion: reduce) {\n html {\n scroll-behavior: auto;\n }\n * {\n animation-duration: 0.01ms !important;\n transition-duration: 0.01ms !important;\n }\n}
