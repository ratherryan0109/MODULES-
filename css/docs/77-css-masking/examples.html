CSS Masking

Example 1: Linear Gradient Mask (Fade)

mask-image with a gradient gradually reveals or hides parts of an element. Black = visible, transparent = hidden.

Fade left → right
Fade bottom → top
Reveal band
.fade-right {
    -webkit-mask-image: linear-gradient(to right, transparent, black);
    mask-image: linear-gradient(to right, transparent, black);
}
/* transparent = hidden, black = fully visible */

Example 2: Radial Gradient Mask (Reveal)

Radial masks create circular, elliptical, or spotlight reveal effects.

Circular reveal
Elliptical
Spotlight
.circular  { mask-image: radial-gradient(circle, black 30%, transparent 70%); }
.spotlight { mask-image: radial-gradient(circle at top left, black, transparent 70%); }

Example 3: Pattern Masks

Use repeating gradients to create pattern masks like stripes or checkerboards.

Diagonal stripes
Horizontal lines
Dotted pattern
.stripes  { mask-image: repeating-linear-gradient(45deg, black 0px, black 8px, transparent 8px, transparent 16px); }
.dots     { mask-image: repeating-radial-gradient(circle, black 0px, black 4px, transparent 4px, transparent 10px); }

Example 4: SVG Masks

An SVG <mask> element defines a reusable mask with shapes. Use it via mask: url(#id).

SVG circle mask
SVG star mask
.masked {
    mask: url('circle.svg') center / contain;
    mask-repeat: no-repeat;
}
/* SVG mask uses white = visible, black = hidden */

Example 5: Image with Mask

Apply masks directly to images for creative cropping.

Circle crop
Fade out
Diamond
.circle  { mask: radial-gradient(circle, black 50%, transparent 52%); }
.fade    { mask: linear-gradient(to bottom, black 50%, transparent 100%); }
.diamond { mask: linear-gradient(45deg, transparent 30%, black 30%, black 70%, transparent 70%); }

Example 6: Text Masking (background-clip)

While not strictly mask-image, background-clip: text uses the text as a mask for the background.

GRADIENT TEXT

.gradient-text {
    background: linear-gradient(135deg, #06b6d4, #6366f1, #ec4899);
    -webkit-background-clip: text;
    background-clip: text;
    -webkit-text-fill-color: transparent;
}
