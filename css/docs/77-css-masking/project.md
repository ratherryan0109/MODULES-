# Project: CSS Masking Showcase

## Objective
Build an interactive masking demonstration page that showcases various CSS masking techniques: gradient masks, SVG masks, image masks, and text masking.

## Requirements
- A hero section with a gradient mask that fades the image into the background color
- A gallery of at least 4 images each masked with a different SVG shape (circle, polygon, star, heart)
- A section demonstrating text masking (gradient text via background-clip)
- Interactive controls to change the mask angle, position, and type on a live preview
- Display the current mask CSS code
- Responsive layout

## Stretch Goals
- Mouse-following radial mask (spotlight effect)
- Animated mask that cycles through different shapes
- Upload user image and apply selected mask
- Mask compositing demo (add, subtract, intersect)

## Learning Outcomes
- Master mask-image with gradients, images, and SVG
- Understand mask positioning and sizing
- Create text masking effects
- Build interactive and animated mask experiences
