CSS Padding - Examples

Example 1: Equal Padding on All Sides

A single value applies equal padding to all four sides inside the element.

padding: 40px on all sides
.box { padding: 40px; }

Example 2: Uneven Padding (Two Values)

Two values set: vertical (top/bottom) and horizontal (right/left).

padding: 10px 50px (narrow top/bottom, wide sides)
padding: 30px 15px (tall, narrow sides)
.wide { padding: 10px 50px; }  /* vert / horiz */
.tall { padding: 30px 15px; }

Example 3: Padding Shorthand (1–4 Values)

Like margin, padding uses the TRBL pattern.

padding: 20px
All sides: 20px
padding: 10px 30px
Vertical: 10px, Horizontal: 30px
padding: 10px 15px 30px
Top: 10px, Sides: 15px, Bottom: 30px
padding: 10px 15px 20px 25px
Top Right Bottom Left
/* 1 value */  padding: 20px;
/* 2 values */ padding: 10px 30px;
/* 3 values */ padding: 10px 15px 30px;
/* 4 values */ padding: 10px 15px 20px 25px;

Example 4: Individual Side Padding

Use padding-top, padding-right, padding-bottom, padding-left for fine-grained control.

top:30px left:50px bottom:10px right:10px
.custom {
  padding-top: 30px;
  padding-right: 10px;
  padding-bottom: 10px;
  padding-left: 50px;
}

Example 5: Padding vs Margin Comparison

Padding adds space inside the element (background visible). Margin adds space outside.

padding:20px (green area inside)
margin:20px (gap outside)
padding — space inside, background visible
margin — space outside, transparent
.inner { padding: 20px; }  /* Inside — bg extends */
.outer { margin: 20px; }    /* Outside — transparent */

Example 6: Box-Sizing Effect on Padding

box-sizing: border-box includes padding in the element's total width; content-box adds it on top.

content-box
200px + 40px

Total width: 240px

border-box
200px includes padding

Total width: 200px

.content-box {
  width: 200px;
  padding: 20px;   /* Total: 240px */
}
.border-box {
  width: 200px;
  padding: 20px;
  box-sizing: border-box;  /* Total: 200px */
}

Example 7: Padding Creating Visual Breathing Room

Padding prevents content from touching the edges of its container, improving readability.

Header

Content with 1rem padding keeps text away from edges. This is much more readable than content that butts right up against the border.

Footer
.card {
  border: 1px solid #94a3b8;
  border-radius: 8px;
}
.card-body { padding: 1rem; }  /* Breathing room */
