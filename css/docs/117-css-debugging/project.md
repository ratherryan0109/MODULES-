# Mini Project: CSS Debugging Battle

## Overview
You'll receive a broken web page with 10+ CSS bugs. Your task is to identify and fix every bug using DevTools and debugging techniques.

## Bug Categories
1. Specificity conflicts (2 bugs)
2. Layout/overflow issues (2 bugs)
3. Responsive breakpoint issues (1 bug)
4. z-index/stacking issues (1 bug)
5. Animation/transition issues (1 bug)
6. Flexbox/Grid alignment (2 bugs)
7. Print style issue (1 bug)

## Requirements
- Document each bug found (symptom, cause, fix)
- Use only DevTools for initial diagnosis
- Fix all bugs with clean CSS (no !important)
- Provide before/after screenshots

## Deliverables
1. `fixed.css` — Corrected stylesheet
2. `debug-report.md` — Detailed bug report

## Evaluation
- Bugs correctly identified: 40 pts
- Clean fixes (no !important): 20 pts
- Debug report quality: 20 pts
- Efficient debugging process: 20 pts
