Example 2: Specificity Conflict

Three conflicting selectors target the same element. The ID selector wins due to highest specificity.

This text has three conflicting selectors. Check computed styles in DevTools.

#special (0,1,0) > .highlight (0,0,1) and .text (0,0,1) 

#special { color: green; }     /* (0,1,0) — wins */
.highlight { color: blue; }    /* (0,0,1) */
.text { color: red; }         /* (0,0,1) */

Example 3: Overflow Detection

Long content overflowing its container is a common layout bug. The red border highlights the container boundary.

This very long text would overflow its container without ellipsis handling.

Solution: overflow: hidden; text-overflow: ellipsis; white-space: nowrap;

.container {
  width: 200px;
  border: 2px solid red; /* Visual indicator */
}
.content {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

Example 5: Collapsing Margins

Adjacent vertical margins collapse to the larger value instead of adding. This often surprises developers.

Box A (margin-bottom: 30px)
Box B (margin-top: 20px)

The gap between boxes is 30px (the larger margin), not 50px.

/* Margin collapse: the gap is 30px, not 50px */
.box-a { margin-bottom: 30px; }
.box-b { margin-top: 20px; }
/* Fix: use padding or a single margin direction */

Example 7: DevTools Console Tips

Use the browser DevTools console for quick CSS debugging.

// Inspect element styles
$0.style.color = 'red';

// Check computed styles
getComputedStyle($0).fontSize;

// Toggle pseudo-classes
$0.style.outline = '2px solid red';
