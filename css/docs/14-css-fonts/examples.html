CSS Fonts - Examples

Example 1: Font Families

The font-family property lists fallbacks from preferred to generic. Always end with a generic family (serif, sans-serif, monospace).

Georgia, serif
Classic newspaper feel
System UI, sans-serif
Modern, clean
Courier New, monospace
Code-like appearance
.serif    { font-family: Georgia, 'Times New Roman', serif; }
.sans     { font-family: 'Segoe UI', system-ui, sans-serif; }
.mono     { font-family: 'Courier New', Consolas, monospace; }

Example 2: Font Size

Font size can be absolute (px, pt) or relative (em, rem, %, vw). Relative units respect user preferences.

12px — too small for body text
16px — browser default, good for body
1.5rem — heading sized
clamp(1rem, 2.5vw, 2rem) — fluid
.small  { font-size: 12px; }
.body   { font-size: 16px; }        /* or 1rem */
.h1     { font-size: 2rem; }
.fluid  { font-size: clamp(1rem, 2.5vw, 2rem); }

Example 4: Font Style and Variant

font-style controls italic/oblique. font-variant: small-caps converts text to uppercase with smaller capitals.

Normal style
Italic style
Small Caps
.normal     { font-style: normal; }
.italic     { font-style: italic; }
.small-caps { font-variant: small-caps; }

Example 5: Font Shorthand

The font shorthand combines style, weight, size/line-height, and family in one declaration.

font: italic 600 1.25rem/1.8 Georgia, serif 
.shorthand {
  font: italic 600 1.25rem/1.8 Georgia, serif;
  /*  style  weight  size/line-height  family  */
}

Example 6: Web Fonts with @font-face

Use @font-face to load custom fonts. Provide multiple formats (woff2, woff) for browser compatibility.

@font-face loads fonts from your server. Example for a self-hosted font: 
@font-face {
  font-family: 'MyCustomFont';
  src: url('fonts/custom.woff2') format('woff2'),
       url('fonts/custom.woff') format('woff');
  font-weight: 400;
  font-style: normal;
  font-display: swap;  /* Show text in fallback while loading */
}

body { font-family: 'MyCustomFont', sans-serif; }

Example 7: Font Loading with Google Fonts

Google Fonts are loaded via a <link> tag in the HTML head. Add &display=swap for better performance.

HTML: <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

CSS: font-family: 'Roboto', sans-serif;
This text would use Roboto if the Google Font link were added to the page head.
<!-- In <head> -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">

/* In CSS */
body { font-family: 'Inter', sans-serif; }

Example 8: Font Rendering Properties

Fine-tune font rendering with font-smooth and text-rendering.

Smooth rendering — These settings improve legibility on modern screens.
text-rendering: optimizeLegibility enables common ligatures. 
body {
  -webkit-font-smoothing: antialiased;  /* macOS */
  -moz-osx-font-smoothing: grayscale;
  text-rendering: optimizeLegibility;
}
