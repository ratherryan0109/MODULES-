# Mini Project: Social Media Icon Sprite

## Goal
Create a social media icon panel using a single sprite sheet image with hover and active states.

## Requirements
- 5 social media icons (Facebook, Twitter, Instagram, LinkedIn, YouTube) in a single sprite
- Each icon is 48×48px
- Default state shows colored icons (row 1 of sprite)
- Hover state shows darker/alternative version (row 2)
- Active/click state shows a third variant (row 3)
- Icons are displayed in a horizontal row
- Smooth transition between states
- Proper `alt` text / accessible labels

## Stretch Goals
- Combine the sprite with a CSS animation for a loading effect
- Add tooltips on hover
- Generate the sprite programmatically using a canvas

## Evaluation Criteria
- Single image file used for all icons
- Correct `background-position` values
- All three states (default, hover, active) work
- Clean, DRY CSS
- Modern fallback using SVG if sprite fails to load
