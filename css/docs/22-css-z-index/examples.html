CSS Z-index - Examples

1. Basic Z-index Ordering

Higher z-index values appear on top.

.box1 { z-index: 1; }
.box2 { z-index: 2; }
.box3 { z-index: 3; }
z-index: 1
z-index: 2
z-index: 3

2. Negative Z-index

Negative z-index goes behind the parent's background.

.box { position: absolute; z-index: -1; }
Parent Background
z-index: -1 (behind)

3. Stacking Context Isolation

Child z-index is limited to its parent's stacking context.

.parent-a { z-index: 1; }
.parent-b { z-index: 2; }
/* Child z-index: 999 is isolated within parent-b */
Even though Child B has z-index: 999, it's isolated within Parent B (z-index: 2), which is above Parent A (z-index: 1).
Parent A (z-index: 1)
Child A (z-index: 999)
Parent B (z-index: 2)
Child B (z-index: 999)

4. Modal with Proper Z-index

Using z-index scale: overlay at 1000, modal at 1001.

.overlay { z-index: 1000; }
.modal { z-index: 1001; }

Modal Title

This modal uses proper z-index layering.

Overlay: z-index 1000

Modal: z-index 1001

5. Tooltips and Dropdowns

Z-index ensures tooltips and dropdowns appear above other content.

.tooltip { z-index: 100; }
.dropdown-menu { z-index: 100; }
