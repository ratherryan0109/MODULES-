Example 8: Configuration (tailwind.config)

Tailwind is fully customizable through its config file where you define colors, fonts, breakpoints, and more.

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: {
          500: '#06b6d4',
          700: '#0891b2',
        }
      },
      fontFamily: {
        sans: ['Inter', 'system-ui', 'sans-serif'],
      }
    }
  }
}
