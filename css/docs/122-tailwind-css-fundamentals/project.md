# Mini Project: Tailwind CSS Marketing Site

## Overview
Build a multi-page marketing website using Tailwind CSS. The site should be fully responsive, have dark mode support, and demonstrate mastery of Tailwind utilities.

## Pages
1. **Home** — Hero section, features grid, testimonials carousel
2. **Pricing** — Pricing cards with feature comparison
3. **About** — Team grid with hover effects

## Requirements
- Responsive design (mobile-first with Tailwind breakpoints)
- Dark mode toggle using Tailwind's dark: prefix
- Custom theme configuration (brand colors, fonts)
- Interactive components (buttons with states, cards with hover)
- No custom CSS — all styling via Tailwind utilities
- One `@apply` component extraction

## Deliverables
1. `index.html` — All pages in one multi-section HTML
2. `tailwind.config.js` — Custom configuration
3. `README.md` — Setup instructions

## Evaluation
- Responsive implementation: 25 pts
- Dark mode integration: 20 pts
- Utility-only styling: 20 pts
- Custom configuration: 15 pts
- Visual quality: 10 pts
- Code organization: 10 pts
