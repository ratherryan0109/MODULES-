# Mini Project: Interactive Product Showcase

## Objective
Build an interactive product showcase page that uses various 2D transforms for product cards, hover effects, and entrance animations.

## Requirements
1. Grid of product cards with image, title, price, and CTA
2. Card hover: lift (translateY), enlarge (scale), shadow deepens
3. Image zoom on hover (scale inside overflow:hidden container)
4. Button press effect (scale down on :active)
5. Entrance animation: cards slide and fade in from different directions
6. Badge/ribbon with skew effect

## Specs
- Use minimum 4 different 2D transform functions
- Demonstrate transform order importance
- Use transform-origin for at least one effect
- Smooth transitions (0.2s-0.5s)
- Responsive grid layout

## Stretch Goals
- Add a quick-view modal with scale entrance
- Implement wishlist heart animation (scale pulse)
- Add parallax tilt effect on hover using mouse position

## Evaluation Criteria
- Correct and effective use of 2D transforms
- Smooth animations and interactions
- Transform order logic is correct
- Code is clean and maintainable
