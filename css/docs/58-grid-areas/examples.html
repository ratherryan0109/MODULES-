Grid Areas Examples

Visual layouts using grid-template-areas and grid-area

1. Basic Page Layout

Classic header, nav, main, aside, footer with named areas.

Header
Nav
Main
Aside
Footer
.page {
    grid-template-columns: 150px 1fr 150px;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header header"
        "nav    main   aside"
        "footer footer footer";
}
header { grid-area: header; }
nav    { grid-area: nav; }
main   { grid-area: main; }
aside  { grid-area: aside; }
footer { grid-area: footer; }

3. Empty Cells with Dots

Using . for empty cells creates intentional whitespace.

Title
Content
Footer
.layout {
    grid-template-areas:
        "title  title  ."
        ".      .      content"
        ".      footer footer";
}
/* Dots (.) create intentionally empty cells */
Notice how the empty cells leave gaps in the layout. The total grid is always a rectangle.

5. Responsive Areas (Desktop Preview)

This layout redefines areas at different breakpoints. Resize to see!

Header
Nav
Main Content
Aside
Footer
/* Mobile: single column */
.container { grid-template-areas: 'header' 'nav' 'main' 'aside' 'footer'; }

/* Tablet: 2 columns */
@media(min-width:600px) {
    .container { grid-template-areas: 'header header' 'nav main' 'aside aside' 'footer footer'; }
}

/* Desktop: 3 columns */
@media(min-width:900px) {
    .container { grid-template-areas: 'header header header' 'nav main aside' 'footer footer footer'; }
}
💡 Resize this page to see the layout adapt from 1 column (mobile) to 2 columns (tablet) to 3 columns (desktop). The HTML never changes!
