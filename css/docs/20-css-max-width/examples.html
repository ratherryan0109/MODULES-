CSS Max-Width - Examples

Example 1: Responsive Container

max-width with width: 100% creates a container that fills small screens but caps at a maximum size on large screens.

max-width: 700px; width: 100%; margin: 0 auto;
Shrinks on small screens, capped at 700px on large screens.
.container {
  max-width: 700px;
  width: 100%;      /* Fills parent when smaller than 700px */
  margin: 0 auto;   /* Centers horizontally */
}

Example 2: With vs Without Max-Width

Without max-width, a block element stretches to fill its container, which can be unreadable on wide screens.

Without max-width
This stretches to fill the entire container width, which can be hard to read.
With max-width
This is capped at 700px, improving readability on wide screens.
.no-limit  { width: 100%; }               /* Stretches fully */
.limited   { max-width: 700px; width: 100%; }  /* Capped */

Example 3: Optimal Line Length (65ch)

Using max-width: 65ch creates the ideal line length for readability (about 65 characters per line).

max-width: 65ch is widely considered the optimal line length for comfortable reading. Lines that are too long cause the reader's eye to have trouble tracking from the end of one line to the start of the next. Lines that are too short cause excessive eye movement and break the reading rhythm. The "ch" unit equals the width of the "0" character in the current font, making it a natural choice for text containers. 
.article {
  max-width: 65ch;    /* ~65 characters per line */
  margin: 0 auto;
  line-height: 1.8;
}

Example 4: Responsive Image

max-width: 100%; height: auto makes images scale down on small screens without cropping or overflowing.

img {
  max-width: 100%;   /* Never wider than parent */
  height: auto;       /* Maintain aspect ratio */
}

Example 5: Min-Width and Max-Width Combined

Set a responsive range: min-width prevents the element from becoming too narrow; max-width prevents it from becoming too wide.

min-width: 150px, max-width: 500px
Grows and shrinks within this range.
.constrained {
  min-width: 150px;
  max-width: 500px;
  width: 100%;           /* Fluid between 150px and 500px */
}

Example 6: Centering with Max-Width and Auto Margins

The classic centering technique: max-width + margin: 0 auto.

max-width: 300px; margin: 0 auto;
Centers horizontally
.center {
  max-width: 300px;
  margin: 0 auto;    /* Auto left/right = centering */
}

Example 7: Max-Height for Scrollable Containers

max-height limits vertical growth and pairs with overflow to create scrollable areas.

max-height: 120px — scroll when content exceeds

Line 2: This container has a maximum height.

Line 3: When content overflows, a scrollbar appears.

Line 4: This is useful for expandable sections.

Line 5: Chat messages, notification panels, etc.

Line 6: The scrollbar only shows if needed.

.scrollable {
  max-height: 120px;
  overflow-y: auto;   /* Scroll only when needed */
}
