# Mini Project: Responsive Blog Layout

Create a responsive blog layout using max-width:
- Container with max-width: 800px centered on the page
- Header with logo and navigation
- Article content with 65ch max-width for reading
- Sidebar that appears beside content on wide screens, below on narrow
- Responsive images (max-width: 100%)
- Footer with max-width constraints
- All sections use consistent max-width patterns
