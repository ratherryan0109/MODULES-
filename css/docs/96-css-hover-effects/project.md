# Mini Project: Interactive Product Catalog with Hover Effects

## Objective
Build a product catalog grid where each item has multiple hover effects for an engaging browsing experience.

## Requirements
1. Product grid (3×3 or auto-fill)
2. Card with image, title, price, and action button
3. Image zoom overlay on hover (scale + darken)
4. Card lift (translateY + shadow)
5. Button fill effect on hover
6. Color swatch selection with border highlight
7. Focus-visible styles for keyboard navigation

## Stretch Goals
- Quick-view button appears on hover
- Wishlist heart toggle with scale animation
- Tooltip on hover for truncated text
- Smooth product variant swap on swatch hover

## Evaluation
- All hover effects use GPU-accelerated properties
- Proper transition durations (200-400ms)
- Works on touch devices (basic functionality)
- Accessible with keyboard navigation
- Respects prefers-reduced-motion
