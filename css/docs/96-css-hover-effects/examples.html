CSS Hover Effects

1. Lift Effect

Element lifts up and casts a shadow on hover using translateY and box-shadow.

.btn:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 20px rgba(59, 130, 246, 0.4);
}

2. Fill Effect

An outlined button fills with color on hover — the background transitions from transparent to solid.

.btn {
  background: transparent; color: #8b5cf6;
  border: 2px solid #8b5cf6;
  transition: all 0.3s ease;
}
.btn:hover { background: #8b5cf6; color: white; }

3. Glow Effect

A multi-layered box-shadow creates a glowing aura around the button on hover.

.btn:hover {
  box-shadow:
    0 0 15px rgba(16, 185, 129, 0.5),
    0 0 30px rgba(16, 185, 129, 0.2);
}

4. Underline Animation

A pseudo-element underline grows from left to right on hover using width transition.

.btn { position: relative; }
.btn::after {
  content: ''; position: absolute;
  bottom: 0; left: 0; width: 0; height: 2px;
  background: #f59e0b;
  transition: width 0.3s ease;
}
.btn:hover::after { width: 100%; }

5. Scale / Press Effect

Scales up on hover and down on click (:active), mimicking a physical button press.

.btn:hover { transform: scale(1.1); }
.btn:active { transform: scale(0.95); }

6. Card Hover

A card that lifts and scales subtly on hover with an enhanced shadow, common in product grids.

Product Name

Premium quality item with great features.

Service

Professional service tailored to you.

.card:hover {
  transform: translateY(-8px) scale(1.02);
  box-shadow: 0 16px 40px rgba(0, 0, 0, 0.1);
}

7. Image Zoom

An image inside an overflow: hidden container scales up with reduced brightness on hover.

Image
.img-wrap { overflow: hidden; }
.img-wrap img {
  transition: transform 0.5s ease, filter 0.3s ease;
}
.img-wrap:hover img {
  transform: scale(1.15);
  filter: brightness(0.7);
}

8. Rotate & Border Grow

A subtle rotation on hover and a border that thickens — small touches that add polish.

Rotate Border 
.rotate:hover { transform: rotate(8deg); }
.border:hover { border-width: 3px; }
