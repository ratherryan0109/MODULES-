# Mini Project: Interactive Image Gallery with Zoom Effects

## Objective
Build an interactive image gallery that uses CSS scale for hover zoom, lightbox open animation, and thumbnail grid effects.

## Requirements
1. Grid of thumbnail images with hover zoom effect
2. Click thumbnail to open lightbox with scale-in animation
3. Lightbox image starts at scale 0 and animates to scale 1
4. Close button with scale-down effect
5. Navigation arrows with hover scale effects
6. Images should not overflow their containers

## Specs
- Use standalone `scale` property throughout
- Minimum 3 different scale values used
- Smooth transitions (0.3s - 0.5s)
- Responsive grid layout
- Overflow hidden on thumbnail containers

## Stretch Goals
- Add a masonry layout with scale entrance animations
- Implement pinch-to-zoom on mobile using scale
- Add a scale-based ripple effect on gallery items
- Create a slideshow with scale transitions between images

## Evaluation Criteria
- Scale animations are smooth and performant
- Lightbox open/close animations work correctly
- Images don't overflow their containers
- Responsive design works on all screen sizes
