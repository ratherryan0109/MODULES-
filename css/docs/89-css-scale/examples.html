CSS Scale — Practical Examples

1. Uniform Scale (Enlarge)

Element scales equally on both axes.

scale(1.3)
scale(1.5)
scale: 1.3; /* 30% larger */

2. Uniform Scale (Shrink)

Element scales down.

scale(0.8)
scale(0.5)
scale: 0.5; /* half size */

3. Non-Uniform Scale (X vs Y)

Different scale values for X and Y axes.

scale(1.5, 1)
scale(1, 1.5)
scale: 1.5 1; /* X stretches, Y stays */

4. Negative Scale (Mirror)

Negative values flip the element.

scale(-1, 1)
scale: -1 1; /* horizontal mirror */

5. Scale with Transform Origin

Same scale value but different origins.

center
top left
bottom right

6. Scale on Click (Active)

Button press effect with scale.

:active
.btn:active { scale: 0.95; } /* press effect */
