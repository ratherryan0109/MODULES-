CSS Logical Properties — Interactive Examples

Language direction:

1. Margin and Padding with Logical Properties

The card below uses margin-inline: auto for centering and padding-inline/padding-block for spacing. The left accent border uses border-inline-start — it automatically moves to the right side in RTL!

Logical Card

This card is centered with margin-inline: auto and has padding-inline: 24px. The accent border is on border-inline-start.

.card { margin-inline: auto; margin-block: 20px; padding-inline: 24px; padding-block: 16px; border-inline-start: 4px solid #89b4fa; }

2. Logical Border Radius

border-start-start-radius targets the corner at the start of both axes. In LTR it's top-left; in RTL it's top-right.

start-start
.box { border-start-start-radius: 20px; border-end-end-radius: 20px; }

3. Inline and Block Sizing

inline-size replaces width, block-size replaces height. They adapt to the writing mode.

inline-size: 100%
inline-size: 50%
inline-size: 33.33%
.item { inline-size: 100%; block-size: 40px; } .item.half { inline-size: 50%; }

4. Text Alignment with Start/End

text-align: start aligns to the left in LTR and to the right in RTL without any additional overrides.

text-align: start — ← in LTR, → in RTL

text-align: end — → in LTR, ← in RTL

text-align: center — always centered

5. Absolute Positioning with Inset

The badge uses inset-inline-end and inset-block-start to position in the top-right (LTR) or top-left (RTL).
