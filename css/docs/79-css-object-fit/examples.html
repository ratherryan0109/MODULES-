CSS Object-Fit & Object-Position

Example 1: object-fit Values Comparison

All images are 160x140 containers receiving a 300x200 source image. Each value handles it differently.

fill (stretched)
contain (fits)
cover (cropped)
none (original)
scale-down
.fill       { object-fit: fill; }       /* Stretches, may distort */
.contain    { object-fit: contain; }    /* Fits entire image, may show bars */
.cover      { object-fit: cover; }      /* Covers area, may crop */
.none       { object-fit: none; }       /* Original size, may overflow */
.scale-down { object-fit: scale-down; } /* Smaller of none vs contain */
cover is the most commonly used value — it fills the container while preserving aspect ratio, cropping the excess.

Example 2: object-position

Controls which part of the image stays visible when using object-fit: cover. All images use a wide 400x200 source.

center (default)
top
right
20% 80%
.center  { object-fit: cover; object-position: center; }  /* Default — keeps middle visible */
.top     { object-fit: cover; object-position: top; }     /* Keeps top visible */
.right   { object-fit: cover; object-position: right; }   /* Keeps right side visible */
.custom  { object-fit: cover; object-position: 20% 80%; } /* Custom x y position */
object-position is essential for portrait photos placed in landscape containers — choose which part stays visible.

A square image in a rectangular container. fill distorts, cover crops to fill.

Example 4: Aspect-Ratio + object-fit

Combine aspect-ratio with object-fit: cover for predictable image crops across different aspect ratios.

1:1 square
4:3 landscape
16:9 widescreen
3:4 portrait
.image {
    width: 100%;
    height: 100%;
    object-fit: cover; /* Crops to fill the aspect-ratio box */
}
.container { aspect-ratio: 16/9; } /* Set desired ratio on parent */

Example 5: Video with object-fit: cover

Example 6: Gallery Grid with object-fit

Uniform thumbnail grid where all images are square-cropped with cover.

.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
    gap: 8px;
}
.gallery img {
    width: 100%;
    height: 100%;
    object-fit: cover; /* Forces all images into uniform squares */
    aspect-ratio: 1;
}
object-fit: cover is the standard for image galleries — it handles mixed portrait/landscape images perfectly.
