# Project: Responsive Image Gallery with Object-Fit

## Objective
Build a professional responsive image gallery that demonstrates mastery of object-fit and object-position for consistent, visually appealing image displays.

## Requirements
- Responsive grid using auto-fill with minmax()
- Images of various original aspect ratios displayed consistently using object-fit: cover
- A toggle that switches between object-fit values (cover, contain, fill, none, scale-down) to show the differences
- Object-position controls (center, top, bottom, left, right, custom) with live preview
- Each image container should have a visible background color to show letterboxing when using contain
- Display the current CSS properties and values being used
- Aspect-ratio applied to containers for layout stability

## Stretch Goals
- Upload your own image to test
- Side-by-side comparison of different object-fit values
- Video element demonstration with object-fit: cover
- Animate between object-fit values

## Learning Outcomes
- Master object-fit values and their visual effects
- Use object-position for precise image framing
- Combine with aspect-ratio for stable layouts
- Build responsive media components
