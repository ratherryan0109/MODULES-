CSS Animation Properties

1. Animation Fill Mode

none snaps back, forwards holds the end state, both also pre-applies the start state during delay. All have a 0.3s delay.

none
forwards
both
animation: slideIn 1s ease 0.3s none;    /* snaps back */
animation: slideIn 1s ease 0.3s forwards; /* holds end */
animation: slideIn 1s ease 0.3s both;     /* holds both */

2. Iteration Count

Control how many times the animation plays: once, three times, or infinitely.

1x
3x
∞
animation-iteration-count: 1;
animation-iteration-count: 3;
animation-iteration-count: infinite;

3. Animation Direction

normal plays forward, reverse plays backward, alternate reverses each cycle.

normal
reverse
alternate
alt-rev
animation-direction: normal;           /* forward only */
animation-direction: reverse;          /* backward only */
animation-direction: alternate;        /* forward, then back */
animation-direction: alternate-reverse; /* back, then forward */

4. Staggered Delays

Same pop animation with increasing animation-delay values for a cascading entrance effect.

0s
0.1s
0.2s
0.3s
0.4s
.item:nth-child(1) { animation-delay: 0s; }
.item:nth-child(2) { animation-delay: 0.1s; }
.item:nth-child(3) { animation-delay: 0.2s; }
.item:nth-child(4) { animation-delay: 0.3s; }
.item:nth-child(5) { animation-delay: 0.4s; }

5. Pause on Hover

Hover over the box to pause the floating animation via animation-play-state: paused.

hover
.element {
  animation: float 1.5s ease infinite;
}
.element:hover {
  animation-play-state: paused;
}

6. Animation Duration Comparison

Different animation-duration values — from fast (0.5s) to slow (4s).

0.5s
1s
2s
4s
animation-duration: 0.5s;  /* fast */
animation-duration: 1s;    /* normal */
animation-duration: 2s;    /* slow */
animation-duration: 4s;    /* very slow */

7. Timing Functions

Each box moves at the same speed but with a different easing curve.

linear
ease
ease-in
ease-out
cubic-bezier
animation-timing-function: linear;
animation-timing-function: ease;
animation-timing-function: ease-in;
animation-timing-function: ease-out;
animation-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55);
