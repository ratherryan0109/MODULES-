# Mini Project: CSS Performance Audit & Optimization

## Overview
Audit a provided CSS stylesheet (or any real website), identify performance issues, and implement optimizations.

## Requirements
1. Run a CSS coverage audit
2. Extract critical above-fold CSS
3. Implement async loading for non-critical CSS
4. Fix any layout-triggering animations
5. Add content-visibility where applicable
6. Minify and purge unused CSS
7. Measure before/after metrics (file size, FCP, coverage %)

## Deliverables
1. `audit-report.md` — Findings and metrics
2. `critical.css` — Extracted critical CSS
3. `optimized.html` — Page with optimized CSS loading

## Evaluation
- Coverage audit: 15 pts
- Critical CSS extraction: 20 pts
- Async loading implementation: 15 pts
- Animation fixes: 15 pts
- content-visibility usage: 10 pts
- Size reduction: 15 pts
- Report quality: 10 pts
