Each item gets an equal share of the available space.

Example 2: Proportional Growth (1 : 2 : 1)

The middle item gets twice the extra space of its siblings.

flex: 1
flex: 2
flex: 1
flex-grow: 1  |  flex-grow: 2  |  flex-grow: 1
.sidebar { flex: 1; }
.main   { flex: 2; }  /* Takes twice the extra space */

Example 3: Sidebar + Content Layout

A practical layout: narrow sidebar, wide content area.

Main Content

Example 7: How flex-grow Works

The formula: flex-grow distributes extra space proportionally. Total extra space is divided by the sum of all grow factors, then each item gets its share.

25%

50%

Sum of grow factors = 4. Each "share" = extra space ÷ 4.
