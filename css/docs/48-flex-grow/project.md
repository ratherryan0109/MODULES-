# Mini Project: Flexible Content Layout

## Overview
Build a flexible content page with sidebar, main content, and aside using `flex-grow`.

## Requirements
1. Three-column layout: sidebar, main, aside
2. Sidebar: flex-grow: 1, Main: flex-grow: 3, Aside: flex-grow: 1
3. At 900px container width, demonstrate visible proportions
4. Add a fixed-width (flex-grow: 0) element somewhere
5. Responsive: collapse to single column on mobile

## Bonus
- Add live flex-grow adjustment controls
- Show computed widths in pixels

## Evaluation
- Correct grow proportions: 30pts
- Fixed-width element: 20pts
- Responsive behavior: 25pts
- Code quality: 25pts
