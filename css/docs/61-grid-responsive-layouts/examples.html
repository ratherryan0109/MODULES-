2. auto-fill vs auto-fit — The Difference

auto-fill (keeps empty space)

At large viewports, empty column tracks remain visible.

1
2
3
grid-template-columns:
  repeat(auto-fill, minmax(120px, 1fr));

auto-fit (collapses empty space)

At large viewports, empty tracks collapse; items expand.

1
2
3
grid-template-columns:
  repeat(auto-fit, minmax(120px, 1fr));
💡 At a 900px viewport with 3 items: auto-fill has an empty 4th column slot; auto-fit expands all 3 to fill the space.
