# Mini Project: CSS Architecture Refactor

## Overview
Given a legacy CSS file (500+ lines of messy, duplicated, !important-ridden styles), refactor it into a clean, maintainable architecture.

## Requirements
1. Create a CSS architecture plan (layers, naming convention, custom properties)
2. Refactor all colors into custom properties
3. Implement BEM naming throughout
4. Remove all !important usage
5. Remove all unused CSS
6. Add defensive CSS patterns
7. Organize with @layer

## Deliverables
1. `architecture.md` — Plan document
2. `styles.css` — Refactored stylesheet
3. `refactor-report.md` — Changes made

## Evaluation
- Custom properties used: 15 pts
- BEM naming consistent: 20 pts
- No !important: 15 pts
- Defensive CSS added: 15 pts
- Layer organization: 15 pts
- Documentation: 10 pts
- File size reduction: 10 pts
