Example 1: Numeric types (int, float, complex) Basic

Core foundation for understanding this topic. This example demonstrates how to work with numeric types (int, float, complex) in Python.


# Example 1: Numeric types (int, float, complex)
# Core foundation for understanding this topic

def demonstrate_numeric_types_int_float_complex():
    """Demonstrate Numeric types (int, float, complex) with Python code."""
    # Initialize sample data
    data: list = [1, 2, 3, 4, 5]
    print(f"Original data: {data}")

    # Apply numeric types (int, float, complex) pattern
    result: list = process_data(data)
    print(f"Processed result: {result}")

    return result

def process_data(items: list) -> list:
    """Process items according to numeric types (int, float, complex) rules."""
    # Processing logic
