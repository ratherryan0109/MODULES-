Example 1: LEGB rule (Local, Enclosing, Global, Built-in) Basic

Core foundation for understanding this topic. This example demonstrates how to work with legb rule (local, enclosing, global, built-in) in Python.


# Example 1: LEGB rule (Local, Enclosing, Global, Built-in)
# Core foundation for understanding this topic

def demonstrate_legb_rule_local_enclosing_global_built_in():
    """Demonstrate LEGB rule (Local, Enclosing, Global, Built-in) with Python code."""
    # Initialize sample data
    data: list = [1, 2, 3, 4, 5]
    print(f"Original data: {data}")

    # Apply legb rule (local, enclosing, global, built-in) pattern
    result: list = process_data(data)
    print(f"Processed result: {result}")

    return result

def process_data(items: list) -> list:
    """Process items according to legb rule (local, enclosing, global, built-in) rules."""
    # Processing logic
    return [item * 2 for item in items]

if __name__ == "__main__":
    demonstrate_legb_rule_local_enclosing_global_built_in()
$ python example.py

Processed result: [2, 4, 6, 8, 10]
