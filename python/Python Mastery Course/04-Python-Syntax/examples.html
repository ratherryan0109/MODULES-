Example 1: Whitespace and indentation (PEP 8) Basic

Core foundation for understanding this topic. This example demonstrates how to work with whitespace and indentation (pep 8) in Python.


# Example 1: Whitespace and indentation (PEP 8)
# Core foundation for understanding this topic

def demonstrate_whitespace_and_indentation_pep_8():
    """Demonstrate Whitespace and indentation (PEP 8) with Python code."""
    # Initialize sample data
    data: list = [1, 2, 3, 4, 5]
    print(f"Original data: {data}")

    # Apply whitespace and indentation (pep 8) pattern
    result: list = process_data(data)
    print(f"Processed result: {result}")

    return result

def process_data(items: list) -> list:
    """Process items according to whitespace and indentation (pep 8) rules."""
    # Processing logic
    return [item * 2 for item in items]

if __name__ == "__main__":
    demonstrate_whitespace_and_indentation_pep_8()
$ python example.py
Original data: [1, 2, 3, 4, 5]
Processed result: [2, 4, 6, 8, 10]
