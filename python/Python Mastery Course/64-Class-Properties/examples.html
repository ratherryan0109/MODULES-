Example 1: @property decorator (getter) Basic

Core foundation for understanding this topic. This example demonstrates how to work with @property decorator (getter) in Python.


# Example 1: @property decorator (getter)
# Core foundation for understanding this topic

def demonstrate_property_decorator_getter():
    """Demonstrate @property decorator (getter) with Python code."""
    # Initialize sample data
    data: list = [1, 2, 3, 4, 5]
    print(f"Original data: {data}")

    # Apply @property decorator (getter) pattern
    result: list = process_data(data)
    print(f"Processed result: {result}")

    return result

def process_data(items: list) -> list:
    """Process items according to @property decorator (getter) rules."""
    # Processing logic
    return [item * 2 for item in items]

if __name__ == "__main__":
    demonstrate_property_decorator_getter()
$ python example.py

Processed result: [2, 4, 6, 8, 10]
