Python Introduction — Examples

Real Python examples demonstrating print(), strings, and program structure.

Example 1: Hello, World! Basic

The simplest Python program. Every programmer's first step.


# The classic first program
# Displays "Hello, World!" on the screen
print("Hello, World!")
$ python hello.py
Hello, World!

Example 2: Printing Different Data Types Basic

You can print strings, numbers, and calculation results.


# Printing different types of data
print("Text")       # String
print(42)           # Integer (whole number)
print(3.14159)      # Float (decimal number)
print(2 + 2)        # Expression — Python calculates first, then prints
print("Result:", 10 * 5)  # Mixing text and numbers
$ python types.py
Text
42
3.14159
4
Result: 50

Example 3: Multiple Arguments and Custom Separators Basic

Using commas to print multiple items and the sep parameter to control spacing.


# Multiple items (comma-separated)
print("apple", "banana", "cherry")

# Custom separator with sep parameter
print("apple", "banana", "cherry", sep=", ")
print("apple", "banana", "cherry", sep=" | ")
print("apple", "banana", "cherry", sep="")
$ python separators.py
apple banana cherry
apple, banana, cherry
apple | banana | cherry
applebananacherry

Example 4: Controlling Line Endings Medium

By default, print() ends with a newline. The end parameter changes this behavior.


# Default: each print adds a newline
print("Line 1")
print("Line 2")
print("Line 3")

# Using end="" keeps output on the same line
print("Hello ", end="")
print("World")             # Output: Hello World

# Using end with custom character
print("Processing", end="... ")
print("Done!")             # Output: Processing... Done!
$ python endings.py
Line 1
Line 2
Line 3
Hello World
Processing... Done!

Example 5: String Repetition for Borders Medium

Multiplying a string by a number repeats it — perfect for creating visual borders.


# Repeating characters with string multiplication
print("=" * 30)
print("     SECTION HEADER")
print("=" * 30)

# Different border styles
print("-" * 30)
print("*" * 30)
print("#" * 30)
$ python borders.py
==============================
SECTION HEADER
==============================
------------------------------
******************************
##############################

Example 6: Multi-line Output with One Print Medium

Using \n (newline) inside a string to create multi-line output from a single print().


# Using \n (newline character) inside a string
print("Line 1\nLine 2\nLine 3")

# Practical: formatted address
print("John Doe\n123 Main Street\nNew York, NY 10001\nUSA")

# Poem with line breaks
print("Roses are red,\nViolets are blue,\nPython is awesome,\nAnd so are you!")
$ python multiline.py
Line 1
Line 2
Line 3
John Doe
123 Main Street
New York, NY 10001
USA
Roses are red,
Violets are blue,
Python is awesome,
And so are you!

Example 7: Program with Comments Basic

Comments explain code without affecting execution. They are for the human reader.


# This is a comment — Python ignores it
print("This code runs")  # Inline comment

# Comments can explain WHY you did something
# (not WHAT the code does — the code shows that)

# Calculate total cost
print(5 * 10)   # 5 items at $10 each = $50
print(3 * 8)    # 3 items at $8 each = $24
print(50 + 24)  # Total cost
$ python comments.py
This code runs
50
24
74

Example 8: Common Mistakes Debugging

Learn what goes wrong so you can fix it when it happens.


# MISTAKE 1: Missing quotes
# print(Hello)  # ← NameError
print("Hello")   # ← Fixed: added quotes

# MISTAKE 2: Mismatched quotes
# print("Hello')  # ← SyntaxError
print("Hello")    # ← Fixed: matching quotes

# MISTAKE 3: Missing parentheses (Python 3)
# print "Hello"   # ← SyntaxError
print("Hello")    # ← Fixed: added parentheses

# MISTAKE 4: Wrong capitalization
# Print("Hello")  # ← NameError
print("Hello")    # ← Fixed: lowercase p
$ python fixes.py
Hello
Hello
Hello
Hello
Try uncommenting each mistake to see the error. Understanding errors is a key programming skill!
